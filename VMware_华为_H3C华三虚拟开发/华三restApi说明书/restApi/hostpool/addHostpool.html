创建主机池

创建主机池。

接口 URI

/cas/casrs/hostpool/add/{name}

访问方法

HTTP 方法
POST
参数
路径参数（Path Parameter）
name 主机池名称 。
字符串（String）类型。必须。
 

返回结果

状态码
  • 204(No Content)：成功。
消息头
HTTP/1.1 204 No Content
Content-Type: application/xml
Date: Wed, 17 Sep 2014 00:48:42 GMT
Server: H3C-CVM

 

应用举例

1.创建名称为“aa”的主机池

请求
POST http://localhost:8080/cas/casrs/hostpool/add/aa
accept: application/xml
回应
HTTP/1.1 204 No Content
Date: Wed, 17 Sep 2014 00:48:42 GMT
Server: H3C-CVM

无消息体
