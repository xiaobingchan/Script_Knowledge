H3C CAS云管理平台

查询CAS系统中指定主机池下的共享文件系统列表

查询CAS系统中指定主机池下的所有主机列表。

接口 URI

/cas/casrs/hostpool/shareFile/{hpId}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
hpId 主机池id 。
长整数（long）类型。必须。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Content-Type: application/xml
Content-Length: 1656
Date: Wed, 17 Sep 2014 01:03:11 GMT
Server: H3C-CVM
消息体
<?xml version="1.0" encoding="UTF-8"?>
<list>
  <rsShareFileSystem>
    <id>2</id>
    <name>lun-03</name>
    <type>1</type>
    <path>/vms/lun-03</path>
  </rsShareFileSystem>
</list>
元素含义
rsShareFileSystem各子元素
id 共享文件系统 ID。
长整型（Long）类型。
name 共享文件系统名称。
字符串（String）类型。
description 共享文件系统描述。
字符串（String）类型。
type 共享文件类型。1：iSCSI，2：FC。
整型（Integer）类型。
path mount目录。
字符串（String）类型。

 

应用举例

1.查询 CAS系统中ID为“1”的主机池下的共享文件系统列表。

请求
GET http://localhost:8080/cas/casrs/hostpool/shareFile/1
accept: application/xml
...
回应

Content-Type: application/xml
Content-Length: 1656

<?xml version="1.0" encoding="UTF-8"?>
<list>
  <rsShareFileSystem>
    <id>2</id>
    <name>lun-03</name>
    <type>1</type>
    <path>/vms/lun-03</path>
  </rsShareFileSystem>
</list>
