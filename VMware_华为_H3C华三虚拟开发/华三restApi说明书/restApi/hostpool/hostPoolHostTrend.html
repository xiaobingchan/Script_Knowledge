H3C CAS云管理平台

查询指定主机池下的主机趋势信息

根据主机池id，type及interval查询指定主机池下的主机趋势信息。

接口 URI

/cas/casrs/hostpool/hostTrend/{hpId}/{type}/{interval}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
hpId 主机池id 。
长整数（long）类型。必须。
type 查询类型 。两种类型： CPU利用率：cpu，内存利用率：mem
字符串（String）类型。必须。
interval 时间长度 。 单位为分钟。
整数（Integer）类型。必须。
