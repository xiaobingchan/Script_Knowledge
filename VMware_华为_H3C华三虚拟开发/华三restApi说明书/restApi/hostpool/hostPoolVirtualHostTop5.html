H3C CAS云管理平台

查询指定主机池下的虚拟机Top5信息

根据主机池id及type查询指定主机池下的Top5虚拟机信息。

接口 URI

/cas/casrs/hostpool/virtHostTopN/{hpId}/{type}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
hpId 主机池id 。
长整数（long）类型。必须。
type 查询类型 。两种类型： CPU利用率：cpu，内存利用率：mem
字符串（String）类型。必须。

状态码

元素含义
rate各子元素
id 虚拟机ID。
长整型（Long）类型。
name 虚拟机名称。
字符串（String）类型。
rate 利用率。
小数（Double）类型。
hostId 所属的主机ID。 。
长整型（Long）类型。
clusterId 所属的集群 ID。
长整型（Long）类类型。
