查询CAS系统中指定主机池下的主机列表

查询CAS系统中指定主机池下的主机列表。

接口 URI

/cas/casrs/hostpool/host/{hpId}?hostName={hostName}&offset={offset}&limit={limit}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
hpId 主机池id 。
长整数（long）类型。必须。
路径参数（Query Parameter）
hostName 主机名称类型 。
字符串（String）类型。可选。
offset 查询起始位置 。
整数（Integer）类型。必须。
limit 查询返回数据量 。
整数（Integer）类型。必须。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Total: 1
Content-Type: application/xml
Content-Length: 370
Date: Tue, 16 Sep 2014 09:44:07 GMT
消息体
<?xml version="1.0" encoding="UTF-8"?>
<list>
  <host>
    <id>4</id>
    <hostPoolId>2</hostPoolId>
    <clusterId>1</clusterId>
    <name>server21</name>
    <vmNum>31</vmNum>
    <vmRunCount>29</vmRunCount>
    <vmShutoff>2</vmShutoff>
    <cpuRate>29.675</cpuRate>
    <memRate>52.79483333333334</memRate>
    <status>1</status>
    <storage>884583</storage>
    <version>CVK 2.0 (C0205)</version>
  </host>
</list>
元素含义
host各子元素
id 主机 ID。
长整型（Long）类型。
hostPoolId 主机池ID。
长整型（Long）类型。
clusterId 集群ID。
长整型（Long）类型。
name 主机名称。
字符串（String）类型。
vmNum 主机下虚拟机数量。
整型（Integer）类型。
vmRunCount 主机下运行的虚拟机数量。
整型（Integer）类型。
vmShutoff 主机下关闭的虚拟机数量。
整型（Integer）类型。
cpuRate 主机CPU利用率。
小数（Double）类型。
memRate 主机内存利用率。
小数（Double）类型。。
status 主机状态， 0：不正常，1：正常。
整型（Integer）类型。
storage 主机存储大小。
字符串（String）类型。
version 主机版本。
字符串（String）类型。
