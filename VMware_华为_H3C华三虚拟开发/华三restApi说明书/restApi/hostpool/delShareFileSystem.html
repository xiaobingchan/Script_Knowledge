删除主机池下的共享文件系统

删除主机池下的共享文件系统。

接口 URI

/cas/casrs/hostpool/delete/{id}?hpId={hpId}

访问方法

HTTP 方法
DELETE
参数
路径参数（Path Parameter）
id 共享文件系统id 。
长整数（long）类型。必须。
路径参数（Query Parameter）
hpId 主机池id 。
长整数（long）类型。必须。

返回结果

状态码
  • 成功：
    204(No Content)
  • 失败：
    409(Conflict)
消息头
HTTP/1.1 204 No Content
Server: Apache-Coyote/1.1
Date: Wed, 17 Sep 2014 02:31:35 GMT

Error-Code: 1202
Error-Message: 共享文件系统“lun-03”正在被使用，不允许删除该共享文件系统。
Charset: GBK
Content-Type: text/html;charset=utf-8
Content-Length: 1017
Date: Wed, 17 Sep 2014 02:31:20 GMT
Server: H3C-CVM

accept: application/xml
