H3C CAS云管理平台

查询指定主机池下的主机Top5信息

根据主机池id及type查询指定主机池下的Top5主机信息。

接口 URI

/cas/casrs/hostpool/hostTopN/{hpId}/{type}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
hpId 主机池id 。
长整数（long）类型。必须。
type 查询类型 。两种类型： CPU利用率：cpu，内存利用率：mem
字符串（String）类型。必须。

返回结果

状态码
  • 200(OK)：成功。

元素含义
rate各子元素

name 主机名称。

rate 利用率。
小数（Double）类型。
