连接主机池下的所有主机（强烈建议不使用该接口）

连接主机池下的所有主机。

由于该方法可能造成CAS系统数据异常，后续可能会删除该接口，建议不要使用，如有需要请使用/cas/casrs/host/connect/v2接口分别连接主机。

接口 URI

/cas/casrs/hostpool/connectAllHost/{hpId}

访问方法

HTTP 方法
PUT
参数
路径参数（Path Parameter）
hpId 主机池id 。
长整数（long）类型。必须。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Content-Type: application/xml
Content-Length: 185
Date: Wed, 17 Sep 2014 02:47:08 GMT
Server: H3C-CVM
消息体
<?xml version="1.0" encoding="UTF-8"?>
<rsResult>
  <state>0</state>
  <errorCode>0</errorCode>
  <message>连接主机池“pool2”下所有主机成功。</message>
</rsResult>
元素含义
rsResult各子元素
state 执行状态。 0： 成功，1：失败，2，部分成功
整型（Integer）类型。
errorCode 错误码。
整型（Integer）类型。
message 成功信息。
字符串（String）类型。

应用举例

1.连接id为“2”的主机池下所有主机。

请求
PUT http://localhost:8080/cas/casrs/hostpool/connectAllHost/2
accept: application/xml
回应
HTTP/1.1 200 OK
Content-Type: application/xml
Content-Length: 185
Date: Wed, 17 Sep 2014 02:47:08 GMT
Server: H3C-CVM

<?xml version="1.0" encoding="UTF-8"?><rsResult>
  <state>0</state>
  <errorCode>0</errorCode>
  <message>连接主机池“pool2”下所有主机成功。</message>
</rsResult>
