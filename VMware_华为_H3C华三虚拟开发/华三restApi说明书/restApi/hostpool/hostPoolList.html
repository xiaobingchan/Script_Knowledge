H3C CAS云管理平台

查询CAS系统中所有主机池列表

查询CAS系统中所有的主机池列表。

接口 URI

/cas/casrs/hostpool/all

访问方法

HTTP 方法
GET
参数

无

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Server: H3C-CVM
Content-Type: application/xml
Content-Length: 363
Date: Tue, 16 Sep 2014 07:18:11 GMT
消息体
<?xml version="1.0" encoding="UTF-8"?>
<list>
  <hostpool>
    <id>1</id>
    <name>iPool-1</name>
    <operatorGroupId>1</operatorGroupId>
    <operatorGroupCode>00</operatorGroupCode>
    <childNum>3</childNum>
    </hostpool>
  <hostpool>
    <id>2</id>
    <name>iPool-2</name>
    <operatorGroupId>1</operatorGroupId>
    <operatorGroupCode>00</operatorGroupCode>
    <childNum>2</childNum>
  </hostpool>
</list>
元素含义
cluster各子元素
id 主机 ID。
长整型（Long）类型。
name 主机池名称。
字符串（String）类型。
operatorGroupId 操作员分组ID。
长整型（Long）类型。
operatorGroupCode 操作员分组编号。
字符串（String）类型。
childNum 直接子节点数量。主机池下的一级主机和集群数量。
整型（Integer）类型。
