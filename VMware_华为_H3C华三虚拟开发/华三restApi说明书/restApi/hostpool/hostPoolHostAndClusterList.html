查询CAS系统中主机池下的主机及集群列表

查询CAS系统中主机池下的主机及集群列表

接口 URI

/cas/casrs/hostpool/{hpId}/allChildNode

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
hpId 主机池id 。
长整数（long）类型。必须。

返回结果

状态码
  • 200(OK)：成功。

元素含义
clusterList各子元素
id 集群 ID。
长整型（Long）类型。
hostPoolId 主机池 ID。
长整型（Long）类型。
name 集群名称。
字符串（String）类型。
description 集群描述。
字符串（String）类型。
enableHA 是否启用HA，枚举值：1=启用，0=不启用。
整型（Integer）类型。
priority 集群启动优先级，枚举值：0：低级，1：中级，2：高级。
整型（Integer）类型。
enableLB 是否启用LB，0：不启用负载均衡 1：启用负载均衡。
整型（Integer）类型。
persistTime 持续时间。
整型（Integer）类型。
checkInterval 检查间隔时长。
整型（Integer）类型。
enableIPM 是否启用电源智能管理， 0：不启用电源智能管理 1：启用电源智能管理。
整型（Integer）类型。
persistTimeIPM 持续时间。
整型（Integer）类型。
ipmLowerMonitorId 主机回收监控策略ID。
长整型（Long）类型。
ipmLowerMonitorName 主机回收监控策略名称。
字符串（String）类型。
ipmUpperMonitorId 主机扩展监控策略ID。
长整型（Long）类型。
ipmUpperMonitorName 主机扩展监控策略名称。
字符串（String）类型。
checkIntervalIPM 检查间隔。
整型（Integer）类型。
operatorGroupId 操作员分组ID。
长整型（Long）类型。
operatorGroupCode 操作员分组编号。
字符串（String）类型。
childNum 直接子节点数量。集群下的主机数量。
整型（Integer）类型。
hostList各子元素
id 主机 ID。
长整型（Long）类型。
user 访问主机用户名称。
字符串（String）类型。
pwd 访问主机密码。
字符串（String）类型。
hostPoolId 主机池 ID。
长整型（Long）类型。
name 主机名称。
字符串（String）类型。
ip 服务器IP地址。
字符串（String）类型。
status 主机状态， 0：不正常，1：正常。
整型（Integer）类型。
haEnable 主机HA状态。
整型（Integer）类型。0表示主机未在HA中，1表示主机在HA中。
