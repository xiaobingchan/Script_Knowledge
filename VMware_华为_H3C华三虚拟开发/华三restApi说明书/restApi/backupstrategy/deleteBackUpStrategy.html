H3C CAS云管理平台

删除备份策略

删除指定的备份策略。

接口 URI

/cas/casrs/backupStrategy/delBackupStrategy?id={id}&name={name}

访问方法

HTTP 方法
DELETE
消息体
无
元素含义
路径参数（Path Parameter）
id 备份策略 ID。
长整数（long）类型，必须。
name 备份策略名称。
字符串（String）类型，可选。

返回结果

状态码
  • 204 No Content：成功。

应用举例

删除ID为”2“的备份策略。

请求
DELETE http://192.168.0.4:8080/cas/casrs/backupStrategy/delBackupStrategy?id=2
accept: application/xml
回应
HTTP/1.1 204 No Content
Date: Wed, 17 Sep 2014 02:45:06 GMT
Server: H3C-CVM

无消息体
