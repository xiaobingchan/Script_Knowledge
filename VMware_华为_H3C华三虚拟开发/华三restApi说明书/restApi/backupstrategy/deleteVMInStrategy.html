解除备份策略与虚拟机的关系

解除指定备份策略与指定ID虚拟机的关联关系。

接口 URI

/cas/casrs/backupStrategy/delBackupStgVm?backupId={backupId}&vmId={vmId}

访问方法

HTTP 方法
DELETE
消息体
无
元素含义
路径参数（Path Parameter）
backupId 备份策略 ID。
长整数（long）类型，必须。
vmId 虚拟机 ID。
长整数（long）类型，必须。

返回结果

状态码
  • 204 No Content：成功。

应用举例

删除ID为”4“的备份策略中的ID为”1“的虚拟机。

请求
DELETE http://192.168.10.80:8080/cas/casrs/backupStrategy/delBackupStgVm?backupId=4&&vmId=1
accept: application/xml
回应
HTTP/1.1 204 No Content
Date: Wed, 17 Sep 2014 02:45:06 GMT
Server: H3C-CVM

无消息体
