H3C CAS云管理平台

查询备份策略中虚拟机列表

查询指定ID备份策略的虚拟机列表。

接口 URI

/cas/casrs/backupStrategy/backupStrategyVms/{id}

访问方法

HTTP 方法
GET
消息体
无
元素含义
路径参数（Path Parameter）
id 备份策略 ID。
长整数（long）类型，必须。

返回结果

状态码
  • 200 OK：成功。
回应
<?xml version="1.0" encoding="UTF-8"?>
<list>
  <backupStrategyVm>
    <id>9</id>
    <domainId>313</domainId>
    <domainName>test_001</domainName>
    <domainDesc/>
    <title>test_001</title>
  </backupStrategyVm>
  <backupStrategyVm>
    <id>10</id>
    <domainId>312</domainId>
    <domainName>test_002</domainName>
    <domainDesc/>
    <title>test_002</title>
  </backupStrategyVm>
</list>
元素含义
backupStrategyVm各子元素
id 快照策略-虚拟机表ID。
长整型（Long）类型。
domainId 虚拟机ID。
长整型（Long）类型。
domainName 虚拟机名称。
字符串（String）类型。
domainDesc 虚拟机描述。
字符串（String）类型。
title 虚拟机显示名称。
字符串（String）类型。

应用举例

查询ID为”4“的备份策略的虚拟机列表。

请求
GET http://192.168.10.80:8080/cas/casrs/backupStrategy/backupStrategyVms/4
accept: application/xml
