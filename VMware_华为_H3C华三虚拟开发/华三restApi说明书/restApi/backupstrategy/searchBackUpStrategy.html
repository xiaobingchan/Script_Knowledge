H3C CAS云管理平台

查询备份策略详细信息

查询指定ID备份策略的详细信息。

接口 URI

/cas/casrs/backupStrategy/backupStrategyInfo/{id}

访问方法

HTTP 方法
GET
消息体
无
元素含义
路径参数（Path Parameter）
id 备份策略 ID。
长整数（long）类型，必须。

返回结果

状态码
  • 200 OK：成功。
回应

  <targetAddr>192.168.0.1</targetAddr>
  <loginName>root</loginName>
  <password>root</password>
  <frequency>0</frequency>

  <incBackupFrequency>1</incBackupFrequency>

元素含义
backupStrategy各子元素
id 备份策略id。
长整型（Long）类型。
name 备份策略名称。
字符串（String）类型。
description 描述。
字符串（String）类型。
storeMode 备份类型：0：共享文件系统；1：远端服务器。
整型（Integer）类型。
storeLocation 共享文件位置。
字符串（String）类型。
targetAddr 目的的服务器IP地址
字符串（String）类型，当 storeMode 为 1：远端服务器 时 存在。
loginName 目的服务器用户名。
字符串（String）类型，当 storeMode 为 1：远端服务器 时 存在。
password 目的服务器密码。
字符串（String）类型，当 storeMode 为 1：远端服务器 时 存在。
frequency 备份策略的频率,0:每月，1：每周，2：每日。
整型（Integer）类型。
day 备份策略的日期。
整型（Integer）类型。
hour 备份策略的时间：时。
整型（Integer）类型。
minutes 备份策略的时间：分。
整型（Integer）类型。
state 备份策略是否生效 1：生效，0：无效。
整型（Integer）类型。
incBackupFlag 是否启用增量备份：不启用：0，启用增量备份：1，启用差异备份：2。
整型（Integer）类型。
incBackupFrequency 频率:，1：每周，2：每日，3：每小时。
整型（Integer）类型，当incBackupFlag为1：启用增量备份或2：启用差异备份时 存在。
incBackupDay 哪天：几日，或星期几，每天 。
整型（Integer）类型，当incBackupFlag为1：启用增量备份或2：启用差异备份时 存在。
incBackupHour 增量备份的时间：时。
整型（Integer）类型，当incBackupFlag为1：启用增量备份或2：启用差异备份时 存在。。
incBackupMinutes 增量备份的时间：分。
整型（Integer）类型，当incBackupFlag为1：启用增量备份或2：启用差异备份时 存在。
serverType 服务器类型，0：选择FTP，1:选择SCP。
整型（Integer）类型，当 storeMode 为 1：远端服务器 时 存在。
updateVMList 是否更新虚拟机列表。
布尔（boolean）类型。
backupThreshold 磁盘I/O阈值(MB)。
整型（Integer）类型。
keepTimes 全量备份保留个数。
整型（Integer）类型。
hourEnd 全量备份时间段截止小时。
整型（Integer）类型。
minutesEnd 全量备份时间段截止分。
整型（Integer）类型。
incHourEnd 增量备份时间段截止小时。
整型（Integer）类型，当incBackupFlag为1：启用增量备份或2：启用差异备份时 存在。
incMinutesEnd 增量备份时间段截止分。
整型（Integer）类型，当incBackupFlag为1：启用增量备份或2：启用差异备份时 存在。
vmList 备份策略的虚拟机信息。
集合（List）类型。
vmList各子元素
domainId 虚拟机ID。
长整型（Long）类型，必须。
