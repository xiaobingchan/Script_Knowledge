H3C CAS云管理平台

增加备份策略

根据指定的参数增加备份策略。

接口 URI

/cas/casrs/backupStrategy/addBackupStrategy

访问方法

HTTP 方法
POST
消息体
<backupStrategy>
    <name>名称</name>
    <description>description</description>
    <storeMode>1</storeMode>
    <storeLocation>/vms/store</storeLocation>
    <targetAddr>192.168.0.1</targetAddr>
    <loginName>root</loginName>
    <password>root</password>
    <frequency>0</frequency>
    <day>5</day>
    <hour>7</hour>
    <minutes>0</minutes>
    <state>0</state>
    <incBackupFlag>2</incBackupFlag>
    <incBackupFrequency>1</incBackupFrequency>
    <incBackupDay>3</incBackupDay>
    <incBackupHour>7</incBackupHour>
    <incBackupMinutes>0</incBackupMinutes>
    <serverType>0</serverType>
    <updateVMList>true</updateVMList>
    <backupThreshold>10</backupThreshold>
    <keepTimes>5</keepTimes>
    <hourEnd>8</hourEnd>
    <minutesEnd>0</minutesEnd>
    <incHourEnd>8</incHourEnd>
    <incMinutesEnd>10</incMinutesEnd>
    <vmList>
        <domainId>313</domainId>
    </vmList>
    <vmList>
        <domainId>312</domainId>
    </vmList>
</backupStrategy>
元素含义
backupStrategy各子元素
name 备份策略名称，名称的最大输入长度为32个字符，且只允许输入汉字、字母、数字、减号、下划线及点。
字符串（String）类型，必须。
description 描述，最大输入长度为120个字符。
字符串（String）类型，可选项。
storeMode 备份类型：0：共享文件系统；1：远端服务器。
整型（Integer）类型，必须。
storeLocation 共享文件位置。
字符串（String）类型，必须。
targetAddr 目的的服务器IP地址
字符串（String）类型，当 storeMode 为 1：远端服务器 时 必须。
loginName 目的服务器用户名。
字符串（String）类型，当 storeMode 为 1：远端服务器 时 必须。
password 目的服务器密码。
字符串（String）类型，当 storeMode 为 1：远端服务器 时 必须。
frequency 备份策略的频率,0:每月，1：每周，2：每日。
整型（Integer）类型，必须。
day 备份策略的日期。
整型（Integer）类型，必须。
hour 备份策略的时间：时。
整型（Integer）类型，必须。
minutes 备份策略的时间：分。
整型（Integer）类型，必须。
state 备份策略是否生效 1：生效，0：无效。
整型（Integer）类型，必须。
incBackupFlag 是否启用增量备份：不启用：0，启用增量备份：1，启用差异备份：2。
整型（Integer）类型，必须。
incBackupFrequency 频率:，1：每周，2：每日，3：每小时。
整型（Integer）类型，当incBackupFlag为1：启用增量备份或2：启用差异备份时必须。
incBackupDay 哪天：几日，或星期几，每天 。
整型（Integer）类型，当incBackupFlag为1：启用增量备份或2：启用差异备份时必须。
incBackupHour 增量备份的时间：时。
整型（Integer）类型，当incBackupFlag为1：启用增量备份或2：启用差异备份时必须。。
incBackupMinutes 增量备份的时间：分。
整型（Integer）类型，当incBackupFlag为1：启用增量备份或2：启用差异备份时必须。
serverType 服务器类型，0：选择FTP，1:选择SCP。
整型（Integer）类型，当 storeMode 为 1：远端服务器 时 必须。
updateVMList 是否更新虚拟机列表。
布尔（boolean）类型，必须，默认为true。
backupThreshold 磁盘I/O阈值(MB)。
整型（Integer）类型，必须。
keepTimes 全量备份保留个数。
整型（Integer）类型，可选。
hourEnd 全量备份时间段截止小时。
整型（Integer）类型，必须。
minutesEnd 全量备份时间段截止分。
整型（Integer）类型，必须。
incHourEnd 增量备份时间段截止小时。
整型（Integer）类型，当incBackupFlag为1：启用增量备份或2：启用差异备份时必须。
incMinutesEnd 增量备份时间段截止分。
整型（Integer）类型，当incBackupFlag为1：启用增量备份或2：启用差异备份时必须。
vmList 备份策略的虚拟机信息。
集合（List）类型。
vmList各子元素
domainId 虚拟机ID。
长整型（Long）类型，必须。

返回结果

状态码
  • HTTP/1.1 204 No Content：成功。
  • 409(Conflict)：失败。
回应
无

    <storeLocation>/vms/store</storeLocation>

    <loginName>root</loginName>
    <password>root</password>
    <frequency>0</frequency>
    <day>5</day>
    <hour>7</hour>
    <minutes>0</minutes>
    <state>0</state>
    <incBackupFlag>2</incBackupFlag>
    <incBackupFrequency>1</incBackupFrequency>
    <incBackupDay>3</incBackupDay>
    <incBackupHour>7</incBackupHour>
    <incBackupMinutes>0</incBackupMinutes>
    <serverType>0</serverType>
    <updateVMList>true</updateVMList>
    <backupThreshold>10</backupThreshold>
    <hourEnd>8</hourEnd>
    <minutesEnd>0</minutesEnd>
    <incHourEnd>8</incHourEnd>
    <incMinutesEnd>10</incMinutesEnd>
    <vmList>
        <domainId>313</domainId>
    </vmList>
    <vmList>
        <domainId>312</domainId>
    </vmList>
</backupStrategy>
回应
无
