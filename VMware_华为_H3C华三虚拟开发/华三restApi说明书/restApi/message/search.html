H3C CAS云管理平台

查询任务详细信息

根据任务id查询任务详细信息。

接口 URI

/cas/casrs/message/{id}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 任务id 。
长整数（long）类型。必须。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
消息体
<?xml version="1.0" encoding="UTF-8"?>
<taskMsg>
  <msgId>1393831560400</msgId>
  <name>删除虚拟机“vm110-1”。</name>
  <targetId>-1</targetId>
  <targetName>vm110-1</targetName>
  <detail>删除虚拟机“vm110-1”成功。</detail>
  <completed>true</completed>
  <result>0</result>
  <progress>100</progress>
  <failMsg>删除虚拟机“vm110-1”成功。</failMsg>
  <eventType>21</eventType>
</taskMsg>
元素含义
host各子元素
msgId 任务 ID。
长整数（long）类型。
name 任务名称描述
字符串（String）类型。
targetId 操作目标ID
长整数（long）类型。
targetName 操作目标名称。
字符串（String）类型。
detail 任务详细信息。
字符串（String）类型。
completed 任务是否已完成
枚举类型:true;false。
result 执行结果 0:成功 1:失败 2:部分成功。
枚举（Integer）类型。
failMsg 失败信息(当任务成功时内容为任务信息)。
字符串（String）类型。
eventType 任务类型。
整数（int）类型。
 

应用举例

查看任务ID为1393831560400的任务详细信息。

请求
GET http://localhost:8080/cas/casrs/message/1393831560400
accept: application/xml
...

Content-Type: application/xml
...
