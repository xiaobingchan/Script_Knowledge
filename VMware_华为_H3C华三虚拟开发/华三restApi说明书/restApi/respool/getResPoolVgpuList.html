H3C CAS云管理平台

查询CAS系统中资源池下的vGPU资源列表

根据资源池ID查询CAS系统中资源池下的vGPU资源列表。

接口 URI

/cas/casrs/resPool/queryResPoolVgpuList?resPoolId={resPoolId}&offset={offset}&limit={limit}

访问方法

HTTP 方法
GET
参数
查询参数（Query Parameter）
resPoolId 资源池id 。
长整数（long）类型。必须。
offset 查询起始位置 。
整数（int）类型。 可选。
limit 查询记录数 。
整数（int）类型。 可选。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Set-Cookie: JSESSIONID=949B37D4A682C9E03AEBEE02B167E78B; Path=/cas
Total: 1
Content-Type: application/xml
Content-Length: 898
Date: Sat, 03 Feb 2018 08:48:43 GMT
Server: H3C-CVM
消息体
<?xml version="1.0" encoding="UTF-8"?><list>
  <vgpu>
    <bus>0000:05:00.0</bus>
    <name>GRID M60-0Q</name>
    <type>nvidia-12</type>
    <heads>2</heads>
    <framebuffer>512M</framebuffer>
    <maxResolution>2560x1600</maxResolution>
    <maxInstance>16</maxInstance>
    <uuid>c2177883-f1bb-47f0-914d-32a22e3a8804<uuid>
    <clusterId>1</clusterId>
    <hostId>13</hostId>
    <hostName>cvk2059</hostName>
    <isUsed>1</isUsed>
    <title>win7-pt2</title>
  </vgpu>
</list>
元素含义
vgpu各子元素
bus GPU槽号。
字符串（String）类型。
clusterId 资源所在的集群ID。
长整数（long）类型。
hostId 资源所在的主机ID。
长整数（long）类型。
hostName 资源所在的主机名称。
字符串（String）类型。
type vGPU的类型。
字符串（String）类型。
name vGPU的名称。
字符串（String）类型。
maxInstance vGPU的最大虚拟个数。
整数（int）类型。
framebuffer vGPU的帧缓存。
字符串（String）类型。
maxResolution vGPU的最大分辨率。
字符串（String）类型。
heads vGPU的显示器接口数。
整数（int）类型。
uuid vGPU的UUID唯一标识符。
字符串（String）类型。
isUsed vGPU是否被使用。
整数（int）类型。枚举值：0 空闲 1 使用中。
title 使用此vGPU的虚拟机显示名称。若没有被使用则无。
字符串（String）类型。
