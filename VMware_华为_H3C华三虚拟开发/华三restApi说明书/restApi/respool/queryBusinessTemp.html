H3C CAS云管理平台

查询业务模板详细信息

根据业务模板ID查询业务模板详细信息 。

接口 URI

/cas/casrs/resPool/temp/{id}

访问方法

HTTP 方法
GET
消息体
无
元素含义
路径参数（Path Parameter）
id 业务模板 ID。
长整数（long）类型，必须。
 

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Cache-Control: no-cache, no-store, max-age=0, must-revalidate
Pragma: no-cache
Expires: 0
X-XSS-Protection: 1; mode=block
X-Frame-Options: DENY
X-Content-Type-Options: nosniff
Content-Type: application/xml
Content-Length: 274
Date: Sat, 24 Feb 2018 09:43:51 GMT
Server: CVM
消息体
<?xml version="1.0" encoding="UTF-8"?><businessTem>
  <id>1</id>
  <name>非编业务</name>
  <description/>
  <priority>0</priority>
  <proportion>99</proportion>
  <command>ww</command>
  <commandResult>www</commandResult>
  <handleMode>0</handleMode>
</businessTem>
元素含义
businessTem各子元素
id 业务模板 ID。
长整型（Long）类型。
name 业务模板名称。
字符串（String）类型。
description 业务模板描述。
字符串（String）类型。
priority 优先级。 0：高，2：低
整型（Integer）类型。
proportion 资源分配比例1~100。
整型（Integer）类型。
command 停止业务命令。
字符串（String）类型。
commandResult 命令返回结果。
字符串（String）类型。
handleMode 停止业务失败，处理方式。 0：强制关闭虚拟机，1：查找下一个低优先级虚拟机
整型（Integer）类型。
