H3C CAS云管理平台

查询资源池下资源使用历史列表

根据资源池ID查询资源使用历史列表。

接口 URI

/cas/casrs/resPool/history?resPoolId={resPoolId}&offset={offset}&limit={limit}

访问方法

HTTP 方法
GET
参数

查询参数（Query Parameter）
resPoolId 资源池id 。
长整数（long）类型。必须。
offset 查询起始位置 。
整数（int）类型。 可选。
limit 查询记录数 。
整数（int）类型。 可选。

元素含义
resPool各子元素
id 资源使用历史 ID。
长整型（Long）类型。
bus 槽位号。
字符串（String）类型。
producers 厂商。
字符串（String）类型。
hostId 主机id。
长整型（Long）类型。
hostName 主机名称。
字符串（String）类型。
startTime 资源开始使用时间。
日期（Date）类型。
endTime 资源结束使用时间。
日期（Date）类型。
type 虚拟GPU切分类型。
字符串（String）类型。
name 虚拟GPU切分类型的名称。
字符串（String）类型。
uuid vGPU UUID。
字符串（String）类型。
heads 显示器接口数。
整型（Integer）类型。
framebuffer 帧缓冲区。
字符串（String）类型。
maxResolution 最大分辨率。
字符串（String）类型。
maxInstance 最大虚拟个数。
整型（Integer）类型。
title 使用资源的虚拟机显示名称。
字符串（String）类型。
