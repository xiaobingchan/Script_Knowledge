对物理GPU进行切分操作，创建vGPU

根据主机ID和槽位号对物理GPU进行切分操作，创建vGPU。

接口 URI

/cas/casrs/resPool/addVgpu?hostId={hostId}&bus={bus}&type={type}

访问方法

HTTP 方法
POST
参数

查询参数（Query Parameter）
hostId

主机id。

长整数（Long）类型。必须。
bus 物理GPU槽位号 。
字符串（String）类型。必须。
type 切分类型 。
字符串（String）类型。必须。

返回结果

状态码
  • 200 ：成功。
 
消息体
<?xml version="1.0" encoding="UTF-8"?>
<taskMsg>
  <msgId>1410943942346</msgId>
  <completed>false</completed>
  <progress>0</progress>
  <eventType>0</eventType>
</taskMsg>
元素含义
 
返回切分GPU任务id，可根据任务id查询任务进度。

应用举例

1.在ID为1的主机上，切分槽位号为0000:05:00.0（需做URL Encode）的物理GPU，切分类型为nvidia-11。

请求
POST http://localhost:8080/cas/casrs/resPool/addVgpu?hostId=1&bus=0000%3a05%3a00.0&type=nvidia-11
accept: application/xml
