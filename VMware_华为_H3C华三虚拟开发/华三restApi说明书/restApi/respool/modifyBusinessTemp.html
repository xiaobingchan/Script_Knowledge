修改业务模板

修改业务模板。

接口 URI

/cas/casrs/resPool/temp/modify

访问方法

HTTP 方法
PUT
消息体
<businessTem>
    <id>1</id>
    <description>test</description>
    <priority>2</priority>
    <proportion>1</proportion>
</businessTem>
元素含义
businessTem各子元素
id 业务模板ID。

长整形（Long）类型，必须。
description 描述信息。
字符串（String）类型。
priority 优先级：0表示高，2表示低。
整数（int）类型，必须。
proportion 资源分配比例。
整数（int）类型，必须。
command 停止业务命令
字符串（String）类型。
commandResult 停止业务命令返回果
字符串（String）类型。
handleMode 停止业务失败，处理方式 0：关闭虚拟机；1：查找下一个低优先级虚拟机。
整数（int）类型。

返回结果

状态码
  • 204 (No Content)：成功。
消息头

HTTP/1.1 204 No Content

 

应用举例

1.修改业务模板。

请求
PUT http://localhost:8080/cas/casrs/resPool/temp/modify
<businessTem>
    <id>1</id>
    <description>test</description>
    <priority>2/priority>
    <proportion>1</proportion>
</businessTem>
回应
HTTP/1.1 204 No Content
Content-Type: application/xml
...
无消息体
