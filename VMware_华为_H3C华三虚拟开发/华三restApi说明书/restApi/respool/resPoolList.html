H3C CAS云管理平台

查询CAS系统中的资源池列表

根据集群ID查询CAS系统中的资源池列表。

接口 URI

/cas/casrs/resPool/queryResPool?clusterId={clusterId}&offset={offset}&limit={limit}

访问方法

HTTP 方法
GET
参数

查询参数（Query Parameter）
clusterId 集群id 。
长整数（long）类型。必须。
offset 查询起始位置 。
整数（int）类型。 可选。
limit 查询记录数 。
整数（int）类型。 可选。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Total: 2
Content-Type: application/xml
Content-Length: 623
Date: Sat, 03 Feb 2018 08:33:01 GMT
Server: H3C-CVM
消息体
<?xml version="1.0" encoding="UTF-8"?><list>
  <resPool>
    <id>15</id>
    <name>t1</name>
    <type>1</type>    
    <clusterId>1</clusterId>
    <num>4</num>
    <vmNum>7</vmNum>
  </resPool>
  <resPool>
    <id>22</id>
    <name>a1</name>
    <type>2</type>
    <vgpuType>GRID M60-0B</vgpuType>
    <clusterId>1</clusterId>
    <num>1</num>
    <vmNum>0</vmNum>
  </resPool>
</list>
元素含义
resPool各子元素
id 资源池 ID。
长整型（Long）类型。
name 资源池名称。
字符串（String）类型。
type 资源类型。
整型（Integer）类型。枚举值：1 GPU 2 vGPU
clusterId 集群id。
长整型（Long）类型。
num 资源数量。
整型（Integer）类型。
vmNum 虚拟机数量。
整型（Integer）类型。
vgpuType vGPU切分类型。
字符串（String）类型。
