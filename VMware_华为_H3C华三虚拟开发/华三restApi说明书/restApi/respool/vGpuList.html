查询GPU下的vGPU资源列表

查询GPU下的vGPU资源列表。

接口 URI

/cas/casrs/resPool/vGpuList?hostId={hostId}&bus={bus}

访问方法

HTTP 方法
GET
参数

查询参数（Query Parameter）
hostId 主机id 。
长整数（long）类型。必须。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Total: 2
Content-Type: application/xml
Content-Length: 623
Date: Sat, 03 Feb 2018 08:33:01 GMT
Server: H3C-CVM
消息体
<?xml version="1.0" encoding="UTF-8"?>
  <vgpuConfig>
    <bus>0000:05:00.0</bus>
    <typeInfo>
        <name>GRID M60-0Q</name>
        <type>nvidia-12</type>     
        <availableInstances>16</availableInstances>     
        <heads>2</heads>
        <framebuffer>512M</framebuffer>     
        <maxResolution>2560x1600</maxResolution>      
        <maxInstance>16</maxInstance> 
    </typeInfo>
    <vgpu>a40c5370-0718-11e8-9558-60da833da413</vgpu> 
    <vgpu>397ace56-43c9-41dd-bbf7-27b583ec123a</vgpu>
 </vgpuConfig>
元素含义
vgpuConfig各子元素
bus 物理GPU槽位号。
字符串（String）类型。
vgpu 切分的vgpu的唯一标识。
字符串（String）类型。
typeInfo 支持切分成的虚拟GPU类型信息。
自定义类型。
type_info各子元素
name 类型名称。
字符串（String）类型。
type 类型。
字符串（String）类型。
availableInstances 还可以创建多少个实例。
整型（Integer）类型。
heads 显示器接口数。
整型（Integer）类型。
framebuffer 帧缓冲区。
字符串（String）类型。
maxResolution 最大分辨率。
字符串（String）类型。
maxInstance 最大虚拟个数。
整型（Integer）类型。
