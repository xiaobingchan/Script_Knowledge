H3C CAS云管理平台

查询主机上的物理GPU列表

根据主机ID查询主机上的物理GPU列表。

接口 URI

/cas/casrs/resPool/hostGpuList?hostId={hostId}

访问方法

HTTP 方法
GET
参数

查询参数（Query Parameter）
hostId 主机id 。
长整数（long）类型。必须。

返回结果

元素含义
gpu各子元素
bus 物理GPU槽位号。
字符串（String）类型。
productor 厂家名称。
字符串（String）类型。
model GPU型号。
字符串（String）类型。
vgpuEnable 是否已做切分，0：未做切分；1：已做切分。
整型（Integer）类型。
typeInfo

支持切分成的虚拟GPU类型信息。

自定义类型。
typeInfo各子元素
name 类型名称。
字符串（String）类型。
type 类型。
字符串（String）类型。
availableInstances 还可以创建多少个实例。
整型（Integer）类型。
heads 显示器接口数。
整型（Integer）类型。
framebuffer

帧缓冲区。

字符串（String）类型。
maxResolution 最大分辨率。
字符串（String）类型。
maxInstance 最大虚拟个数。
整型（Integer）类型。
