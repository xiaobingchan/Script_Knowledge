查询CAS系统中所有业务模板列表

查询CAS系统中所有的业务模板列表。

接口 URI

/cas/casrs/resPool/queryBusinessTem?offset={offset}&limit={limit}

访问方法

HTTP 方法
GET
参数

查询参数（Query Parameter）
offset 返回的第一条记录在总结果集中的位置。
整数（int）类型，非必须。
limit 返回记录总数。
整数（int）类型，非必须。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Total: 0
Content-Type: application/xml
Content-Length: 562
Date: Sat, 03 Feb 2018 08:33:01 GMT
Server: H3C-CVM
消息体
<?xml version="1.0" encoding="UTF-8"?><list>
  <businessTem>
    <id>11</id>
    <name>hight</name>
    <priority>0</priority>
    <proportion>60</proportion>
    <command>ls</command>
    <commandResult>0</commandResult>
    <handleMode>0</handleMode>
  </businessTem>
  <businessTem>
    <id>19</id>
    <name>ttt</name>
    <description>wedasd</description>
    <priority>0</priority>
    <proportion>5</proportion>
    <handleMode>1</handleMode>
  </businessTem>
</list>
元素含义
businessTem各子元素
id 业务模板 ID。
长整型（Long）类型。
name 业务模板名称。
字符串（String）类型。
description 业务模板描述。
字符串（String）类型。
priority 优先级。 0：高，2：低
整型（Integer）类型。
proportion 资源分配比例1~100。
整型（Integer）类型。
command 停止业务命令。
字符串（String）类型。
commandResult 命令返回结果。
字符串（String）类型。
handleMode 停止业务失败，处理方式。 0：强制关闭虚拟机，1：查找下一个低优先级虚拟机
整型（Integer）类型。
