H3C CAS云管理平台

查询集群下空闲GPU资源列表

查询集群下空闲GPU资源列表。

接口 URI

/cas/casrs/resPool/queryGpuList?clusterId={clusterId}&id={id}

访问方法

HTTP 方法
GET
参数

查询参数（Query Parameter）
clusterId 集群id 。
长整数（long）类型。必须。
id 资源池id 。当修改资源池时查询集群下空闲GPU需要此项。
长整数（long）类型。可选。

状态码

元素含义
gpu各子元素
bus GPU槽号。
字符串（String）类型。
producers 设备产商。
字符串（String）类型。
clusterId 资源所在的集群ID。
长整数（long）类型。
hostId 资源所在的主机ID。
长整数（long）类型。
hostName 资源所在的主机名称。
字符串（String）类型。
