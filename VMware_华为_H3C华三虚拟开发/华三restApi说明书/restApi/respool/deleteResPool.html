H3C CAS云管理平台

删除资源池

根据资源池ID删除资源池。

接口 URI

/cas/casrs/resPool/delete?id={id}

访问方法

HTTP 方法
DELETE
参数

查询参数（Query Parameter）
id 资源池id 。
长整数（long）类型。必须。

返回结果

状态码
  • 204 (No Content)：成功。

应用举例

1.删除 CAS系统中ID为1的资源池。

请求
DELETE http://localhost:8080/cas/casrs/resPool/delete?id=1
accept: application/xml
...
回应
HTTP/1.1 204 No Content
Server: Apache-Coyote/1.1
Date: Sat, 03 Feb 2018 09:00:20 GMT
无消息体
