查询资源池详细信息

根据资源池ID查询资源池详细信息 。

接口 URI

/cas/casrs/resPool/{id}

访问方法

HTTP 方法
GET
消息体
无
元素含义
路径参数（Path Parameter）
id 资源池ID。
长整数（long）类型，必须。

元素含义
resPool各子元素
id
资源池 ID。
长整型（Long）类型。
name
资源池名称。
字符串（String）类型。
type 资源类型。
整型（Integer）类型。枚举值：1 GPU 2 vGPU
clusterId 集群id。
长整型（Long）类型。
resPoolDevice各子元素
id
资源池设备 ID。
长整型（Long）类型。
bus GPU槽号。
字符串（String）类型。
producers 设备厂商。
字符串（String）类型。
clusterId 资源所在的集群ID。
长整数（long）类型。
hostId 资源所在的主机ID。
长整数（long）类型。
hostName 资源所在的主机名称。
字符串（String）类型。
type vGPU的类型。
字符串（String）类型。
name vGPU的名称。
字符串（String）类型。
maxInstance vGPU的最大虚拟个数。
整数（int）类型。
framebuffer vGPU的帧缓存。
字符串（String）类型。
maxResolution vGPU的最大分辨率。
字符串（String）类型。
heads vGPU的显示器接口数。
整数（int）类型。
uuid vGPU的UUID唯一标识符。
字符串（String）类型。
isUsed 资源设备使用标识。
整型（Integer）类型。枚举值：0 空闲 1 使用中
title 虚拟机显示名称。
字符串（String）类型。
resPoolVm各子元素
domainId
虚拟机 ID。
长整型（Long）类型。
businessTempId 业务模板ID。
长整型（Long）类型。
deviceDriver 驱动类型。
字符串（String）类型。枚举值：VFIO,KVM
title 虚拟机显示名称。
字符串（String）类型。
description 虚拟机描述信息。
字符串（String）类型。
hostId 虚拟机所在的主机ID。
长整型（Long）类型。
hostName 虚拟机所在的主机名称。
字符串（String）类型。
businessTempName 业务模板的名称。
字符串（String）类型。
isUsed
资源设备使用标识。
整型（Integer）类型。枚举值：0 空闲 1 使用中
