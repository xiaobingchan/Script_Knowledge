H3C CAS云管理平台

查询资源池下虚拟机列表

根据资源池ID查询资源池下虚拟机列表。

接口 URI

/cas/casrs/resPool/vm?resPoolId={resPoolId}&offset={offset}&limit={limit}

访问方法

HTTP 方法
GET
参数

查询参数（Query Parameter）
resPoolId 资源池id 。
长整数（long）类型。必须。
offset 查询起始位置 。
整数（int）类型。 可选。
limit 查询记录数 。
整数（int）类型。 可选。

返回结果

状态码
  • 200(OK)：成功。

元素含义
resPoolVm各子元素
domainId 业务虚拟机ID。
长整数（long）类型。
title 虚拟机显示名称。
字符串（String）类型。
description 虚拟机描述。
字符串（String）类型。
businessTempId 业务模板ID。
长整数（long）类型。
businessTempName 业务模板名称。
字符串（String）类型。
deviceDriver 驱动类型。
字符串（String）类型。枚举值：VFIO KVM。
hostId 主机ID。
长整数（long）类型。
hostName 主机名称。
字符串（String）类型。
isUsed 虚拟机是否正在使用资源池下资源。
整数（int）类型。枚举值：0 未使用 1 正在使用。
