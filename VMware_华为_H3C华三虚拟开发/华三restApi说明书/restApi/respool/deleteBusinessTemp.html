删除业务模板

删除业务模板。

接口 URI

/cas/casrs/resPool/temp/delete

访问方法

HTTP 方法
DELETE
参数
查询参数（Query Parameter）
id 业务模板id 。
长整数（long）类型。必须。

返回结果

状态码
  • 成功：
    204(No Content)
消息头

HTTP/1.1 204 No Content
Content-Type: application/xml

 

应用举例

1.删除id为1的业务模板。

请求
DELETE http://localhost:8080/cas/casrs/resPool/temp/delete?id=1
accept: application/xml
回应
HTTP/1.1 204 No Content
Server: Apache-Coyote/1.1
Set-Cookie: JSESSIONID=7276BAB7CE175185AB7CBD8D8E5684BF; Path=/cas
Date: Sat, 03 Feb 2018 09:00:20 GMT
无消息体
