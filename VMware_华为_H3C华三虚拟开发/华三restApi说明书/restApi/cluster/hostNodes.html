查询CAS系统中指定集群的主机列表

查询CAS系统中指定集群的主机列表。

接口 URI

/cas/casrs/cluster/hostNodes?clusterId={clusterId}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
clusterId 要查询的集群id 。
整数（int）类型。必选。
 

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Server: Apache-Coyote/1.1
Set-Cookie: JSESSIONID=CC6193BACB2F99D2D4CEC754B2676600; Path=/cas
Content-Type: application/xml
Content-Length: 141
Date: Fri, 02 Nov 2012 08:40:47 GMT
消息体
<?xml version="1.0" encoding="UTF-8"?>
<list>
	<host>
		<id>14</id>
		<user>root</user>
		<pwd>414414</pwd>
		<hostPoolId>1</hostPoolId>
		<clusterId>1</clusterId>
		<name>cp</name>
		<ip>192.168.10.210</ip>
		<status>1</status>
		<haEnable>1</haEnable>
	</host>
</list>
元素含义
cluster各子元素
id 主机 ID。
长整数（long）类型。
user 主机登录用户名。
字符串（String）类型。
pwd 主机登录密码。
字符串（String）类型。
hostPoolId 主机所在主机池ID。
整形（int）类型。
clusterId 主机所在集群ID。
整数（int）类型。
name 主机名。
字符串（String）类型。
ip 主机ip。
字符串（String）类型。
status 主机状态。
整数（int）类型。0表示不正常，1表示正常，2表示主机不在HA中，3表示维护状态。
haEnable 主机HA状态。
整数（int）类型。0表示主机未在HA中，1表示主机在HA中。
