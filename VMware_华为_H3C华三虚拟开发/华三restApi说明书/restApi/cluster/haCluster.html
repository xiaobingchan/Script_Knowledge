修改集群高可靠性集群

修改集群高可靠性集群。

接口 URI

/cas/casrs/cluster/editHa

访问方法

HTTP 方法
put
消息体 
<cluster>
    <id>1</id>
    <enableHA>1</enableHA>
    <priority>2</priority>
    <HaMinHost>1</HaMinHost>
</cluster>
参数
路径参数（Path Parameter）
id 集群ID 。
长整数（Long）类型。必须。
enableHA 是否启用HA 0:不启用HA 1:启用HA。
整数（Integer）类型。
priority 启动优先级 0:低级 1:中级 2:高级 。
整数（Integer）类型。
HaMinHost 最少提供ha服务主机数。
整数（Integer）类型。

返回结果

状态码
  • 204(No Content)：成功。
消息头
HTTP/1.1 204 No Content
Content-Type: application/xml
Date: Wed, 17 Sep 2014 00:48:42 GMT
Server: H3C-CVM
