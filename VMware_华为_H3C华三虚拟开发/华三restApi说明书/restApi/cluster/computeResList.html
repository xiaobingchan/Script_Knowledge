查询CAS系统中指定集群的计算资源

查询CAS系统中指定集群的计算资源。

接口 URI

/cas/casrs/cluster/computeResList?clusterIds={clusterIds}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
clusterIds 要查询的集群id列表 。
字符串（string）类型。必选。以‘,’分割，如：1,2,3。
 

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Server: Apache-Coyote/1.1
Set-Cookie: JSESSIONID=CC6193BACB2F99D2D4CEC754B2676600; Path=/cas
Content-Type: application/xml
Content-Length: 141
Date: Fri, 02 Nov 2012 08:40:47 GMT
消息体
<?xml version="1.0" encoding="UTF-8"?>
<list>
	<computeRes>
		<hostPoolId>1</hostPoolId>
		<hostPoolName>cp</hostPoolName>
		<clusterId>1</clusterId>
		<clusterName>fjc</clusterName>
		<enableHA>0</enableHA>
		<vmNum>4</vmNum>
		<runVmNum>0</runVmNum>
		<clusterMaxCpu>4</clusterMaxCpu>
		<clusterMaxMem>3859</clusterMaxMem>
	</computeRes>
	<computeRes>
		<hostPoolId>1</hostPoolId>
		<hostPoolName>cp</hostPoolName>
		<clusterId>2</clusterId>
		<clusterName>fjc_test</clusterName>
		<enableHA>1</enableHA>
		<vmNum>0</vmNum>
		<runVmNum>0</runVmNum>
	</computeRes>
</list>
元素含义
cluster各子元素
hostPoolId 主机池 ID。
长整数（long）类型。
clusterId 集群 ID。
长整数（long）类型。
hostPoolName 主机池名称。
字符串（String）类型。
clusterName 集群名称。
字符串（String）类型。
enableHA 是否启用高可靠性。
整数（int）类型。1表示启用，0表示不启用。
vmNum 虚拟机个数。
整数（int）类型。
runVmNum 运行的虚拟机个数。
整数（int）类型。
clusterMaxCpu 集群中cpu最大个数。
整数（int）类型。集群中不存在主机，该值可空。
clusterMaxMem 集群中内存的最大值。
整数（int）类型。单位GB。集群中不存在主机，该值可空。
