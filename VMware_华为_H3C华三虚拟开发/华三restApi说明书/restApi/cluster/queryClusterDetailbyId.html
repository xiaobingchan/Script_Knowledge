查询CAS系统中指定集群信息

查询CAS系统中指定集群信息。

接口 URI

/cas/casrs/cluster/{id}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 集群id 。
长整数（long）类型。必选

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Server: Apache-Coyote/1.1
Set-Cookie: JSESSIONID=CC6193BACB2F99D2D4CEC754B2676600; Path=/cas
Content-Type: application/xml
Content-Length: 141
Date: Fri, 02 Nov 2012 08:40:47 GMT
消息体
<?xml version="1.0" encoding="UTF-8"?>
<cluster>
	<id>2</id>
	<hostPoolId>1</hostPoolId>
	<name>fjc_test</name>
	<enableHA>1</enableHA>
	<priority>1</priority>
	<enableLB>1</enableLB>
	<persistTime>5</persistTime>
	<checkInterval>10</checkInterval>
	<enableIPM>1</enableIPM>
	<persistTimeIPM>5</persistTimeIPM>
	<ipmLowerMonitorId>5</ipmLowerMonitorId>
	<ipmLowerMonitorName>主机唤醒监控策略</ipmLowerMonitorName>
	<ipmUpperMonitorId>2</ipmUpperMonitorId>
	<ipmUpperMonitorName>资源扩展监控策略</ipmUpperMonitorName>
	<checkIntervalIPM>10</checkIntervalIPM>
	<operatorGroupId>0</operatorGroupId>
	<operatorGroupCode>00</operatorGroupCode>
	<lbMonitorId>1</lbMonitorId>
	<lbMonitorName>动态资源调度监控策略</lbMonitorName>
	<HaMinHost>1</HaMinHost>
</cluster>
元素含义
cluster各子元素
id 集群 ID。
长整数（long）类型。
hostPoolId 主机池 ID。
长整数（long）类型。
name 集群名称。
字符串（String）类型。
description 集群描述。
字符串（String）类型。
enableHA 是否启用高可靠性。
整数（int）类型。1表示启用，0表示不启用。
priority 启动优先级。
整数（int）类型。
enableLB 是否启用动态资源调度。
整数（int）类型。1表示启用，0表示不启用。
persistTime 启动动态资源调度时配置的持续时间。
整数（int）类型。单位分钟。
checkInterval 启动动态资源调度时配置的检查时间间隔。
整数（int）类型。单位分钟。
lbMonitorId 启用动态资源调度监控策略ID。
整数（int）类型。
lbMonitorName 启用动态资源调度监控策略名称。
字符串（string）类型。
enableIPM 是否启用电源智能管理。
整数（int）类型。1表示启用，0表示不启用。
persistTimeIPM 启用电源智能管理时配置的持续时间。
整数（int）类型。单位分钟。
checkIntervalIPM 启用电源智能管理时配置的检查时间间隔。
整数（int）类型。单位分钟。
ipmLowerMonitorId 启用电源智能管理主机回收策略ID。
整数（int）类型。
ipmLowerMonitorName 启用电源智能管理主机回收策略名称。
字符串（string）类型。
ipmUpperMonitorId 启用电源智能管理主机扩展策略ID。
整数（int）类型。
ipmUpperMonitorName 启用电源智能管理主机扩展策略名称。
字符串（string）类型。
operatorGroupId 操作员分组ID。
整数（int）类型。
operatorGroupCode 操作员分组编码。
字符串（string）类型。
HaMinHost HA生效的最小节点数。
整数（int）类型。
