查询CAS系统中指定集群性能的topN主机列表

查询CAS系统中指定集群性能的topN主机列表。

接口 URI

/cas/casrs/cluster/hostTopN/{clusterId}/{type}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
clusterId 要查询的集群id 。
长整数（long）类型。必选。
type 要查询的性能类型 。
字符串（string）类型。必选。cpu表示查询cpu的利用率，memory表示查询内存利用率

元素含义
cluster各子元素
id 主机 ID。
长整数（long）类型。
name 主机名。

rate 查询的对应类型的利用率。
浮点（float）类型。
