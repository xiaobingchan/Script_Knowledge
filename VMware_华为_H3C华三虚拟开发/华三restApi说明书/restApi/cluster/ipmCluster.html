H3C CAS云管理平台

电源智能管理

电源智能管理。

接口 URI

/cas/casrs/cluster/editIpm

访问方法

HTTP 方法
PUT
消息体 
<cluster>
    <id>1</id>
    <enableIpm>1</enableIpm>
    <persistTimeIpm>5</persistTimeIpm>
    <checkIntervalIpm>10</checkIntervalIpm>
    <ipmLowerMonitorId>4</ipmLowerMonitorId>
    <ipmUpperMonitorId>4</ipmUpperMonitorId>
</cluster>
参数
路径参数（Path Parameter）
id 集群ID 。
长整数（Long）类型。必须。
enableIpm 是否启用电源智能管理 0:不启用电源智能管理 1:启用电源智能管理。
整数（Integer）类型。
persistTimeIpm 持续时间。
整数（Integer）类型。
checkIntervalIpm 检查间隔。
整数（Integer）类型。
ipmLowerMonitorId 主机回收监控策略ID。
长整数（Long）类型。
ipmUpperMonitorId 主机扩展监控策略ID。
长整数（Long）类型。
 

返回结果

状态码
  • 204(No Content)：成功。
消息头
HTTP/1.1 204 No Content
Content-Type: application/xml
Date: Wed, 17 Sep 2014 00:48:42 GMT
Server: H3C-CVM
