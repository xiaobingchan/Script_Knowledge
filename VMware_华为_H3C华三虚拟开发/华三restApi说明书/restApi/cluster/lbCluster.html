动态资源调度

动态资源调度。

接口 URI

/cas/casrs/cluster/editLb

访问方法

HTTP 方法
PUT
消息体 
<cluster>
    <id>1</id>
    <name>clu</name>
    <enableLB>1</enableLB>
    <persistTime>5</persistTime>
    <checkInterval>10</checkInterval>
    <lbMonitorId>4</lbMonitorId>
</cluster>
参数
路径参数（Path Parameter）
id 集群ID 。
长整数（Long）类型。必须。
enableLB 是否启用LB 0:不启用负载均衡 1:启用负载均衡。
整数（Integer）类型。
persistTime 持续时间。
整数（Integer）类型。
checkInterval 检查间隔。
整数（Integer）类型。
lbMonitorId 动态资源调度引用的监控策略ID。
长整数（Long）类型。
 

返回结果

状态码
  • 204(No Content)：成功。
消息头
HTTP/1.1 204 No Content
Content-Type: application/xml
Date: Wed, 17 Sep 2014 00:48:42 GMT
Server: H3C-CVM

 

应用举例

1.创建名称为“aa”的集群

请求

accept: application/xml

    <persistTime>5</persistTime>
    <checkInterval>10</checkInterval>

</cluster>
