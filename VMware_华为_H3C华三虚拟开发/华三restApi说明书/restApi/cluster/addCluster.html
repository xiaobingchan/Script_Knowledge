创建集群

创建集群。

接口 URI

/cas/casrs/cluster/add

访问方法

HTTP 方法
POST
消息体 
<cluster>
    <hostPoolId>1</hostPoolId>
    <name>clu</name>
    <description>cluxs</description>
    <enableHA>1</enableHA>
    <priority>2</priority>
    <enableLB>1</enableLB>
    <persistTime>5</persistTime>
    <checkInterval>10</checkInterval>
    <enableIPM>0</enableIPM>
    <persistTimeIPM>0</persistTimeIPM>
    <checkIntervalIPM>0</checkIntervalIPM>
    <lbMonitorId>4</lbMonitorId>
    <HaMinHost>1</HaMinHost>
</cluster>
参数
路径参数（Path Parameter）
hostPoolId 主机池ID 。
长整数（Long）类型。必须。
name 集群名称 。
字符串（String）类型。必须。
description 集群描述信息 。
字符串（String）类型。
enableHA 是否启用HA 0:不启用HA 1:启用HA。
整数（Integer）类型。
priority 启动优先级 0:低级 1:中级 2:高级 。
整数（Integer）类型。
enableLB 是否启用LB 0:不启用负载均衡 1:启用负载均衡。
整数（Integer）类型。
persistTime 持续时间。
整数（Integer）类型。
checkInterval 检查间隔。
整数（Integer）类型。
enableIPM 是否启用电源智能管理 0:不启用电源智能管理 1:启用电源智能管理。
整数（Integer）类型。
persistTimeIPM 持续时间。
整数（Integer）类型。
checkIntervalIPM 检查间隔。
整数（Integer）类型。
lbMonitorId 动态资源调度引用的监控策略ID。
长整数（Long）类型。
HaMinHost 最少提供ha服务主机数。
整数（Integer）类型。
 

返回结果

状态码
  • 204(No Content)：成功。
消息头
HTTP/1.1 204 No Content
Content-Type: application/xml
Date: Wed, 17 Sep 2014 00:48:42 GMT
Server: H3C-CVM
