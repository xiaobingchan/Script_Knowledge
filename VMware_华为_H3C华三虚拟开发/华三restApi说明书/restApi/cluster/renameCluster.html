集群重命名

集群重命名。

接口 URI

/cas/casrs/cluster//rename/{clusterId}/{newName}

访问方法

HTTP 方法
PUT
参数
路径参数（Query Parameter）
clusterId 集群id 。
长整数（Long）类型。必须，无缺省值。
newName 集群名称 。
字符串（String）类型。必须，无缺省值。

返回结果

状态码
  • 204(No Content)：成功。
消息头
HTTP/1.1 204 No Content
Content-Type: application/xml
Date: Wed, 17 Sep 2014 00:48:42 GMT
Server: H3C-CVM

 

应用举例

1.集群重命名为“aa”。

请求
POST http://localhost:8080/cas/casrs/cluster/rename/1/aa
accept: application/xml
回应
HTTP/1.1 204 No Content
Date: Wed, 17 Sep 2014 00:48:42 GMT
Server: H3C-CVM

无消息体
