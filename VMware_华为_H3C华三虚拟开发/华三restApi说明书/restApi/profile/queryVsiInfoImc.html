从IMC中查询VSI的网络ID，VSI的ID和版本的ID

从IMC中查询VSI的网络ID，VSI的ID和版本的ID。

接口 URI

/cas/casrs/profile/getVsiInfoImc

返回结果

状态码
  • 200(OK)：成功。
消息头

HTTP/1.1 200 OK
Total: 7
消息体
<?xml version="1.0" encoding="UTF-8"?>
<list>
  <vsiTypeGroup id="1">
    <name>acl</name>
    <vlanId>1</vlanId>
    <type>0</type>
    <typeName>00</typeName>
    <description>00</description>
    <link>00</link>
  </vsiTypeGroup>
</list>


元素含义
vsiTypeGroup各子元素
name 网络名称。
字符串（String）类型。
vlanId VLANID 。
整形（Integer）类型。
type 网络类型。0标识预定义，1标识自定义
长整形（Long）类型。
typeName 网络类型名称。
字符串（String）类型。
description 网络描述。
字符串（String）类型。
link 查询该网络详情链接。href表示访问链接。op表示连接访问方式。rel表示连接访问返回信息。
字符串（String）类型。
