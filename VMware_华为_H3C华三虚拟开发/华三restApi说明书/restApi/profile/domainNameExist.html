检查网络策略模板名称是否存在

判断网络策略模板名称是否存在。

接口 URI

/cas/casrs/profile/isExist/{name}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
name 网络策略模板名称 。
字符串（String）类型。必须，无缺省值。
 
 

返回结果

状态码
  • 200(OK)：成功。
消息头

HTTP/1.1 200 OK
result: true
Content-Type: application/xml
Transfer-Encoding: chunked
Date: Wed, 17 Sep 2014 01:17:17 GMT

消息体

true

元素含义

返回网络策略模板名称是否存在

 

应用举例

1.检查名称为Default的网络策略模板是否存在。

请求
POST http://localhost:8080/cas/casrs/profile/isExist/Default
accept: application/xml
回应
HTTP/1.1 200 OK
result: true
Content-Type: application/xml
Transfer-Encoding: chunked
Date: Wed, 17 Sep 2014 01:17:17 GMT
...
true
