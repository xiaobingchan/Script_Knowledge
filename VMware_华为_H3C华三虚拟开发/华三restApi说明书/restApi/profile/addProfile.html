创建网络策略模板

创建网络策略模板。

接口 URI

/cas/casrs/profile

访问方法

HTTP 方法
POST
消息体
<portProfile>
	<name>w</name>
	<description>test</description>
	<vlanId>1</vlanId>
	<inbound>1</inbound>
	<inAvgBandwidth>2000</inAvgBandwidth>
	<inPeakBandwidth>2000</inPeakBandwidth>
	<inBurstSize>1000</inBurstSize>
	<outbound>1</outbound>
	<outAvgBandwidth>1000</outAvgBandwidth>
	<outPeakBandWidth>1000</outPeakBandWidth>
	<outBurstSize>2000</outBurstSize>
	<enableVsi>1</enableVsi>
	<vsiMngId>2001:0db8:85a3:08d3:1319:8a2e:0370:7344</vsiMngId>
	<vsiTypeId>123542</vsiTypeId>
	<vsiTypeVer>112</vsiTypeVer>
	<vsiTypeVerId>2</vsiTypeVerId>
	<vsiIdFormat>UUID</vsiIdFormat>
	<aclStrategyId>1</aclStrategyId>
	<aclName>example</aclName>
	<vnetPriority>0</vnetPriority>
</portProfile>
元素含义
portProfile各子元素
name 网络策略模板名称。
字符串（String）类型。
description 描述信息。
字符串（String）类型。
vlanId VLAN ID。
长整数（long）类型。
inbound 入方向流量限制：1表示限制，0表示不限制。
整数（int）类型。
inAvgBandwidth 入方向流量控制：平均带宽，单位Kbps
长整数（long）类型。
inBurstSize 入方向流量控制：突发缓冲，单位Kbps
长整数（long）类型。
outbound 出方向流量限制：1表示限制，0表示不限制。
整数（int）类型。
outAvgBandwidth 出方向流量控制：平均带宽，单位Kbps
长整数（long）类型。
outBurstSize 出方向流量控制：突发缓冲，单位Kbps
长整数（long）类型。
enableVsi 是否启用vsi：1表示启用，0表示不启用
整数（int）类型。
vsiMngId VSI管理ID，为合法的IPv6地址
字符串（String）类型。
vsiTypeId VSI类型ID
长整数（long）类型。
vsiTypeVer VSI类型版本
整数（long）类型。
vsiTypeVerId VSI类型版本ID
整数（long）类型。
vsiIdFormat VSI ID格式
枚举类型：UUID
aclStrategyId 绑定的ACL ID
长整数（long）类型。
aclName 绑定的ACL名称
字符串（String）类型。
vnetPriority 网络优先级：0表示低，1表示中，2表示高。
整数（int）类型。

状态码
  • 204 (No Content)：成功。
