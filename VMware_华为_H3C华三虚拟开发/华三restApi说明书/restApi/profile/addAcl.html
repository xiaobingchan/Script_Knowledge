H3C CAS云管理平台

增加ACL策略

增加ACL策略。

接口 URI

/cas/casrs/acl

访问方法

HTTP 方法
POST
消息体
<?xml version="1.0" encoding="UTF-8"?>
<aclStrategy>
  <name>example-acl</name>
  <description>示例acl</description>
  <defaultAclAction>1</defaultAclAction>
  <defaultAclOutAction>1</defaultAclOutAction>
  <rule>
    <protocol>17</protocol>
    <srcIp>192.168.10.0</srcIp>
    <srcMask>255.255.255.0</srcMask>
    <srcPort>80</srcPort>
    <destIp>192.168.20.0</destIp>
    <destMask>255.255.255.0</destMask>
    <destPort>2336</destPort>
    <action>1</action>
    <priority>1</priority>
    <direction>1</direction>
  </rule>
  <type>3</type>
  <timeRangeEnabled>0</timeRangeEnabled>
</aclStrategy>

返回结果

状态码
  • 204(No Content)：成功。
消息头

HTTP/1.1 204 No Content
元素含义
aclStrategy各子元素
name ACL策略名。
字符串（String）类型。
description ACL策略描述。
字符串（String）类型。
defaultAclAction 入方向默认动作 (0:拒绝;1:允许)。
整数（Integer）类型。
defaultAclOutAction 出方向默认动作 (0:拒绝;1:允许)。
整数（Integer）类型。
type ACL类型，3：IPV4，2：二层ACL（基于MAC）。
整数（Integer）类型。
timeRangeEnabled 启用时间段，1启用，即ACL只在指定时间内生效，0，不启用。
整数（Integer）类型。
effectTimeRange ACL生效时间段，多个时间段以逗号分隔，时间段格式为"0-3,5-5,8-12"
字符串（String）类型。
createTime ACL策略创建时间。
时间（Date）类型。

rule各子元素
protocol ACL规则协议号码, ICMP为1,TCP为6,UDP为17，所有协议为65535。
整数（int）类型。
srcIp 源IP地址。
字符串（String）类型。
srcMask 源子网掩码。
字符串（String）类型。
srcPort 源端口。
整数（Integer）类型。
destIp 目的IP地址。
字符串（String）类型。
destMask 目的子网掩码。
字符串（String）类型。
destPort 目的端口。
整型（Integer）类型。
action ACL规则对报文采取的动作(0:拒绝;1:允许)。
整数（int）类型。
priority 优先级。
整数（int）类型。
