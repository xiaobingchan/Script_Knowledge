H3C CAS云管理平台

删除ACL策略

删除网络策略模板。

接口 URI

/cas/casrs/acl/{id}

访问方法

HTTP 方法
DELETE
参数
路径参数（Path Parameter）
id ACL策略id 。
长整数（long）类型。必须。

返回结果

状态码
  • 成功：
    204(No Content)
消息头

HTTP/1.1 204 No Content
Content-Type: application/xml

 

应用举例

1.删除id为3的ACL策略。

请求
DELETE http://192.168.10.3:8080/cas/casrs/acl/3
accept: application/xml
回应
HTTP/1.1 204 No Content
Server: Apache-Coyote/1.1
