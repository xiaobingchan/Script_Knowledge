查询CAS系统中网络策略模板列表

查询CAS系统中网络策略模板列表。

接口 URI

/cas/casrs/profile/

访问方法

HTTP 方法
GET
参数

无

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Server: Apache-Coyote/1.1
Set-Cookie: JSESSIONID=CC6193BACB2F99D2D4CEC754B2676600; Path=/cas
Content-Type: application/xml
Content-Length: 141
Date: Fri, 02 Nov 2012 08:40:47 GMT
消息体
<?xml version="1.0" encoding="UTF-8"?>
<portProfiles>
	<portProfile>
	<id>5</id>
	<name>w</name>
	<vlanId>0</vlanId>
	<inbound>1</inbound>
	<inAvgBandwidth>2000</inAvgBandwidth>
	<inPeakBandwidth>2000</inPeakBandwidth>
	<inBurstSize>1000</inBurstSize>
	<outbound>1</outbound>
	<outAvgBandwidth>1000</outAvgBandwidth>
	<outPeakBandWidth>1000</outPeakBandWidth>
	<outBurstSize>2000</outBurstSize>
	<enableVsi>0</enableVsi>
	<aclStrategyId>1</aclStrategyId>
	<aclName>example</aclName>
	</portProfile>
......
</portProfiles>
元素含义
portProfile各子元素
id 网络策略模板 ID。
长整数（long）类型。
name 网络策略模板名称。
字符串（String）类型。
description 描述信息。
字符串（String）类型。
vlanId VLAN ID。
长整数（long）类型。
inbound 入方向流量限制：1表示限制，0表示不限制。
整数（int）类型。
inAvgBandwidth 入方向流量控制：平均带宽，单位Kbps
长整数（long）类型。
inBurstSize 入方向流量控制：突发缓冲，单位Kbps
长整数（long）类型。
outbound 出方向流量限制：1表示限制，0表示不限制。
整数（int）类型。
outAvgBandwidth 出方向流量控制：平均带宽，单位Kbps
长整数（long）类型。
outBurstSize 出方向流量控制：突发缓冲，单位Kbps
长整数（long）类型。
aclStrategyId 绑定的ACL ID
长整数（long）类型。
aclName 绑定的ACL名称
字符串（String）类型。
