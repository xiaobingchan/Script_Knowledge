根据条件查询ACL 策略列表

根据条件查询ACL 策略列表。

接口 URI

/cas/casrs/acl?name={name}&offset={offset}&limit={limit}

访问方法

HTTP 方法
GET
参数
查询参数（Path Parameter）
name acl策略名称。
字符串（String）类型，非必须。
limit 返回记录总数。
整数（int）类型，非必须，缺省为 10。如果传入参数小于 0，则使用缺省值 10。
offset 返回的第一条记录在总结果集中的位置。
整数（int）类型，非必须，缺省为 0。如果传入参数小于 0，则使用缺省值 0。

返回结果

状态码
  • 200(OK)：成功。
消息头

HTTP/1.1 200 OK
消息体
<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<list>
  <aclStrategy id="1">
    <name>a</name>
    <defaultAclAction>1</defaultAclAction>
    <defaultAclOutAction>1</defaultAclOutAction>
    <createTime>2015-06-12T13:39:00+08:00</createTime>
    <rule id="1">
      <protocol>6</protocol>
      <action>1</action>
      <priority>1</priority>
      <direction>1</direction>
    </rule>
    <type>3</type>
    <timeRangeEnabled>0</timeRangeEnabled>
  </aclStrategy>
  <aclStrategy id="3">
    <name>example-acl</name>
    <description>示例acl</description>
    <defaultAclAction>1</defaultAclAction>
    <defaultAclOutAction>1</defaultAclOutAction>
    <type>3</type>
    <timeRangeEnabled>0</timeRangeEnabled>
    <effectTimeRange>4-4,7-7,10-10,12-14</effectTimeRange>
    <createTime>2015-08-05T10:45:37+08:00</createTime>
    <rule id="4">
      <protocol>17</protocol>
      <srcIp>192.168.10.0</srcIp>
      <srcMask>255.255.255.0</srcMask>
      <srcPort>80</srcPort>
      <destIp>192.168.20.0</destIp>
      <destMask>255.255.255.0</destMask>
      <destPort>2336</destPort>
      <action>1</action>
      <priority>1</priority>
      <direction>1</direction>
    </rule>
  </aclStrategy>
</list>
元素含义
aclStrategy各子元素
id ACL策略ID。
长整型（Long）类型。
name ACL策略名。
字符串（String）类型。
description ACL策略描述。
字符串（String）类型。
defaultAclAction 入方向默认动作 (0:拒绝;1:允许)。
整数（Integer）类型。
defaultAclOutAction 出方向默认动作 (0:拒绝;1:允许)。
整数（Integer）类型。
type ACL类型，3：IPV4，2：二层ACL（基于MAC）。
整数（Integer）类型。
timeRangeEnabled 启用时间段，1启用，即ACL只在指定时间内生效，0，不启用。
整数（Integer）类型。
effectTimeRange ACL生效时间段，多个时间段以逗号分隔，时间段格式为"0-3,5-5,8-12"
字符串（String）类型。
createTime ACL策略创建时间。
时间（Date）类型。
rule各子元素
protocol ACL规则协议号码, ICMP为1,TCP为6,UDP为17，所有协议为65535。
整数（int）类型。
srcIp 源IP地址。
字符串（String）类型。
srcMask 源子网掩码。
字符串（String）类型。
srcPort 源端口。
整数（Integer）类型。
destIp 目的IP地址。
字符串（String）类型。
destMask 目的子网掩码。
字符串（String）类型。
destPort 目的端口。
整型（Integer）类型。
action ACL规则对报文采取的动作(0:拒绝;1:允许)。
整数（int）类型。
priority 优先级。
整数（int）类型。
