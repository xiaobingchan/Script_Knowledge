查询告警信息列表

查询告警信息列表。

接口 URI

/cas/casrs/warnManage/warnManageList?offset={offset}&limit={limit}&eventTime_from={eventTime_from}&eventTime_to={eventTime_to}&state={state}&sortDir={sortDir}&sortField={sortField}

访问方法

HTTP 方法
GET
参数
查询参数（Query Parameter）
offset 查询起始位置 。
整数（int）类型。 必须。
limit 查询记录数 。
整数（int）类型。 必须。
eventTime_from 开始时间 。
字符串（String）类型。
eventTime_to 结束时间 。
字符串（String）类型。
state 告警状态 。
整数（int）类型。枚举值：1 确认 2 未确认
sortDir 排序顺序 。
字符串（String）类型。枚举值：Asc 顺序 Desc 逆序
sortField 排序元素 。
字符串（String）类型。枚举值：eventTime 告警时间

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Expires: Thu, 01 Jan 1970 00:00:00 GMT
Set-Cookie: JSESSIONID=1aw0joul8i38n;Path=/
Content-Type: application/xml
Content-Length: 444
Server: Jetty(6.1.x)
消息体
<?xml version="1.0" encoding="UTF-8"?>
<list>
  <event>
    <id>2</id>
    <state>1</state>
    <eventLevel>2</eventLevel>
    <eventType>2</eventType>
    <eventName>2</eventName>
    <eventTime>2013-11-15T11:08:10+08:00</eventTime>
    <eventDesc>192.168.0.8</eventDesc>
    <eventSrc>imc</eventDesc>
  </event>...
</list>
元素含义
event各子元素
id 告警管理id。
长整数（long）类型。
state 告警状态。
整数（int）类型。枚举值：1 确认 2 未确认
eventLevel 告警级别。
整数（int）类型。枚举值：1 紧急 2重要 3 次要 4 警告
eventType 告警类型。
整数（int）类型。枚举值：20 主机，30 虚拟机，40 集群， 51 虚拟机管理软件
eventName 告警名称。
字符串（String）类型。
eventTime 告警时间。
日期（Date）类型。
eventDesc 告警信息描述。
字符串（String）类型。
eventSrc 告警来源。
字符串（String）类型。
