删除指定存储池

删除指定主机下的指定存储池。

接口 URI

/cas/casrs/storage/delete

访问方法

HTTP 方法
DELETE
参数
查询参数（Query Parameter）
id 主机ID 。
长整数（Long）类型。必须，无缺省值。
poolName 存储池的名称 。
字符串（String）类型。必须，无缺省值。
hostName 主机的名称。
字符串（String）类型。必须，无缺省值。
type 存储池的类型。
字符串（String）类型。无缺省值。

返回结果

状态码
  • 204(No Content)：成功。
  • 409(Conflict)：失败。
消息头
HTTP/1.1 204 No Content
Server: Apache-Coyote/1.1
Date: Fri, 22 Nov 2013 07:45:07 GMT

应用举例

删除主机cvm28下的存储池localTest。

请求
DELETE http://192.168.0.28:8080/cas/casrs/storage/delete?id=1&poolName=localTest&hostName=cvm28
accept: application/xml
...
