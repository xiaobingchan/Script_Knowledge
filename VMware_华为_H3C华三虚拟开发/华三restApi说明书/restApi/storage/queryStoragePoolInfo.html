H3C CAS云管理平台

查询存储池详细信息

在指定主机下查询指定存储池的详细信息

接口 URI

/cas/casrs/storage/info?hostId={hostId}&poolName={poolName}

访问方法

HTTP 方法
GET
参数
查询参数（Query Parameter）
hostId 存储池所属的主机id 。
长整数（Long）类型。必须，无缺省值。
poolName 存储池名称 。
字符串（String）类型。必须，无缺省值。

返回结果

状态码
  • 200 ：成功。
消息体
HTTP/1.1 200 OK
Server: Apache-Coyote/1.1
Content-Type: application/xml
Content-Length: 116
Date: Tue, 16 Sep 2014 06:58:43 GMT
Server: H3C-CVM

<?xml version="1.0" encoding="UTF-8"?>
<storagePool>
  <name>isopool</name>
  <path>/vms/isos</path>
  <type>本地文件目录</type>
  <totalSize>235105</totalSize>
  <freeSize>76469</freeSize>
  <status>1</status>
  <autoStart>false</autoStart>
</storagePool>
元素含义
storagePool各子元素
name 存储池名称。
字符串（String）类型。
path 存储池路径。
字符串（String）类型。
type 存储池类型。
字符串（String）类型。
totalSize 存储池容量大小。
长整数（long）类型。
freeSize 存储池可用容量。
长整数（long）类型。
status 存储池的状态码
整型（int）类型。
autoStart 是否自动启动
整型（int）类型。

应用举例

查询主机cvm28下的存储池isopool的详细信息。

请求
GET http://192.168.0.28:8080/cas/casrs/storage/info?hostId=1&poolName=isopool
accept: application/xml
回应
HTTP/1.1 200 OK
Server: Apache-Coyote/1.1
Content-Type: application/xml
Content-Length: 116
Date: Tue, 16 Sep 2014 06:58:43 GMT
Server: H3C-CVM

<?xml version="1.0" encoding="UTF-8"?>
<storagePool>
  <name>isopool</name>
  <path>/vms/isos</path>
  <type>本地文件目录</type>
  <totalSize>235105</totalSize>
  <freeSize>76469</freeSize>
  <status>1</status>
  <autoStart>false</autoStart>
