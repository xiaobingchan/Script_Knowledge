H3C CAS云管理平台

增加存储卷

在指定存储池下增加指定存储卷

接口 URI

/cas/casrs/storage/volume

访问方法

HTTP 方法
POST
消息体
<volAddParameter>
  <hostId>4</hostId>
  <spName>t2</spName>
  <volName>jj08</volName>
  <capacity>39582</capacity>
  <format>qcow2</format>
  <baseFile>/vms/t2/jj01</baseFile>
  <mode>1</mode>
	<clusterSize>65536</clusterSize>
</volAddParameter>
元素含义
vmParamter各子元素
hostId 指定存储池所在的主机 ID。
长整数（long）类型。
spName 存储池名称。
字符串（String）类型。
volName 存储卷名称。
字符串（String）类型。
capacity 存储卷的最大容量。
长整数（long）类型。
format 存储卷的格式：支持raw和qcow2格式。
枚举：raw，qcow2。
baseFile 基础镜像文件（绝对路径），可选。
字符串（String）类型。
mode 预分配方式（Other 精简、1:置零、0:延迟置零），可选。
整型（Integer）类型。
clusterSize 存储卷簇大小，枚举值：65536（64K）、131072（128K）、262144（256K）、524288（512K）、1048576（1M）、2097152（2M）
字符串（String）类型。

  • 409(Conflict)：失败。
