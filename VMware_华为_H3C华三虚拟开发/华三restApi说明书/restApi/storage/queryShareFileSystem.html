查询CAS系统中集群挂载共享文件系统的主机列表

查询CAS系统中指定集群中挂载了指定共享文件系统的主机列表

接口 URI

/cas/casrs/storage/fs/queryHost?id={id}&fsId={fsId}

访问方法

HTTP 方法
GET
参数
路径参数（Query Parameter）
id 集群id 。
长整数（Long）类型。必须，无缺省值。
fsId 共享文件系统id 。
长整数（Long）类型。必须，无缺省值。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Server: Apache-Coyote/1.1
Content-Type: application/xml
Content-Length: 2489
Date: Mon, 25 Nov 2013 09:21:00 GMT
消息体
<?xml version="1.0" encoding="UTF-8"?>
<list>
  <host>
    <id>23</id>
  </host>
  <host>
    <id>24</id>
  </host>
  <host>
    <id>25</id>
  </host>
......

</list>
元素含义
host各子元素
id 主机Id
长整数（Long）类型。

 

应用举例

1.查询 CAS系统中id为2的集群中挂载了id为4的共享文件系统的主机列表。

请求
GET http://192.168.0.2:8080/cas/casrs/storage/fs/queryHost?id=2&fsId=4
accept: application/xml
...
回应
HTTP/1.1 200 OK
Server: Apache-Coyote/1.1
Content-Type: application/xml
Content-Length: 2489
Date: Mon, 25 Nov 2013 09:21:00 GMT

<?xml version="1.0" encoding="UTF-8"?>
<list>
  <host>
    <id>23</id>
  </host>
  <host>
    <id>24</id>
  </host>
  <host>
    <id>25</id>
  </host>
</list>
