格式化共享文件系统

在指定主机池下格式化指定共享文件系统

接口 URI

/cas/casrs/storage/shareFile/formatOCFS2

访问方法

HTTP 方法
PUT
消息体
<rsFormatOcfs>
  <id>5</id>
  <maxAccessNum>2</maxAccessNum> 
  <fsName>t27</fsName> 
</rsFormatOcfs>
元素含义
rsFormatOcfs各子元素
id 共享文件系统的ID。
长整数（long）类型。
maxAccessNum 最大访问节点数。
整型（int）类型，取值范围1~32。
fsName 引用该共享文件系统的存储池名称。
字符串（String）类型。

返回结果

状态码
  • 204 ( No Content)：成功。

应用举例

在主机池下修改共享文件系统t27。

请求
PUT http://192.168.0.4:8080/cas/casrs/storage/shareFile/formatOCFS2
accept: application/xml


<rsFormatOcfs>
  <id>5</id>
  <maxAccessNum>2</maxAccessNum> 
  <fsName>t27</fsName> 
</rsFormatOcfs>
