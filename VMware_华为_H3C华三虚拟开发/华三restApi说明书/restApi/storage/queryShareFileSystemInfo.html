H3C CAS云管理平台

查询共享文件系统详细信息

在指定主机池下查询指定共享文件系统的详细信息

接口 URI

/cas/casrs/storage/shareFile/{id}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 共享文件系统的ID 。
长整数（Long）类型。必须，无缺省值。

返回结果

状态码
  • 200 ：成功。

应用举例

查询主机池下的共享文件系统t27的详细信息。

请求
GET http://192.168.0.28:8080/cas/casrs/storage/shareFile/5
accept: application/xml
