H3C CAS云管理平台

查询TARGET列表

查询TARGET列表。

接口 URI

/cas/casrs/storage/iscsiStorage?hostId={hostId}&hostIp={hostIp}

访问方法

HTTP 方法
GET
参数
路径参数（Query Parameter）
hostId 主机id 。
长整数（Long）类型。必须，无缺省值。
hostIp ISCSI网络存储IP地址。
字符串（String）类型。必须，无缺省值。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Cache-Control: no-cache, no-store, max-age=0, must-revalidate
Pragma: no-cache
Expires: 0
X-XSS-Protection: 1; mode=block
X-Frame-Options: DENY
X-Content-Type-Options: nosniff
Set-Cookie: JSESSIONID=F5D8852DD44C829A3184B4FC1EB83C4F; Path=/cas/; HttpOnly
Content-Type: application/xml
Content-Length: 774
Date: Tue, 15 Aug 2017 07:38:17 GMT
Server: H3C-CVM
消息体

<?xml version="1.0" encoding="UTF-8"?><list>
  <iscsiStorage>
    <srcName>gun</srcName>
  </iscsiStorage>
  <iscsiStorage>
    <srcName>t11</srcName>
  </iscsiStorage>
  <iscsiStorage>
    <srcName>tar09</srcName>
  </iscsiStorage>
</list>
元素含义
iscsiStorage各子元素
srcName target名称。
字符串（String）类型。

 

应用举例

1.查询 CAS系统中主机id为1的主机下，可访问192.168.0.11的存储target列表。

请求
GET http://192.168.0.2:8080/cas/casrs/storage/iscsiStorage?hostId=1&hostIp='192.168.0.11'
accept: application/xml
