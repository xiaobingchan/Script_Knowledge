删除指定存储卷

删除指定存储池下的指定存储卷。

接口 URI

/cas/casrs/storage/volume/delete/{id}/{poolName}/{volName}

访问方法

HTTP 方法
DELETE
参数
路径参数（Path Parameter）
id 存储卷所在存储池所属的主机id 。
长整数（Long）类型。必须，无缺省值。
poolName 存储卷所在存储池的名称 。
字符串（String）类型。必须，无缺省值。
volName 存储卷名称。
字符串（String）类型。必须，无缺省值。

返回结果

状态码
  • 204(No Content)：成功。
  • 409(Conflict)：失败。
消息头
HTTP/1.1 204 No Content
Server: Apache-Coyote/1.1
Date: Fri, 22 Nov 2013 07:45:07 GMT

应用举例

运行 ID为“52”的虚拟机操作。

请求
DELETE http://192.168.0.4:8080/cas/casrs/storage/volume/delete/4/t2/jj08
accept: application/xml
...
回应
HTTP/1.1 204 No Content
Server: Apache-Coyote/1.1
