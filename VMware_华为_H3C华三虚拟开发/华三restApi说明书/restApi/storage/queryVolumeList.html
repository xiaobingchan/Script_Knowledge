H3C CAS云管理平台

查询CAS系统中存储卷列表

查询CAS系统中主机下指定存储池下的存储卷列表。

接口 URI

/cas/casrs/storage/volume?hostId={hostId}&poolName={poolName}&offset={offset}&pageSize={pageSize}

访问方法

HTTP 方法
GET
参数
路径参数（Query Parameter）
hostId 存储池所属的主机id 。
长整数（Long）类型。必须，无缺省值。
poolName 存储池名称 。
字符串（String）类型。必须，无缺省值。
volumeName 存储卷名称 。
字符串（String）类型。
offset 分页查询结果偏移值 。
整型（Integer）类型。可选项，缺省值为0。
pageSize 分页查询结果数量 。
整型（Integer）类型。可选项，缺省值为30。

元素含义
storagePool各子元素
name 存储卷名称。
字符串（String）类型。
size 文件分配大小，单位M。
双精度（Double）类型。
allocation 已用空间，单位M。
双精度（Double）类型。
format 类型。
字符串（String）类型。
backingStore 基础镜像文件。
字符串（String）类型。
users 存储卷的使用者。
字符串数组（String[]）类型。
