H3C CAS云管理平台

删除指定共享文件系统

删除指定主机池下的指定共享文件系统。

接口 URI

/cas/casrs/hostpool/shareFile/delete/{id}

访问方法

HTTP 方法
DELETE
参数
路径参数（Path Parameter）
id 共享文件系统的ID。
长整数（Long）类型。必须，无缺省值。

返回结果

状态码
  • 204(No Content)：成功。
  • 409(Conflict)：失败。
消息头
HTTP/1.1 204 No Content
Server: Apache-Coyote/1.1
Date: Fri, 22 Nov 2013 07:45:07 GMT

应用举例

删除主机cvm28下的共享文件系统t27_edit。

请求
DELETE http://192.168.0.28:8080/cas/casrs/hostpool/shareFile/delete/5
accept: application/xml
...
回应
HTTP/1.1 204 No Content
Server: Apache-Coyote/1.1
Content-Type: application/xml
Content-Length: 116
Date: Tue, 16 Sep 2014 06:58:43 GMT
Server: H3C-CVM
