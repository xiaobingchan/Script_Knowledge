H3C CAS云管理平台

刷新存储池

在指定主机下刷新指定存储池

接口 URI

/cas/casrs/storage/refresh

访问方法

HTTP 方法
GET
消息体
参数
路径参数（Query Parameter）
id 存储池所属的主机id 。
长整数（Long）类型。必须，无缺省值。
poolName 存储池名称 。
字符串（String）类型。必须，无缺省值。

返回结果

状态码
  • 200 ：成功。

应用举例

刷新主机cvm28下的存储池isopool。

请求
GET http://192.168.0.28:8080/cas/casrs/storage/refresh?id=1&poolName=isopool
accept: application/xml
回应
HTTP/1.1 200 OK
Server: Apache-Coyote/1.1
Content-Type: application/xml
Content-Length: 116
Date: Tue, 16 Sep 2014 06:58:43 GMT
Server: H3C-CVM

<?xml version="1.0" encoding="UTF-8"?>
<rsResult>
  <state>0</state>
  <errorCode>0</errorCode>
</rsResult>
