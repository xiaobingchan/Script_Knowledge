H3C CAS云管理平台

增加共享文件系统

在指定主机池下增加指定共享文件系统

接口 URI

/cas/casrs/storage/shareFile/add

访问方法

HTTP 方法
POST
消息体
<rsShareFileSystem>
  <name>t27</name>
  <desc>jj08</desc>
  <type>1</type>
  <hpId>qcow2</hpId>
  <path>/vms/t27</path>
  <rsFsLunInfo> 
    <targetHost>192.168.0.27</targetHost>
    <naa>1H3C682ce2123d8011e4a715ac162d88e4c4</naa>
    <lun>ip-192.168.0.27:3260-iscsi-t27-lun-1</lun>
    <capacity>107374182400</capacity>
  </rsFsLunInfo>
</rsShareFileSystem>
元素含义
rsShareFileSystem各子元素
name 共享文件系统的名称。
长整数（long）类型。
type 共享文件系统的类型。
整型（int）类型。
hpId 主机池ID。
长整数（long）类型。
path 路径。
字符串（String）类型。
readOnly 是否只读。
布尔类型。
rsFsLunInfo各子元素
targetHost 目的主机IP。
字符串（String）类型
naa naa。
字符串（String）类型。
lun lun。
字符串（String）类型。
capacity 容量。
长整数（Long）类型。

返回结果

状态码
  • 204 ( No Content)：成功。

应用举例

在主机池下增加共享文件系统t27。

请求
POST http://192.168.0.4:8080/cas/casrs/storage/shareFile/add
accept: application/xml


<rsShareFileSystem>
  <name>t27</name>
  <desc>jj08</desc>
  <type>1</type>
  <hpId>qcow2</hpId>
  <path>/vms/t27</path>
  <rsFsLunInfo> 
    <targetHost>192.168.0.27</targetHost>
    <naa>1H3C682ce2123d8011e4a715ac162d88e4c4</naa>
    <lun>ip-192.168.0.27:3260-iscsi-t27-lun-1</lun>
    <capacity>107374182400</capacity>
  </rsFsLunInfo>
</rsShareFileSystem>
回应
HTTP/1.1 204 No Content
Content-Type: application/xml

无消息体
