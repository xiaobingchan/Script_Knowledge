H3C CAS云管理平台

查询操作员的全部信息

查询操作员的全部信息。

接口 URI

/cas/casrs/operator/operatorInfoList

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Server: Apache-Coyote/1.1
Content-Type: application/xml
Content-Length: 704
Date: Thu, 21 Nov 2013 06:48:36 GMT
消息体
<?xml version="1.0" encoding="UTF-8"?>
<operator>
  <id>1</id>
  <loginName>a</loginName>
  <authType>0</authType>
  <name>a</name>
  <email>fd@d.c</email>
  <phone>1253</phone>
  <groupId>1</groupId>
  <enable>1</enable>
  <onlineNum>0</onlineNum>
  <cluster>
    <id>1</id>
    <name>cluter</name>
  </cluster>
</operator>
元素含义
operatorInfoList各子元素
id 操作员 ID。
长整数（long）类型。
loginName 操作员登录名。
字符串（String）类型。
name 操作员名称。
字符串（String）类型。
authType 认证类型。
整数（int）类型。枚举值：0 密码类型，1 LADP认证
email 邮件。
字符串（String）类型。
phone 电话号码。
字符串（String）类型。
groupId 操作员分组id。
长整数（long）类型。
enable 是否禁止操作员。
boolean类型。
onlineNum 在线数量。
整数（int）类型。
id 集群id。
长整数（long）类型。
name 集群名称。
字符串（String）类型。
id 主机 ID。
长整数（long）类型。
ipAddr IP地址。
字符串（String）类型。
id 虚拟机 ID。
长整数（long）类型。
domainName 虚拟机名称。
字符串（String）类型。
title 虚拟机title。
字符串（String）类型。
uuid 虚拟机的UUID。
字符串（String）类型。
