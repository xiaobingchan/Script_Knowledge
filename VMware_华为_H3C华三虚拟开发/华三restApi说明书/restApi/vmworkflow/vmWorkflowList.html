H3C CAS云管理平台

查询虚拟机电子流列表

根据提供的特定条件，查询虚拟机电子流列表。

接口 URI

/cas/casrs/user/vmWorkflow

访问方法

HTTP 方法
GET
参数
查询参数（Query Parameter）
offset 返回的第一条记录在总结果集中的位置。
整数（int）类型，非必须，缺省为 0。如果传入参数小于 0，则使用缺省值 0。
limit 返回记录总数。
整数（int）类型，非必须，缺省为 10。如果传入参数小于 0，则使用缺省值 10。
sortField 排序字段。
字符串（String）类型，非必须。
userId 用户ID。
整数（long）类型，必须。
total 仅返回符合条件的记录数 。
布尔（boolean）类型，非必须，缺省为 false。为 true 时 offset、limit和sortField属性无效，返回消息体为空，消息头 Total 属性返回符合条件的记录数。

返回结果

状态码
  • 200(OK)：成功。
消息头

当 total 参数为 true 时，Total 属性返回符合条件的总记录数。此时，无消息体。 
HTTP/1.1 200 OK
Total: 1
Content-Type: application/xml
Content-Length: 0
Server: Jetty(6.1.x)
消息体
<?xml version="1.0" encoding="UTF-8"?>
<list>
  <vmWorkflow>
    <id>1</id>
    <domainId>1</domainId>
    <domainName>jjun-test1</domainName>
    <description>蒋军申请虚拟机测试1</description>
    <userId>5</userId>
    <orgId>10</orgId>
    <status>0</status>
    <type>1</type>
    <createDate>2012-06-05T10:00:51+08:00</createDate>
    <handleDate>2012-06-05T10:00:51+08:00</handleDate>
  </vmWorkflow>
</list>
元素含义
operateLog各子元素
id 虚拟机工作流 ID。
长整数（long）类型。
domainId 虚拟机ID/虚拟机模板ID。
长整数（long）类型。
domainName 虚拟机名称。
字符串（String）类型。
description 虚拟机描述。
字符串（String）类型。
userId 用户ID。
长整数（long）类型。
orgId 组织ID
长整数（long）类型。
status 状态。 0：待审批 10：审批通过 20：拒绝 30：撤销
整数（int）类型。
type 申请类型。1:申请虚拟机，0:注销虚拟机
字符串（String）类型。
createDate 申请时间。
日期（Date）类型。
handleDate 处理时间。
日期（Date）类型。

应用举例

1.统计用户ID为5的虚拟机工作流数目。

请求
GET http://192.168.0.243:8888/cas/casrs/user/vmWorkflow?offset=0&limit=10&sortField=id&userId=5&total=true
accept: application/xml
...
回应
HTTP/1.1 200 OK
Total: 1
Content-Type: application/xml
Content-Length: 0
Server: Jetty(6.1.x)
