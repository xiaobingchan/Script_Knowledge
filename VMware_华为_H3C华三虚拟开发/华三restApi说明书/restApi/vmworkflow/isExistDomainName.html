检查虚拟机名称是否已经存在

根据提供的虚拟机名称，检查虚拟机名称是否已经存在。

接口 URI

/cas/casrs/vm/isExist?name={domainName}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
domainName 虚拟机名称。
字符串（String）类型，必须，无缺省值。

返回结果

状态码
  • 204(No Content)：成功。
  • 409(Conflict)：失败。
消息头
HTTP/1.1 200 OK
result: false
Content-Type: application/xml
Transfer-Encoding: chunked

应用举例

1.检查虚拟机名称testdomain是否已经存在。

请求
GET http://192.168.0.243:8888//cas/casrs/vm/isExist?name=domainName
accept: application/xml
...
回应
HTTP/1.1 200 OK
result: false
Content-Type: application/xml
Transfer-Encoding: chunked
