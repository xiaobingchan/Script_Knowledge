H3C CAS云管理平台

增加(申请/注销)虚拟机电子流

为指定ID的用户增加(申请/注销)虚拟机电子流（type=1申请，type=0注销）。

接口 URI

/cas/casrs/vmWorkflow

访问方法

HTTP 方法
POST
消息体
<vmWorkflow>
    <domainId>1</domainId>
    <domainName>jjun-test1</domainName>
    <description>李四申请虚拟机测试1</description>
    <userId>5</userId>
    <orgId>10</orgId>
    <status>0</status>
    <type>1</type>
</vmWorkflow>
元素含义
operateLog各子元素
domainId 虚拟机ID/虚拟机模板ID。
长整数（long）类型。
domainName 虚拟机名称。
字符串（String）类型。
description 虚拟机描述。
字符串（String）类型。
userId 用户ID。
长整数（long）类型。
orgId 所属的组织ID。
长整数（long）类型。
status 状态。 0：待审批 10：审批通过 20：拒绝 30：撤销
整数（int）类型。
type 申请类型。1:申请虚拟机，0:注销虚拟机
字符串（String）类型。

 

返回结果

状态码
  • 201 (Created)：成功。

应用举例

为ID为”5“的租户增加(申请/注销)虚拟机电子流。

请求
POST http://192.168.0.243:8888/cas/casrs/vmWorkflow/
accept: application/xml
<vmWorkflow>
    <domainId>1</domainId>
    <domainName>jjun-test1</domainName>
    <description>蒋军申请虚拟机测试1</description>
    <userId>5</userId>
    <orgId>10</orgId>
    <status>0</status>
    <type>1</type>
</vmWorkflow>
回应
HTTP/1.1 201 Created
Expires: Thu, 01 Jan 1970 00:00:00 GMT
Set-Cookie: JSESSIONID=f9dt6kjmf7d0;Path=/
Content-Type: application/xml
Location: 1
Content-Length: 0
Server: Jetty(6.1.x)

无消息体
