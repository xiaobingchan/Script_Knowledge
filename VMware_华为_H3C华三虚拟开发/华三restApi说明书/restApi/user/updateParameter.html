H3C CAS云管理平台

更新租户参数

更新租户的所有参数配置。

接口 URI

/cas/casrs/user/parameter/{userId}

访问方法

HTTP 方法
PUT
消息体
<list>
  <parameter>
    <name>zxk</name>
    <value>test01</value>    
  </parameter>
</list>
元素含义
parameter各子元素
name 参数名称。
字符串（String）类型。
value 参数值。
字符串（String）类型。

返回结果

状态码
  • 204(No Content)：成功。

应用举例

更新用户ID为5的参数配置。

请求
PUT http://192.168.0.4:8080/cas/casrs/user/parameter/5
accept: application/xml


<list>
  <parameter>
    <name>myhome.column.number</name>
    <value>5</value>
  </parameter>
  <parameter>
    <name>myhome.column.width</name>
    <value>280</value>
  </parameter>
</list>
回应
HTTP/1.1 204 No Content
Server: Apache-Coyote/1.1
Set-Cookie: JSESSIONID=A74EBA14A1C0E12AD2EF9B3F7A876C28; Path=/cas
Date: Mon, 21 May 2012 11:12:54 GMT

无消息体
