H3C CAS云管理平台

查询租户操作日志列表

根据提供的特定条件，查询租户操作日志列表。

接口 URI

/cas/casrs/operatorLog/

访问方法

HTTP 方法
GET
参数
查询参数（Query Parameter）
offset 返回的第一条记录在总结果集中的位置。
整数（int）类型，非必须，缺省为 0。如果传入参数小于 0，则使用缺省值 0。
limit 返回记录总数。
整数（int）类型，非必须，缺省为 10。如果传入参数小于 0，则使用缺省值 10。
orderBy 排序字段。
字符串（String）类型，非必须。
userId 用户ID。
整数（long）类型，必须。
total 仅返回符合条件的记录数 。
布尔（boolean）类型，非必须，缺省为 false。为 true 时 offset、limit和orderBy属性无效，返回消息体为空，消息头 Total 属性返回符合条件的记录数。

返回结果

状态码
  • 200(OK)：成功。
消息头

当 total 参数为 true 时，Total 属性返回符合条件的总记录数。此时，无消息体。 
HTTP/1.1 200 OK
Total: 7
消息体
<?xml version="1.0" encoding="UTF-8"?>
<list>
  <operateLog>
    <id>71</id>
    <loginName>company_test_user2</loginName>
    <userName>company_test_user2</userName>
    <operTime>2012-05-10T09:04:57+08:00</operTime>
    <address>192.168.0.122</address>
    <category>1</category>
    <description>用户登录。</description>
    <result>0</result>
  </operateLog>
	...

</list>
元素含义
operateLog各子元素
id 操作日志 ID。
长整数（long）类型。
loginName 租户登录名。
字符串（String）类型。
userName 租户姓名。
字符串（String）类型。
operTime 操作时间。
日期（Date）类型。
address 登录客户端IP地址。
字符串（String）类型。
category 操作类别。 1：登录/注销操作 10：虚拟机动作
整数（int）类型。
description 操作描述。
字符串（String）类型。
result 操作结果。 0：成功 1：失败。
整数（int）类型。

 

应用举例

1.统计用户ID为5的操作日志数目。

请求
GET http://localhost:8080/cas/casrs/operatorLog?offset=0&limit=10&sortField=id&userId=5&total=true
accept: application/xml
...
回应
HTTP/1.1 200 OK
Server: Apache-Coyote/1.1
Total: 8
Content-Type: application/xml
Content-Length: 0
