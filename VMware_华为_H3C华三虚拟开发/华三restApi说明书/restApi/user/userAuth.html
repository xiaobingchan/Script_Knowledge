H3C CAS云管理平台

租户登录

租户登录身份认证。

接口 URI

/cas/casrs/user/auth/{loginName}/{password}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
loginName 登录名。
字符串（String）类型。
password 登录密码。密码需经过MD5加密，然后再进行BASE64编码。
字符串（String）类型。

返回结果

状态码
  • 204(No Content)：成功。
  • 409(Conflict)：失败
消息头
HTTP/1.1 200 OK
Server: Apache-Coyote/1.1
Content-Type: application/xml
Content-Length: 194
Date: Mon, 21 May 2012 12:33:30 GMT
消息体
<?xml version="1.0" encoding="UTF-8"?>
<user>
  <id>6</id>
  <loginName>zxk</loginName>
  <userName>zxk</userName>
  <authType>0</authType>
  <userGroupId>2</userGroupId>
  <orgId>1</orgId>
</user>
元素含义
user各子元素
id 用户ID。
长整数（long）类型。
loginName 租户登录名。
字符串（String）类型。
userName 租户姓名。
字符串（String）类型。
orgId 所属的组织ID。
长整数（long）类型。
userGroupId 所属用户分组ID。
长整数（long）类型。
authType 认证方式。 0：简单密码认证，1：LDAP认证
整数（int）类型。

应用举例

对登录名为"zxk"，原始密码为“admin”的租户进行认证。

请求
DELETE http://192.168.0.4:8080//cas/casrs/user/auth/zxk/WpIxDAPVsRM=
accept: application/xml
...
