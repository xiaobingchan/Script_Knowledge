H3C CAS云管理平台

查询租户参数配置

查询指定租户的所有参数配置。

接口 URI

/cas/casrs/user/parameter/{userId}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
userId 租户id 。
长整数（Long）类型。必须，无缺省值。
 

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Server: Apache-Coyote/1.1
Content-Type: application/xml
Content-Length: 344
Date: Mon, 21 May 2012 12:42:13 GMT
消息体
<?xml version="1.0" encoding="UTF-8"?>
<list>
  <parameter>
    <id>10</id>
    <name>myhome.column.number</name>
    <value>5</value>
  </parameter>
	...

</list>
元素含义
parameter各子元素
id 参数 ID。
长整数（long）类型。
name 参数名称。
字符串（String）类型。
value 参数值。
字符串（String）类型。

 

应用举例

查询用户ID为5所有参数配置。

请求
GET http://localhost:8080/cas/casrs/user/parameter/6
accept: application/xml
...
回应
HTTP/1.1 200 OK
Content-Type: application/xml
...

  <parameter>
    <id>10</id>
    <name>myhome.column.number</name>
    <value>5</value>
  </parameter>
  <parameter>

    <name>myhome.column.width</name>

  </parameter>
  <parameter>
    <id>12</id>
