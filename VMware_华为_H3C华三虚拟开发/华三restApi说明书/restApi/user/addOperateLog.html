H3C CAS云管理平台

增加用户操作日志

为指定ID的用户增加操作日志。

接口 URI

/cas/casrs/operatorLog/

访问方法

HTTP 方法
POST
消息体
<operateLog>
    <userId>1</userId>
    <orgId>1</orgId>
    <loginName>zxk</loginName>
    <userName>zxk</userName>
    <address>192.168.0.105</address>
    <category>1</category>
    <description>用户登录</description>
    <result>0</result>
</operateLog>
元素含义
operateLog各子元素
userId 用户ID。
长整数（long）类型。
loginName 租户登录名。
字符串（String）类型。
userName 租户姓名。
字符串（String）类型。
orgId 所属的组织ID。
长整数（long）类型。
address 登录客户端IP地址。
字符串（String）类型。
category 操作类别。 1：登录/注销操作 10：虚拟机动作
整数（int）类型。
description 操作描述。
字符串（String）类型。
result 操作结果。 0：成功 1：失败。
整数（int）类型。

 

返回结果

状态码
  • 201 (Created)：成功。

应用举例

为ID为”5“的租户增加登录操作日志。

请求
POST http://192.168.0.4:8080/cas/casrs/operatorLog/
accept: application/xml


<operateLog>
    <userId>5</userId>
    <orgId>1</orgId>
    <loginName>zxk</loginName>
    <userName>zxk</userName>
    <address>192.168.0.105</address>
    <category>1</category>
    <description>用户登录</description>
    <result>0</result>
</operateLog>
回应
HTTP/1.1 204 No Content
Server: Apache-Coyote/1.1
Set-Cookie: JSESSIONID=A74EBA14A1C0E12AD2EF9B3F7A876C28; Path=/cas

无消息体
