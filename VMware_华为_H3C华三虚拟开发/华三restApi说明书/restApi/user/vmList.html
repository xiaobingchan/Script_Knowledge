查询租户的虚拟机列表

根据提供的特定条件，查询租户的虚拟机列表。

接口 URI

/cas/casrs/user/vm

访问方法

HTTP 方法
GET
参数
查询参数（Query Parameter）
offset 返回的第一条记录在总结果集中的位置。
整数（int）类型，非必须，缺省为 0。如果传入参数小于 0，则使用缺省值 0。
limit 返回记录总数。
整数（int）类型，非必须，缺省为 10。如果传入参数小于 0，则使用缺省值 10。
orderBy 排序字段。
字符串（String）类型，非必须。
userId 用户ID。
整数（long）类型，必须。
userGrpId 用户分组ID。
整数（long）类型，必须。
total 仅返回符合条件的记录数 。
布尔（boolean）类型，非必须，缺省为 false。为 true 时 offset、limit和orderBy属性无效，返回消息体为空，消息头 Total 属性返回符合条件的记录数。

返回结果

状态码
  • 200(OK)：成功。
消息头

当 total 参数为 true 时，Total 属性返回符合条件的总记录数。此时，无消息体。 
HTTP/1.1 200 OK
Total: 7

元素含义
domain各子元素
id 虚拟机 ID。
长整数（long）类型。
hostId 运行的主机ID。
长整数（long）类型。
clusterId 所属的集群 ID。
长整数（long）类型。
hostPoolId 所属的主机池 ID。
长整数（long）类型。
name 虚拟机名称。
字符串（String）类型。
description 虚拟机描述信息。
字符串（String）类型。
memory 虚拟机内存大小，单位（M） 。
整数（int）类型。
cpu 虚拟机CPU个数。
整数（int）类型。
status 虚拟机状态。 2：运行 3：关闭 4：暂停。
整数（int）类型。
uuid 虚拟机UUID。
字符串（String）类型。
osBit 虚拟机安装的操作系统位数。
字符串（String）类型。
system 虚拟机安装的操作系统类型。0：Windows 1：Linux。
整数（int）类型。
autoMigrate 是否允许自动迁移。 0：不允许 1：允许。
整数（int）类型。
createDate 虚拟机创建日期。
日期（Date）类型。
storageCapacity 虚拟机镜像存储大小。 单位（G）
长整数（long）类型。

 

应用举例

1.统计用户ID为5，用户分组为2的虚拟机数目。

请求
GET http://localhost:8080/cas/casrs/user/vm?userId=6&userGrpId=2&total=true
accept: application/xml
...
回应
HTTP/1.1 200 OK
Server: Apache-Coyote/1.1
Total: 6
Content-Type: application/xml
Content-Length: 0

请求
GET http://localhost:8080/cas/casrs/user/vm?offset=0&limit=10&sortField=id&userId=6&userGrpId=2
