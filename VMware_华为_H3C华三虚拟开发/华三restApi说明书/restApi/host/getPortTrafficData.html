H3C CAS云管理平台

查询主机下虚拟交换机端口详细信息

根据主机id和虚拟交换机id查询虚拟交换机端口详细信息。

接口 URI

/cas/casrs/host/vs/vport?id={id}&vsId={vsId}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 主机id 。
整数（Long）类型。必须。
vsId 虚拟交换机id 。
整数（Long）类型。必须。

返回结果

状态码
  • 200(OK)：成功。

元素含义
trafficInfo各子元素
vsName 虚拟交换机名称
字符串（String）类型。
vPortName 虚拟端口名称
字符串（String）类型。
vmName 虚拟机名称
字符串（String）类型。
title 虚拟机显示名称
字符串（String）类型。
vmMac 虚拟机MAC地址
字符串（String）类型。
rxPkts 接收报文数
长整型（Long）类型。
rxBytes 接收字节数
长整型（Long）类型。
rxErrs 接收错包数
长整型（Long）类型。
txPkts 发送报文数
长整型（Long）类型。
txBytes 发送字节数
长整型（Long）类型。
txErrs 发送错包数
长整型（Long）类型。
