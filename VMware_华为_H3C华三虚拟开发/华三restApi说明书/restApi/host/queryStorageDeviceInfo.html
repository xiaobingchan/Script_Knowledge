查询存储适配器的存储设备信息

根据主机id及存储适配器的标示符查看存储适配器的存储设备信息。

接口 URI

/cas/casrs/storageAdapter/device/query

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 主机id。
长整数（long）类型。必须。
wwn 存储适配器标示符。
字符串（String）类型。必须。

返回结果

状态码
  • 200(OK)：成功。

元素含义
lun各子元素
naa 存储设备的NAA。
字符串（String）类型。
storageName 存储池名称。
字符串（String）类型。
status 操作状态。
字符串（String）类型。
lunNumber 存储设备的LUN个数。
字符串（String）类型。
capacity 存储设备的最大容量。
字符串（String）类型。
multipath 多路径策略。0：不启用 1：启用
整数（int）类型。
应用举例
