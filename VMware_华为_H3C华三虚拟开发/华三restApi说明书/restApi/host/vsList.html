H3C CAS云管理平台

查询指定主机所有虚拟交换机

根据主机id查询指定物理主机下的所有虚拟交换机信息。

接口 URI

/cas/casrs/host/id/{hostId}/vswitch

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
hostId 主机id 。
长整数（long）类型。必须。

返回结果

状态码
  • 200(OK)：成功。
消息头

HTTP/1.1 200 OK
Total: 7
消息体
<?xml version="1.0" encoding="UTF-8"?>
<list>
  <vSwitch>
    <id>126</id>
    <name>test01</name>
    <mode>1</mode>
    <pnic>eth1</pnic>
    <status>1</status>
    <address>3.3.0.1</address>
    <netmask>255.255.255.0</netmask>
    <gateway/>
    <enableLacp>false</enableLacp>
  </vSwitch>
  <vSwitch>
    <id>127</id>
    <name>vswitch0</name>
    <mode>0</mode>
    <pnic>eth0</pnic>
    <status>1</status>
    <address>192.168.0.8</address>
    <netmask>255.255.255.0</netmask>
    <gateway>192.168.0.1</gateway>
    <enableLacp>false</enableLacp>
  </vSwitch>
...
</list>
元素含义
vSwitch各子元素
id 虚拟交换机 ID。
长整数（long）类型。
name 虚拟交换机名称。
字符串（String）类型。
mode 转发模式。0 : veb; 1: vepa; 2:多通道。
整数（int）类型。
pnic vSwitch绑定的所有物理网卡名。
字符串（String）类型。
status 虚拟机交换状态。0:不活动 1:活动 。
整数（int）类型。
address IP地址。
字符串（String）类型。
netmask 网络掩码。
字符串（String）类型。
gateway 网关
字符串（String）类型。
enableLacp 是否启用LACP(启用：true，未启用：false) 。
布尔（Boolean）类型。
