查询指定虚拟交换机虚拟口信息

根据提供的特定条件，查询查询指定虚拟交换机虚拟口信息列表。

接口 URI

/cas/casrs/vswitch/vport

访问方法

HTTP 方法
GET
参数
查询参数（Query Parameter）
hostId 主机id 。
长整数（long）类型。必须。
vsName 虚拟交换机名称。
字符串（String）类型。必须。
phyNic 物理网卡名称。
字符串（String）类型，必须。

返回结果

状态码
  • 200(OK)：成功。
消息头

HTTP/1.1 200 OK
Total: 7
消息体
<?xml version="1.0" encoding="UTF-8"?>
<list>
  <trafficInfo>
    <vsName>vswitch0</vsName>
    <vPortName>vnet0</vPortName>
    <vmName>vm-xp</vmName>
    <vmMac>0c:da:41:1d:8c:f4</vmMac>
    <rxPkts>0</rxPkts>
    <rxBytes>0</rxBytes>
    <rxErrs>0</rxErrs>
    <txPkts>0</txPkts>
    <txBytes>0</txBytes>
    <txErrs>0</txErrs>
  </trafficInfo>

	...

</list>
元素含义
trafficInfo各子元素
vsName 虚拟交换机名称。
字符串（String）类型。
vPortName 虚拟端口名称。
字符串（String）类型。
vmNam 虚拟机名称。
字符串（String）类型。
vmMac 虚拟机MAC地址。
字符串（String）类型。
rxPkts 接收报文数。
长整数（long）类型。
rxBytes 接收字节数。
长整数（long）类型。
rxErrs 接收错包数 。
长整数（long）类型。
txPkts 发送报文数。
长整数（long）类型。
txBytes 发送字节数。
长整数（long）类型。
txErrs 发送错包数。
长整数（long）类型。

 

应用举例

1.查询主机id为21，虚拟交换机名称为vswitch0，物理网卡为eth0的虚拟口信息列表。。

请求
GET http://localhost:8080/cas/casrs/vswitch/vport?hostId=21&vsName=vswitch0&phyNic=eth0
accept: application/xml
...
