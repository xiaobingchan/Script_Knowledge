H3C CAS云管理平台

修改主机大页内存配置

根据主机id修改大页内存配置。

接口 URI

/cas/casrs/host/{id}/hugePage

访问方法

HTTP 方法
PUT
消息体
<hugePageConfig>
        <status>1</status>
        <hugepageSize>1G</hugepageSize>
        <hugepageNum>24</hugepageNum>
</hugePageConfig>
参数
路径参数（Path Parameter）
id 主机id 。
长整数（long）类型。必须。
元素含义
各子元素
status 大页配置状态。true：启用，false：未启用
布尔（Boolean)类型。
hugepageSize 页大小。
字符串（String)类型。
hugepageNum 页数。
整型（int)类型。

返回结果

状态码
  • 204(No Content)：成功。

应用举例

修改主机id为6的主机大页内存配置。

请求
PUT http://localhost:8080/cas/casrs/host/6/hugePage
accept: application/xml
...       
<hugePageConfig>
        <status>1</status>
        <hugepageSize>1G</hugepageSize>
        <hugepageNum>24</hugepageNum>
</hugePageConfig>
回应
HTTP/1.1 204 No Content
