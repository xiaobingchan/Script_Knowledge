修改主机的iscsi信息

修改主机的iscsi信息。

接口 URI

/cas/casrs/host/iscsi/modify?id={id}

访问方法

HTTP 方法
PUT
参数
路径参数（Path Parameter）
id 主机id。
长整数（long）类型。必须。
name iscsi的标示符。
字符串（String）类型。必选。

返回结果

状态码
  • 200(OK)：成功。
消息头
<?xml version="1.0" encoding="UTF-8"?>
HTTP/1.1 200 OK
Content-Type: application/xml
Date: Wed, 17 Jun 2015 10:55:23 GMT
Server: H3C-CVM

 

应用举例

将id为"1"的主机的iscsi的标示符修改为iqn.1993-08.org.debian:01:cvk90。

请求
PUT http://192.168.0.8:8080/cas/casrs/host/iscsi/modify?id=1&name=iqn.1993-08.org.debian:01:cvk90
accept: application/xml
回应
HTTP/1.1 200 OK
Content-Type: application/xml
Content-Length: 185
Date: Wed, 17 Jun 2015 10:55:23 GMT
Server: H3C-CVM
<?xml version="1.0" encoding="UTF-8"?>
<rsResult>
  <state>0</state>
  <errorCode>0</errorCode>
  <data>1</data>
  <message>设置主机“cvk90”的标识符信息成功。</message>
</rsResult>
