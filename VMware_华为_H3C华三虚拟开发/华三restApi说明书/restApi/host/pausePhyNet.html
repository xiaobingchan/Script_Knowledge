H3C CAS云管理平台

暂停主机物理网卡

根据主机id暂停指定物理主机的物理网卡。

接口 URI

/cas/casrs/host/pnic/pause?id={id}&interName={interName}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 主机id 。
长整数（long）类型。必须。
interName 物理网卡名称。
字符串（String）类型。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Content-Type: application/xml
Content-Length: 193
Date: Wed, 17 Sep 2014 09:41:54 GMT
Server: H3C-CVM
消息体
<?xml version="1.0" encoding="UTF-8"?>
<rsResult>
  <state>0</state>
  <errorCode>0</errorCode>
  <message>暂停主机“cvk011”上物理网卡“eth1”成功。</message>
</rsResult>
元素含义
rsResult各子元素
state 状态。
整型（Int）类型。
errorCode 错误代码。
整型（Int）类型。
message 消息
字符串（String）类型。
应用举例

暂停主机id为7，名称为eth1的物理网卡。

请求
GET http://localhost:8080/cas/casrs/host/pnic/pause?id=7&interName=eth1
accept: application/xml
...
回应
HTTP/1.1 200 OK
Content-Type: application/xml
...
<?xml version="1.0" encoding="UTF-8"?>
<rsResult>
  <state>0</state>
  <errorCode>0</errorCode>
  <message>暂停主机“cvk011”上物理网卡“eth1”成功。</message>
</rsResult>
