H3C CAS云管理平台

查询主机的CPU、内存、磁盘利用率、磁盘的I/O吞吐量读写、磁盘读写请求(IOPS)、磁盘读写延迟

根据主机id查询指定物理主机下的CPU、内存、磁盘利用率、磁盘的I/O吞吐量读写、磁盘读写请求(IOPS)、磁盘读写延迟。

接口 URI

/cas/casrs/host/id/{hostId}/monitor

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
hostId 主机id 。
长整数（long）类型。必须。

返回结果

状态码
  • 200(OK)：成功。
消息头

HTTP/1.1 200 OK
消息体
<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<host>
  <cpuRate>14.86</cpuRate>
  <memRate>73.48</memRate>
  <disk>
    <device>sda1</device>
    <usage>6.0</usage>
    <rd_stat>1.0</rd_stat>
    <wr_stat>1.0</wr_stat>
    <rd_req>1.0</rd_req>
    <wr_req>1.0</wr_req>
    <rd_latency>1.0</rd_latency>
    <wr_latency>1.0</wr_latency>
  </disk>
  <disk>
    <device>sda7</device>
    <usage>6.0</usage>
    <rd_stat>1.0</rd_stat>
    <wr_stat>1.0</wr_stat>
    <rd_req>1.0</rd_req>
    <wr_req>1.0</wr_req>
    <rd_latency>1.0</rd_latency>
    <wr_latency>1.0</wr_latency>
  </disk>
</host>
元素含义
host各子元素
cpuRate CPU利用率。
小数（double）类型。
memRate 内存利用率。
小数（double）类型。
disk各子元素
device 磁盘名称。
字符串（String）类型。
usage 磁盘利用率数据。
小数（double）类型。
rd_stat 磁盘读速率。
小数（double）类型。
wr_stat 磁盘写速率。
小数（double）类型。
rd_req 磁盘IPOS读。
小数（double）类型。
wr_req 磁盘IPOS写。
小数（double）类型。
rd_latency 磁盘读延迟。
小数（double）类型。
wr_latency 磁盘写延迟。
小数（double）类型。
