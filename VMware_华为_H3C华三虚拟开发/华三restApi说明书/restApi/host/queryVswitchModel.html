H3C CAS云管理平台

查询虚拟交换机实体信息

根据虚拟交换机ID查询虚拟交换机实体信息

接口 URI

/cas/casrs/vswitch/query?vsId={vsId}

访问方法

HTTP 方法
GET
参数
查询参数（Query Parameter）
vsId 虚拟交换机id 。
长整数（long）类型。必须。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Content-Type: application/xml
Transfer-Encoding: chunked
Date: Wed, 17 Sep 2014 01:17:17 GMT
消息体
<?xml version="1.0" encoding="UTF-8"?>
<vSwitch>
  <id>55</id>
  <name>vswitch0</name>
  <description>test vswitch</description>
  <portNum>32</portNum>
  <mode>0</mode>
  <vlanId>6</vlanId>
  <isRuningVmUseVSwitch>false</isRuningVmUseVSwitch>
  <enableLacp>false</enableLacp>
  <bondMode>balance-tcp</bondMode>
  <flag>0</flag>
  <pnicList>eth1</pnicList>
  <mtu>1500</mtu>
</vSwitch>
元素含义
vSwitch各子元素
id 虚拟交换机ID。
长整数（long）类型。
name 虚拟交换机名称。
字符串（String）类型。
portNum 端口数量。
整数（int）类型。
mode 转发模式。0 : veb; 1: vepa; 2:多通道。
整数（int）类型。
isRuningVmUseVSwitch 判断虚拟交换机是否被使用。
布尔（Boolean）类型。
flag 虚拟交换机的创建方式：0：主机下创建，1：集群下创建。
整数（int）类型。
pnicList 物理网卡名称列表。
集合（List）类型。
enableLacp 链路聚合模式(动态链路聚合：true，静态链路聚合：false。
布尔（Boolean）类型。
bondMode 负载分担模式。枚举类型，启用lacp:balance-slb ，balance-tcp；不启用lacp:balance-slb ，balance-tcp, active-backup。
balance-slb : 基本负载分担,balance-tcp : 高级负载分担,active-backup : 主备负载分担
字符串（String）类型。
description 描述信息。
字符串（String）类型。
vlanId VLAN ID。虚拟交换机端口所在VLAN的ID。VLAN ID的取值范围为2～4094。
整数（int）类型。
mtu MTU值，取值范围1000-9000。
整数（int）类型。
