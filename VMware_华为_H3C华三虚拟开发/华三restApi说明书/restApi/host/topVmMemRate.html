H3C CAS云管理平台

查询主机下虚拟机内存利用率TOPN

根据主机id查询指定物理主机下的虚拟机内存利用率TOPN。

接口 URI

/cas/casrs/host/vm/memRate

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 主机id 。
长整数（long）类型。必须。

返回结果

状态码

元素含义
vmrate各子元素
id 虚拟机id。
长整数（long）类型。

rate 虚拟机内存利用率。
小数（Double）类型。
hostId 主机ID。
长整数（long）类型。
