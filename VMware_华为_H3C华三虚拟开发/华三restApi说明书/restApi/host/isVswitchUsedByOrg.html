检查虚拟交换机是否被组织使用

检查指定虚拟交换机是否被组织使用。

接口 URI

/cas/casrs/vswitch/isUsed

访问方法

HTTP 方法
GET
参数
查询参数（Query Parameter）
id 主机id 。
长整数（Long）类型。必须，无缺省值。
vsName 虚拟交换机名称 。
字符串（String）类型。必须，无缺省值。
 
 

返回结果

状态码
  • 200(OK)：成功。
消息头

HTTP/1.1 200 OK
result: false
Content-Type: application/xml
Transfer-Encoding: chunked
Date: Wed, 17 Sep 2014 01:17:17 GMT

消息体

fasle

元素含义

返回虚拟交换机是否被组织使用

 

应用举例

1.检查主机id为55上的名称为vswitch0的虚拟交换机是否被组织使用

请求
POST http://localhost:8080/cas/casrs/vswitch/isUsed?id=55&&vsName=vswitch0
accept: application/xml
回应
HTTP/1.1 200 OK
result: false
Content-Type: application/xml
Transfer-Encoding: chunked
Date: Wed, 17 Sep 2014 01:17:17 GMT
...
fasle
