H3C CAS云管理平台

查询主机大页内存配置

根据主机id查询大页内存配置。

接口 URI

/cas/casrs/host/{id}/hugePage

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 主机id 。
长整数（long）类型。必须。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Content-Type: application/xml
Content-Length: 193
Date: Wed, 17 Sep 2014 09:41:54 GMT
Server: H3C-CVM
消息体
<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<hugePageInfo>
    <live>
        <status>1</status>
        <hugepageSize>1G</hugepageSize>
        <hugepageNum>62</hugepageNum>
    </live>
    <conf>
        <status>1</status>
        <hugepageSize>2M</hugepageSize>
        <hugepageNum>48000</hugepageNum>
    </conf>
    <support>
        <option>1G</option>
        <suggestValue>62</suggestValue>
        <minValue>4</minValue>
        <maxValue>93</maxValue>
    </support>
    <support>
        <option>2M</option>
        <suggestValue>32000</suggestValue>
        <minValue>2048</minValue>
        <maxValue>48000</maxValue>
    </support>
</hugePageInfo>
元素含义
各子元素
live 已生效的大页配置。
自定义类型。
status 大页配置状态，1-启用，0-不启用。
整型（int)类型。
hugepageSize 页大小。
字符串（String)类型。
hugepageNum 页数。
整型（int)类型。
conf 最新的大页配置，但可能未重启主机生效。。
自定义类型。
support 支持配置的大页选项。
整型（int)类型。
option 支持配置的页大小，如 1G、2M。
字符串（String)类型。
suggestValue 建议的页数。
整型（int)类型。
minValue 最小可配置的页数。
整型（int)类型。
maxValue 最大可配置的页数。
整型（int)类型。
