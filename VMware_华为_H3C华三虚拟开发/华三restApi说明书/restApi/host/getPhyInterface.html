H3C CAS云管理平台

查询虚拟交换机物理接口信息

根据主机id查询虚拟交换机物理接口信息。

接口 URI

/cas/casrs/host/pnic?id={id}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 主机id 。
长整数（long）类型。必须。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Content-Type: application/xml
Content-Length: 193
Date: Wed, 17 Sep 2014 09:41:54 GMT
Server: H3C-CVM
消息体
<?xml version="1.0" encoding="UTF-8"?>
<list>
  <trafficInfo>
    <vPortName>eth0</vPortName>
  </trafficInfo>
  <trafficInfo>
    <vPortName>eth1</vPortName>
  </trafficInfo>
  <trafficInfo>
    <vPortName>eth2</vPortName>
  </trafficInfo>
......

</list>
元素含义
trafficInfo各子元素
vPortName 物理接口名称
字符串（String）类型。
应用举例

查询主机id为4的虚拟交换机物理接口信息

请求
GET http://localhost:8080/cas/casrs/host/pnic?id=4
accept: application/xml
...
回应
HTTP/1.1 200 OK
Content-Type: application/xml
...
<?xml version="1.0" encoding="UTF-8"?>
<list>
  <trafficInfo>
    <vPortName>eth0</vPortName>
  </trafficInfo>
  <trafficInfo>
    <vPortName>eth1</vPortName>
  </trafficInfo>
  <trafficInfo>
    <vPortName>eth2</vPortName>
  </trafficInfo>

</list>
