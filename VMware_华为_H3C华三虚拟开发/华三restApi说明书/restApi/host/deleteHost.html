H3C CAS云管理平台

删除主机

删除指定id的主机。

接口 URI

/cas/casrs/host/delete

访问方法

HTTP 方法
DELETE
消息体
<?xml version="1.0" encoding="UTF-8"?>
<taskMsg>
  <msgId>1410943942352</msgId>
  <completed>false</completed>
  <progress>0</progress>
  <eventType>0</eventType>
</taskMsg>
元素含义
taskMsg各子元素
msgId 消息ID。
整数（Lomg）类型。
completed 任务是否完成。
布尔（Boolean）类型。
eventType 事件类型。。
整型（Int）类型。
progress 进度。
整型（Int）类型。

返回结果

状态码
  • 200(OK)：成功。
消息头

HTTP/1.1 200 OK
Content-Type: application/xml
Content-Length: 176
Date: Thu, 18 Sep 2014 01:38:49 GMT
Server: H3C-CVM

应用举例

请求
DELETE http://localhost:8080/cas/casrs/host/delete?id=113
accept: application/xml
