修改主机CPU隔离配置

根据主机id修改主机CPU隔离配置。

接口 URI

/cas/casrs/host/{id}/isolatedCPU

访问方法

HTTP 方法
PUT
消息体
<isolatedCpuConfig>
    <isolatedCpus>12,15,33</isolatedCpus>
</isolatedCpuConfig>
参数
路径参数（Path Parameter）
id 主机id 。
长整数（long）类型。必须。
元素含义
各子元素
isolatedCpus 将要配置隔离的CPU号，以逗号分隔。为空表示不做CPU隔离。
字符串（String)类型。

返回结果

状态码
  • 204(No Content)：成功。

应用举例

修改主机id为6的CPU隔离配置。

请求
PUT http://localhost:8080/cas/casrs/host/6/isolatedCPU
accept: application/xml
...       
<isolatedCpuConfig>
    <isolatedCpus>12,15,33</isolatedCpus>
</isolatedCpuConfig>
回应
HTTP/1.1 204 No Content
