H3C CAS云管理平台

查询主机FC HBA(存储适配器)信息

根据主机id查询该主机FC HBA(存储适配器)信息。

接口 URI

/cas/casrs/host/fc/query

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 主机id 。
长整数（long）类型。必须。

返回结果

状态码
  • 200(OK)：成功。

元素含义
HbaInfo各子元素
localNodeName 本地节点。
字符串（String）类型。
status 状态 活动:Online 不活动Offline。
字符串（String）类型。
localPortName 本地端口。
字符串（String）类型。
localPortName 本地端口。
字符串（String）类型。
remoteNodeName 目的节点。
字符串（String）类型。
remotePortName 目的端口。
字符串（String）类型。
capacity 速率 单位G。
字符串（String）类型。
应用举例

启动主机id为7，名称为eth1的物理网卡。

GET http://localhost:8080/cas/casrs/host/fc/query?id=3
