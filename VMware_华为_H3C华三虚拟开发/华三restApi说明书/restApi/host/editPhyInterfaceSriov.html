修改主机下指定物理网卡SR-IOV

修改主机下指定物理网卡SR-IOV。

接口 URI

/cas/casrs/host/{id}/interface/{name}/sriov

访问方法

HTTP 方法
PUT
参数
路径参数（Path Parameter）
id 主机id 。
长整数（long）类型。必须。
name 物理网卡名称。
字符串（String）类型。必须。
消息体
<sriovInfo>
   <vfCount>2</vfCount>
</sriovInfo>
元素含义
sriovInfo各子元素
vfCount 设置的虚拟网卡数。0表示禁用SR-IOV
整数（int）类型。

返回结果

状态码
  • 200(OK)：成功。
消息头
<?xml version="1.0" encoding="UTF-8"?>
  <rsResult>
    <state>0</state>
    <errorCode>0</errorCode>
    <message>启用主机“host1”上物理网卡“eth1”的SR-IOV成功。</message>
  </rsResult>
元素含义
rsResult各子元素
state 执行状态。 0： 成功，1：失败
整型（Integer）类型。
errorCode 错误码。
整型（Integer）类型。
message 结果信息。
字符串（String）类型。

 

应用举例

1.修改ID为“1”的主机上名称为“eth1”的物理网卡的SR-IOV。

请求
PUT http://localhost:8080/cas/casrs/host/1/interface/eth1/sriov
accept: application/xml

<sriovInfo>
   <vfCount>2</vfCount>
</sriovInfo>
回应
HTTP/1.1 200 OK
Content-Type: application/xml
Content-Length: 181
Date: Wed, 17 Sep 2014 09:46:58 GMT
Server: H3C-CVM
<?xml version="1.0" encoding="UTF-8"?>
  <rsResult>
    <state>0</state>
    <errorCode>0</errorCode>
    <message>启用主机“host1”上物理网卡“eth1”的SR-IOV成功。</message>
  </rsResult>
