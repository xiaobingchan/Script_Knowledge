H3C CAS云管理平台

查询主机某一个虚拟交换机信息

根据主机id查询主机某一个虚拟交换机信息。

接口 URI

/cas/casrs/host/id/{hostId}/vswitch/{vswitchId}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
hostId 主机id 。
长整数（long）类型。必须。
vswitchId 虚拟交换机id 。
长整数（long）类型。必须。

返回结果

状态码
  • 200(OK)：成功。

元素含义
vSwitch各子元素
id 虚拟交换机ID。
长整数（long）类型。
name 虚拟交换机名称。
字符串（String）类型。
portNum 端口数量。
整数（int）类型。
mode 转发模式。0 : veb; 1: vepa; 2:多通道。
整数（int）类型。
pnic vSwitch绑定的所有物理网卡名。
字符串（String）类型。
status 虚拟机交换状态。0:不活动 1:活动。
整数（int）类型。
address IP地址。
字符串（String）类型。
netmask 网络掩码。
字符串（String）类型。
gateway 网关。
字符串（String）类型。
enableLacp 是否启用LACP(启用：true，未启用：false。
布尔（Boolean）类型。
vNIC各子元素
vsName 虚拟交换机名称。
字符串（String）类型。
vPortName 虚拟端口名称。
字符串（String）类型。
vmName 虚拟机名称。
字符串（String）类型。
vmMac 虚拟机MAC地址。
字符串（String）类型。
rxPkts 接收报文数。
长整数（long）类型。
rxBytes 接收字节数。
长整数（long）类型。
rxErrs 接收错包数 。
长整数（long）类型。
txPkts 发送报文数。
长整数（long）类型。
txBytes 发送字节数。
长整数（long）类型。
txErrs 发送错包数。
长整数（long）类型。
