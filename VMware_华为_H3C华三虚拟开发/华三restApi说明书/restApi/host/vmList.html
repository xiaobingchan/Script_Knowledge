H3C CAS云管理平台

查询主机所有虚拟机

根据主机id查询指定物理主机下的所有虚拟机。

接口 URI

/cas/casrs/host/id/{hostId}/vm

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
hostId 主机id 。
长整数（long）类型。必须。

返回结果

状态码
  • 200(OK)：成功。

元素含义
domain各子元素
id 虚拟机 ID。
长整数（long）类型。
hostId 运行的主机ID。
长整数（long）类型。
clusterId 所属的集群 ID。
长整数（long）类型。
hostPoolId 所属的主机池 ID。
长整数（long）类型。
name 虚拟机名称。
字符串（String）类型。
description 虚拟机描述信息。
字符串（String）类型。
memory 虚拟机内存大小，单位（M） 。
整数（int）类型。
cpu 虚拟机CPU个数。
整数（int）类型。
status 虚拟机状态。 2：运行 3：关闭 4：暂停。
整数（int）类型。
uuid 虚拟机UUID。
字符串（String）类型。
osBit 虚拟机安装的操作系统位数。
字符串（String）类型。
system 虚拟机安装的操作系统类型。0：Windows 1：Linux。
整数（int）类型。
osDesc 虚拟机安装的操作系统描述。
字符串（String）类型。
flag 虚拟机所属标志 0:不属于用户以及用户组 1:属于用户 2:属于用户组。
整数（int）类型。
type 虚拟机类型 0:CAS虚拟机 1:VMware虚拟机。
整数（int）类型。
deployed 是否快速部署过。
布尔（Boolean）类型。
existRaw 是否存在raw格式。
布尔（Boolean）类型。
autoMigrate 是否允许自动迁移。 0：不允许 1：允许。
整数（int）类型。
createDate 虚拟机创建日期。
日期（Date）类型。
storageCapacity 虚拟机镜像存储大小。 单位（G）
长整数（long）类型。
