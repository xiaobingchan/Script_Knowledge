修改主机IOMMU状态

修改主机IOMMU状态。

接口 URI

/cas/casrs/host/modifyHostIommuConf?hostId={hostId}&iommuState={iommuState}

访问方法

HTTP 方法
PUT
参数

查询参数（Query Parameter）
hostId 主机id。
长整数（Long）类型。必须。
iommuState IOMMU状态标识。
字符串（String）类型。必须。1为启用，0为禁用。

返回结果

状态码
  • 200：成功
 
消息头

Content-Type: application/xml
Content-Length: 176
Date: Wed, 07 Mar 2018 07:59:11 GMT
Server: H3C-CVM
消息体
<?xml version="1.0" encoding="UTF-8"?><rsResult>
  <state>0</state>
  <errorCode>0</errorCode>
  <message>启用主机“cvk35上的IOMMU成功。</message>
</rsResult>
