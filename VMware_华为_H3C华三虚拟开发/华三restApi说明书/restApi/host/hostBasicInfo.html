H3C CAS云管理平台

查询指定主机的概要信息

根据主机id查询指定物理主机的概要信息。

接口 URI

/cas/casrs/host/summary/{id}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 主机id 。
长整数（long）类型。必须。

返回结果

状态码
  • 200(OK)：成功。
消息头

Content-Type: application/xml
Content-Length: 607

Server: H3C-CVM

元素含义
host各子元素
name 物理机名称。
字符串（String）类型。
ip 物理机ip。
字符串（String）类型。
model 服务器型号。
字符串（String）类型。
vendor 服务器制造商。
字符串（String）类型。
cpuModel CPU型号。
字符串（String）类型。
cpuFrequence CPU频率，单位为MHz。
整型（Int）类型。
rate 虚拟机网络吞吐量（单位MBps）。
小数（double）类型。
diskSize 磁盘大小，单位为MB。
长整数（long）类型。
memorySize 内存大小，单位为MB。
长整数（long）类型。
cpuNum CPU总数 个数*核数。
字符串（String）类型。
vmNum 主机下虚拟机 启动关闭信息。
整型（Int）类型。
cpuRate CPU利用率
小数（double）类型。
memRate 内存利用率
小数（double）类型。
status 状态，取值：0为不正常，1为正常，2为主机不在HA集群中，3为维护状态。
整型（Int）类型。
runTime 运行时间。
字符串（String）类型。
version 主机版本。
字符串（String）类型。
cpuFrequenceGhz CPU频率，单位为GHz。用于前台显示概要信息。
字符串（String）类型。
memory 内存大小 带单位 用于前台显示概要信息。
字符串（String）类型。
storage 磁盘大小 带单位 用于前台显示概要信息。
字符串（String）类型。
