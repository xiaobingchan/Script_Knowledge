H3C CAS云管理平台

查询CAS系统中主机下所有虚拟机列表

根据主机ID查询CAS系统中主机下所有虚拟机列表

接口 URI

/cas/casrs/vm/vmList?hostId={hostId}&offset={offset}&limit={limit}&sortField={sortField}&sortDir={sortDir}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
hostId 主机id 。
长整数（long）类型。必须。
offset 返回的第一条记录在总结果集中的位置。
整数（int）类型，非必须。
limit 返回记录总数。
整数（int）类型，非必须。
sortField 排序字段 。
字符串（String）类型，非必须。id: 虚拟机ID，title：虚拟机显示名称，status: 虚拟机状态，cpu: 虚拟机CPU个数，memory: 虚拟机内存大小；若传入其他字段，将以"id"进行排序。
sortDir 排序类型 。
字符串（String）类型，非必须。Asc：升序，Desc：降序

元素含义
domain各子元素
id 虚拟机 ID。
长整型（Long）类型。
hostId 主机 ID。
长整型（Long）类型。
clusterId 集群 ID。
长整型（Long）类型。
hostPoolId 主机池 ID。
长整型（Long）类型。
name 虚拟机名称。
字符串（String）类型。
title 虚拟机显示名称。
字符串（String）类型。
hostName 主机名称。
字符串（String）类型。
description 虚拟机描述。
字符串（String）类型。
memory 虚拟机内存。
长整型（Long）类型，单位MB。
cpuRate 虚拟机CPU利用率。
小数（double）类型。
cpu 虚拟机CPU利用率。
小数（double）类型。
memRate 虚拟机虚拟CPU个数（CPU个数 * CPU核数） 。
整型（Integer）类型。
vmStatus 虚拟机状态。
字符串（String）类型，shutOff:关闭，shutdown:关闭，running:运行，paused:暂停，unknown:未知
uuid 虚拟机UUID。
字符串（String）类型。
system 虚拟机安装的操作系统，枚举值：0：Windows，1：Linux。
整型（Integer）类型
osDesc 虚拟机安装的操作系统描述。
字符串（String）类型。
flag 虚拟机所属标志 0:不属于用户以及用户组 1:属于用户 2:属于用户组。
整型（Integer）类型
type 虚拟机类型 0:CAS虚拟机 1:VMware虚拟机。
整型（Integer）类型
deployed 是否快速部署过。
布尔（Boolean）类型
existRaw 是否存在raw格式。
布尔（Boolean）类型
haStatus 虚拟机HA状态，枚举值，0:正常 1:未加入HA 2:已加入HA，配置文件不同步
整型（Integer）类型
haManage 虚拟机是否被HA管理，枚举值，0：未被HA管理，1：被HA管理。
整型（Integer）类型
