H3C CAS云管理平台

查询虚拟交换机端口流量的详细信息

根据端口名称vPort和虚拟交换机id查询虚拟交换机端口流量的详细信息

接口 URI

/cas/casrs/host/vport/traffic?vPort={vPort}&vsId={vsId}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
vPort 端口名称
字符串（String）。必须。
vsId 虚拟交换机ID
整数（Long）类型。必须。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Content-Type: application/xml
Content-Length: 193
Date: Wed, 17 Sep 2014 09:41:54 GMT
Server: H3C-CVM

<list>

</list>
元素含义
trendRate各子元素
name 接受/发送流量(Mbps)
字符串（String）类型。
rates各子元素
time 时间
长整数（long）类型。
rate 流量
小数（double）类型。
