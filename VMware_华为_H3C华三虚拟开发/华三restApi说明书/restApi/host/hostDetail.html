H3C CAS云管理平台

查询指定物理主机详细信息

根据主机id查询指定物理主机的详细信息。

接口 URI

/cas/casrs/host/id/{id}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 主机id 。
长整数（long）类型。必须。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
消息体
<?xml version="1.0" encoding="UTF-8"?>
<host type="kvm">
  <id>21</id>
  <name>vmc-8</name>
  <ip>192.168.0.8</ip>
  <model>HP Compaq 8280 Elite CMT PC</model>
  <vendor>Hewlett-Packard</vendor>
  <cpuCount>4</cpuCount>
  <cpuModel>Intel(R) Core(TM) i5-2400 CPU @ 3.10GHz</cpuModel>
  <cpuFrequence>1600</cpuFrequence>
  <diskSize>439882</diskSize>
  <memorySize>7875</memorySize>
  <cpuSockets>2</cpuSockets>
  <cpuCores>2</cpuCores>
 <pNIC>
    <name>eth0</name>
    <description>Intel Corporation 82579LM Gigabit Network Connection</description>
    <status>1</status>
    <macAddr>80:c1:6e:e2:1b:d8</macAddr>
  </pNIC>
  <pNIC>
    <name>eth1</name>
    <description>Intel Corporation 82574L Gigabit Network Connection</description>
    <status>1</status>
    <macAddr>68:05:ca:00:8a:b6</macAddr>
  </pNIC>
  <serialNumber>6CU429FWKW</serialNumber>
</host>
元素含义
host各子元素
id 主机 ID。
长整数（long）类型。
type 服务器支持的Hypervisor类型。如 “kvm”、“xen”、“esx”
字符串（String）类型。
name 主机名称。
字符串（String）类型。
ip 服务器管理IP地址。
字符串（String）类型。
model 服务器型号。
字符串（String）类型。
vendor 服务器制造商。
字符串（String）类型。
cpuCount CPU个数 。
整数（int）类型。
cpuFrequence CPU频率，单位为MHz。
整数（int）类型。
cpuModel CPU型号。
字符串（String）类型。
diskSize 磁盘大小，单位为MB。
长整数（long）类型。
memorySize 内存大小，单位为MB。
长整数（long）类型。
cpuSockets cpu 个数。
整数（int）类型。
cpuCores cpu 核数。
整数（int）类型。
serialNumber 服务器序列号，Host唯一标识。
字符串（String）类型。
pNIC各子元素
name 物理网卡名称，和ifconfig命令看到的结果相同。
字符串（String）类型。
description 物理网卡详细信息。
字符串（String）类型。
macAddr 物理网卡的MAC地址。
字符串（String）类型。
status 物理网卡状态，1-活动；0-不活动。
整数（int）类型。
