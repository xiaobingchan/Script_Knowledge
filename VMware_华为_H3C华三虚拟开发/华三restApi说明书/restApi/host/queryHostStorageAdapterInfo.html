H3C CAS云管理平台

查询主机存储适配器信息

根据主机id查询该主机存储适配器信息。

接口 URI

/cas/casrs/storageAdapter/adapter/query?id={id}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 主机id。
长整数（long）类型。必须。
wwn 存储设备器标示符。
字符串（String）类型。可选。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Content-Type: application/xml
Content-Length: 193
Date: Tue, 16 Jun 2015 10:09:41 GMT
Server: H3C-CVM
消息体
<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<storageAdapters>
  <adapter>
    <name>Local-SCSI</name>
    <model>Local SCSI</model>
    <type>SCSI</type>
    <status>Online</status>
    <wwn>
    </wwn>
    <speed>
    </speed>
  </adapter>
  <adapter>
    <name>Open-iSCSI</name>
    <model>iSCSI Software Adapter</model>
    <type>iSCSI</type>
    <status>Online</status>
    <wwn>iqn.1993-08.org.debian:01:vcvk245</wwn>
    <speed>
    </speed>
  </adapter>
</storageAdapters>
元素含义
StorageAdapter各子元素
name 存储适配器名称。
字符串（String）类型。
model 存储适配器型号。
字符串（String）类型。
type 存储适配器类型。
字符串（String）类型。
status 存储适配器状态。
字符串（String）类型。
wwn 存储适配器的标示符。
字符串（String）类型。
speed 存储适配器的速率 。
字符串（String）类型。
