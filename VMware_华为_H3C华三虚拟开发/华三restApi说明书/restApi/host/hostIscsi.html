查询指定主机的iscsi信息

根据主机id查询指定物理主机下的iscsi信息。

接口 URI

/cas/casrs/host/iscsi

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 主机id 。
长整数（long）类型。必须。

返回结果

状态码
  • 200(OK)：成功。
消息头

HTTP/1.1 200 OK
消息体
<?xml version="1.0" encoding="UTF-8"?>
<rsResult>
  <state>0</state>
  <errorCode>0</errorCode>
  <data>iqn.1993-08.org.debian:01:801c3235d319</data>
</rsResult>
元素含义
rsResult各子元素
state 状态
整数（int）类型。
errorCode 错误码。
整数（int）类型。
data iscsi值。
字符串（String）类型。

应用举例

1.查询主机id为2的iscsi信息。

请求
GET http://localhost:8080/cas/casrs/host/iscsi?id=2
accept: application/xml
...
回应
<?xml version="1.0" encoding="UTF-8"?>
<rsResult>
  <state>0</state>
  <errorCode>0</errorCode>
  <data>iqn.1993-08.org.debian:01:801c3235d319</data>
</rsResult>
