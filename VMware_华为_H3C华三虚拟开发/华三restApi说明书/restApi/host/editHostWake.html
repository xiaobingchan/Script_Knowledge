修改主机唤醒方式

修改指定主机远程唤醒方式

接口 URI

/cas/casrs/host/editWake

访问方法

HTTP 方法
PUT
消息体
<?xml version="1.0" encoding="UTF-8"?>
<host>
  <id>91</id>
  <wakeCategory>1</wakeCategory>
  <ipmiIpaddr>192.168.10.84</ipmiIpaddr>
  <ipmiUser>root</ipmiUser>
  <ipmiPw>password</ipmiPw>
</host>
元素含义
host各子元素
id 主机id
长整数（long）类型。必须。
wakeCategory 远程唤醒方式，0为网络唤醒，1为IPMI唤醒
整数（int）类型。必须。
ipmiIpaddr IPMI唤醒主机地址
字符串（String）类型，可选项。
ipmiUser IPMI唤醒主机用户
字符串（String）类型，可选项。
ipmiPw IPMI唤醒主机密码
字符串（String）类型，可选项。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Content-Type: application/xml
Content-Length: 193
Date: Wed, 17 Sep 2014 09:41:54 GMT
Server: H3C-CVM
消息体
<?xml version="1.0" encoding="UTF-8"?>
<rsResult>
  <state>0</state>
  <errorCode>0</errorCode>
  <data>1</data>
  <message>编辑主机“cvm010”唤醒方式成功。</message>
</rsResult>
元素含义
rsResult各子元素
state 状态。
整型（Int）类型。
errorCode 错误代码。
整型（Int）类型。
data 主机状态
字符串（String）类型。0为主机异常，1为主机正常，3为主机进入维护模式
message 消息
字符串（String）类型。
