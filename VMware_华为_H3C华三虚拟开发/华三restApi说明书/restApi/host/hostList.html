H3C CAS云管理平台

查询CAS系统中所有物理主机列表

查询CAS系统中所有管理的主机列表。

接口 URI

/cas/casrs/host/

访问方法

HTTP 方法
GET
参数

无

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Server: Apache-Coyote/1.1
Set-Cookie: JSESSIONID=CC6193BACB2F99D2D4CEC754B2676600; Path=/cas
Content-Type: application/xml
Content-Length: 141
Date: Fri, 02 Nov 2012 08:40:47 GMT
消息体
<?xml version="1.0" encoding="UTF-8"?>
<list>
  <host type="kvm">
    <id>21</id>
    <name>vmc-8</name>
    <ip>192.168.0.8</ip>
  </host>
  ...
</list>
元素含义
host各子元素
id 主机 ID。
长整数（long）类型。
name 主机名称。
字符串（String）类型。
type 服务器支持的Hypervisor类型。如 “kvm”、“xen”、“esx”
字符串（String）类型。
ip 服务器管理IP地址。
字符串（String）类型。

 

应用举例

1.查询 CAS系统中所有管理的主机列表。

请求
GET http://localhost:8080/cas/casrs/host/
accept: application/xml
...
