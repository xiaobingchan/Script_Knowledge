暂停虚拟交换机

暂停指定虚拟交换机

接口 URI

/cas/casrs/vswitch/pause?id={id}&hostName={hostName}&netName={netName}

访问方法

HTTP 方法
GET
参数
查询参数（ Query  Parameter）
id 主机id 。
长整数（long）类型。必须。
hostName 主机名称 。
字符串（String）类型。必须。
netName 虚拟交换机名称 。
字符串（String）类型。必须。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Content-Type: application/xml
Transfer-Encoding: chunked
Date: Wed, 17 Sep 2014 01:17:17 GMT
消息体
<?xml version="1.0" encoding="UTF-8"?>
<rsResult>
  <state>0</state>
  <errorCode>0</errorCode>
  <message>暂停主机“cvk89”上网络“vswitch12”成功。</message>
</rsResult>
元素含义
rsResult各子元素
state 状态。
整数（int）类型。
errorCode 错误码 。
整数（int）类型。
message 返回的信息。
字符串（String）类型。
 

应用举例

暂停主机id为55，主机名称为cvk89，名称为vswitch12的虚拟交换机。

请求
GET http://localhost:8080/cas/casrs/vswitch/pause?id=55&hostName=cvk89&netName=vswitch12
accept: application/xml
...
回应
HTTP/1.1 200 OK
Content-Type: application/xml
Content-Length: 191

...

  <state>0</state>
  <errorCode>0</errorCode>
  <message>暂停主机“cvk89”上网络“vswitch12”成功。</message>
</rsResult>
