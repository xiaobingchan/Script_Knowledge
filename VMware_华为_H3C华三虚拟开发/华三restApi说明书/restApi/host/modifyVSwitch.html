修改虚拟交换机

修改虚拟交换机。

接口 URI

/cas/casrs/vswitch

访问方法

HTTP 方法
PUT
消息体
<vSwitch>
	<id>4</id>
	<hostId>2</hostId>
	<name>vswitch0</name>
	<portNum>32</portNum>
	<mode>0</mode>
	<pnic>eth0</pnic>
	<status>1</status>
	<address>192.168.0.8</address>
	<netmask>255.255.255.0</netmask>
	<gateway>192.168.0.1</gateway>
	<enableLacp>false</enableLacp>
	<bondMode>balance-tcp</bondMode>
	<description>test vswitch</description>
	<vlanId>6</vlanId>
      <mtu>1500</mtu>
</vSwitch>
元素含义
vSwitch各子元素
id 虚拟交换机id。
长整型（long）类型。
hostId 主机id。
长整型（long）类型。
name 虚拟交换机名称。
字符串（String）类型。
portNum 端口数量。
整数（int）类型。
mode 转发模式。0 : veb; 1: vepa; 2:多通道。
整数（int）类型。
pnic vSwitch绑定的所有物理网卡名。可以为多个， 用英文的逗号分隔，如 <pnic>eth0,eth1<pnic>
字符串（String）类型。
status 虚拟机交换状态。0:不活动 1:活动。
整数（int）类型。
address IP地址。
字符串（String）类型。
netmask 网络掩码。
字符串（String）类型。
gateway 网关。
字符串（String）类型。
enableLacp 链路聚合模式(动态链路聚合：true，静态链路聚合：false。
布尔（Boolean）类型。
bondMode 负载分担模式。枚举类型，启用lacp:balance-slb ，balance-tcp；不启用lacp:balance-slb ，balance-tcp, active-backup。
balance-slb : 基本负载分担,balance-tcp : 高级负载分担,active-backup : 主备负载分担
字符串（String）类型。
description 描述信息。
字符串（String）类型。
vlanId VLAN ID。虚拟交换机端口所在VLAN的ID。VLAN ID的取值范围为2～4094。
整数（int）类型。
mtu MTU值，取值范围1000-9000。
整数（int）类型。

返回结果

状态码
  • 204(No Content)：成功；404(Not Found)：不存在或已删除：
消息头

HTTP/1.1 204 No Content
Content-Type: application/xml
