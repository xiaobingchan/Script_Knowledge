H3C CAS云管理平台

修改主机

修改主机。

接口 URI

/cas/casrs/host/edit

访问方法

HTTP 方法
PUT
消息体
<host>
    <id>112</id>
    <user>root</user>
    <pwd>1q2w3e</pwd>
    <name>cvm80</name>
</host>
元素含义
host各子元素
id

主机id。

长整型（Iong）类型，必须。
user 访问主机用户名。
字符串（String）类型，必须。
pwd 访问主机密码。
字符串（String）类型，必须。
name 服务器IP地址。
字符串（String）类型，必须。

返回结果

状态码
  • 200(OK)：成功。
消息头
<?xml version="1.0" encoding="UTF-8"?>
<rsResult>
  <state>0</state>
  <errorCode>0</errorCode>
  <data>1</data>
  <message>编辑主机“cvm80”成功。</message>
</rsResult>
元素含义
rsResult各子元素
state 执行状态。 0： 成功，1：失败，2，部分成功
整型（Integer）类型。
errorCode 错误码。
整型（Integer）类型。
message 成功信息。
字符串（String）类型。

 

应用举例

1.修改ID为“112”的主机。

请求
PUT http://localhost:8080/cas/casrs/host/edit
accept: application/xml


<host>
    <id>112</id>
    <user>root</user>
    <pwd>1q2w3e</pwd>
    <name>cvm80</name>
</host>
