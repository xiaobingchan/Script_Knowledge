H3C CAS云管理平台

查询主机CPU隔离配置

根据主机id查询主机CPU隔离配置。

接口 URI

/cas/casrs/host/{id}/isolatedCPU

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 主机id 。
长整数（long）类型。必须。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Content-Type: application/xml
Content-Length: 193
Date: Wed, 17 Sep 2014 09:41:54 GMT
Server: H3C-CVM
消息体
<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<isolatedCpuConfig>
    <liveStatus>true</liveStatus>
    <confStatus>false</confStatus>
    <isolatedCpus>12,15,33</isolatedCpus>
</isolatedCpuConfig>
元素含义
各子元素
liveStatus 是否有生效的隔离CPU设置。true：是，false：否
布尔（Boolean)类型。
confStatus 是否配置了隔离但未生效的CPU。 true：是，false：否
布尔（Boolean)类型。
isolatedCpus 已经配置且生效的隔离CPU号，以逗号分隔。
字符串（String)类型。

应用举例

查询主机id为6的主机CPU隔离配置信息。

请求
GET http://localhost:8080/cas/casrs/host/6/isolatedCPU
accept: application/xml
...
回应
HTTP/1.1 200 OK
Content-Type: application/xml
...
<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<isolatedCpuConfig>
    <liveStatus>true</liveStatus>
    <confStatus>false</confStatus>
    <isolatedCpus>12,15,33</isolatedCpus>
</isolatedCpuConfig>
