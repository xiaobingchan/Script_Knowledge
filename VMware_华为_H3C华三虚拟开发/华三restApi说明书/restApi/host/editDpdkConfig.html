H3C CAS云管理平台

修改主机DPDK配置信息

根据主机id修改DPDK配置信息。

接口 URI

/cas/casrs/host/{id}/dpdkConfig

访问方法

HTTP 方法
PUT
消息体
<dpdkConfig>
    <status>true</status>
    <numaNodeInfo>
        <numaNode>1</numaNode>
        <bindCpu>18</bindCpu>
        <bindCpu>33</bindCpu>
        <bindMem>3072</bindMem>
    </numaNodeInfo>
    <numaNodeInfo>
        <numaNode>0</numaNode>
        <bindCpu>6</bindCpu>
        <bindCpu>21</bindCpu>
        <bindCpu>22</bindCpu>
        <bindMem>2048</bindMem>
    </numaNodeInfo>
</dpdkConfig>
参数
路径参数（Path Parameter）
id 主机id 。
长整数（long）类型。必须。
元素含义
各子元素
status DPDK状态。true：启用，false：未启用
布尔（Boolean)类型。
numaNodeInfo 主机每个NUMA节点配置信息。
自定义类型。
numaNode NUMA节点号。
整型（int)类型。
bindCpu NUMA节点中设置了绑定的CPU号。
整型（int)类型。
bindMem NUMA节点绑定的内存大小，单位MB。
整型（int)类型。

返回结果

状态码
  • 204(No Content)：成功。

应用举例

修改主机id为6的主机DPDK配置信息。

请求
PUT http://localhost:8080/cas/casrs/host/6/dpdkConfig
accept: application/xml
...       
 <dpdkConfig>
    <status>true</status>
    <numaNodeInfo>
        <numaNode>1</numaNode>
        <bindCpu>18</bindCpu>
        <bindCpu>33</bindCpu>
        <bindMem>3072</bindMem>
    </numaNodeInfo>
    <numaNodeInfo>
        <numaNode>0</numaNode>
        <bindCpu>6</bindCpu>
        <bindCpu>21</bindCpu>
        <bindCpu>22</bindCpu>
        <bindMem>2048</bindMem>
    </numaNodeInfo>
</dpdkConfig>
