H3C CAS云管理平台

查询主机DPDK配置信息

根据主机id查询DPDK配置信息。

接口 URI

/cas/casrs/host/{id}/dpdkConfig

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 主机id 。
长整数（long）类型。必须。

返回结果

元素含义
各子元素
status DPDK状态。true：启用，false：未启用
布尔（Boolean)类型。
numaNodeInfo 主机每个NUMA节点配置信息。
自定义类型。
numaNode NUMA节点号。
整型（int)类型。
numaCpu NUMA节点中的CPU号。
整型（int)类型。
bindCpu NUMA节点中设置了绑定的CPU号。
整型（int)类型。
bindMem NUMA节点绑定的内存大小，单位MB。
整型（int)类型。
