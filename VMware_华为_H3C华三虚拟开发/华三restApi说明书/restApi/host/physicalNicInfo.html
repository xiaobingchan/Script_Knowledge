H3C CAS云管理平台

查询指定物理主机所有物理网卡信息

根据主机id查询指定物理主机的所有物理网卡信息。

接口 URI

/cas/casrs/host/allPhy

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 主机id 。
长整数（long）类型。必须。

返回结果

状态码
  • 200(OK)：成功。

元素含义
pNIC各子元素
name 网卡名称。
字符串（String）类型。
description 描述。
字符串（String）类型。
status 状态。取值： 1-活动；0-不活动 -1未知。
整数（int）类型。
macAddr MAC地址。
字符串（String）类型。
speed 速率。
字符串（String）类型。
duplex 双工。
字符串（String）类型。
carrier 载波。
字符串（String）类型。
Mtu Mtu。
字符串（String）类型。
