H3C CAS云管理平台

查询主机CPU利用率饼图

根据主机id查询指定物理主机的CPU利用率饼图。

接口 URI

/cas/casrs/host/cpuPie

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 主机id 。
长整数（long）类型。必须。

返回结果

状态码
  • 200(OK)：成功。
消息头

Content-Type: application/xml
Content-Length: 3114
Date: Wed, 17 Sep 2014 03:28:50 GMT
Server: H3C-CVM
消息体
14.268041237113403
