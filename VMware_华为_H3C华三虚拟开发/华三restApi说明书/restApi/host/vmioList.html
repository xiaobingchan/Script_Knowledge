查询主机所有虚拟机的IO吞吐量

根据主机id查询指定物理主机下的所有虚拟机IO吞吐量。

接口 URI

/cas/casrs/host/id/{hostId}/vmio

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
hostId 主机id 。
长整数（long）类型。必须。

返回结果

状态码
  • 200(OK)：成功。
消息头

HTTP/1.1 200 OK
Total: 7
消息体
<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<list>
  <vmio>
    <id>670</id>
    <rate>2.53</rate>
    <time>1353382888</time>
  </vmio>
  <vmio>
    <id>671</id>
    <rate>1.2</rate>
    <time>1353382888</time>
  </vmio>
.......
</list>
元素含义
vmio各子元素
id 虚拟机 ID。
长整数（long）类型。
rate 虚拟机吞吐量（单位KBps）。
小数（double）类型。
time 数据对应的时间戳。
长整数（long）类型。

 

应用举例

1.查询主机id为2的所有虚拟机IO吞吐量。

请求
GET http://localhost:8080/cas/casrs/host/id/2/vmio
accept: application/xml
...
回应
HTTP/1.1 200 OK
Content-Type: application/xml
...
