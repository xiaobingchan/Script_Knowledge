H3C CAS云管理平台

增加主机

增加主机。

接口 URI

/cas/casrs/host/add

访问方法

HTTP 方法
POST
消息体
<host>
<user>root</user>
<pwd>1q2w3e</pwd>
<hostPoolId>1</hostPoolId>
<clusterId>4</clusterId>
<name>192.168.10.80</name>
<enableHA>1</enableHA>
<ignore>false</ignore>
</host>
元素含义
host各子元素
user 访问主机用户名。
字符串（String）类型，必须。
pwd 访问主机密码。
字符串（String）类型，必须。
hostPoolId 主机池ID。
长整型（Iong）类型，必须。
clusterId 集群ID。
长整型（Iong）类型，可选。
name 服务器IP地址。
字符串（String）类型，必须。
enableHA

是否启用HA，1：启用，0：未启用，结合clusterID使用。

整型（Integer）类型，必须。
ignore 是否忽略重复管理。
布尔 （Boolean）类型，必须。

返回结果

状态码
  • 200(OK)：成功。
消息头
<?xml version="1.0" encoding="UTF-8"?>
<taskMsg>
  <msgId>1410943942346</msgId>
  <completed>false</completed>
  <progress>0</progress>
  <eventType>0</eventType>
</taskMsg>
元素含义
返回增加主机任务id，可根据任务id查询任务进度。

应用举例

1.增加IP为“192.168.10.80”的主机。

请求
POST http://localhost:8080/cas/casrs/host/add
accept: application/xml


<host>
<user>root</user>
<pwd>1q2w3e</pwd>
<hostPoolId>1</hostPoolId>
<name>192.168.10.80</name>
<enableHA>1</enableHA>
<ignore>false</ignore>
</host>
