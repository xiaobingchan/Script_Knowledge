H3C CAS云管理平台

查询主机磁盘I/O趋势数据

根据主机ID查询主机磁盘I/O趋势数据

接口 URI

/cas/casrs/host/{hostid}/diskIOTrend?interval={interval}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
hostId 主机Id
整数（Long）类型。必须。
interval 区间时间，单位为分
整型（Integer）类型。可选项。

返回结果

状态码
  • 200(OK)：成功。

HTTP/1.1 200 OK
Content-Type: application/xml
Content-Length: 193
Date: Wed, 17 Sep 2014 09:41:54 GMT
Server: H3C-CVM

<list>

元素含义
trendRate各子元素
name 磁盘名称
字符串（String）类型。
rates各子元素
time 时间
长整数（long）类型。
rate I/O吞吐量
小数（double）类型。

查询主机ID为4，时间间隔为30分钟的主机磁盘I/O趋势数据

请求
GET http://localhost:8080/cas/casrs/host/4/diskIOTrend?interval=30
accept: application/xml
