H3C CAS云管理平台

查询主机上的USB设备

查询指定主机id的USB设备。

接口 URI

/cas/casrs/usb/{hostId}

访问方法

HTTP 方法
GET
消息体
<?xml version="1.0" encoding="UTF-8"?>
<host>
  <id>3</id>
  <name>cvk04</name>
  <usb>
    <name>usb_usb1</name>
    <value>usb_usb1 Linux 3.13.6 ehci_hcd EHCI Host Controller</value>
    <vendorId>0x1d6b</vendorId>
    <vendor>Linux 3.13.6 ehci_hcd</vendor>
    <productId>0x0002</productId>
    <product>EHCI Host Controller</product>
    <bus>1</bus>
    <device>1</device>
  </usb>
。。。
</host>
元素含义
host各子元素
id 主机ID。
整数（Long）类型。
name 主机名称。
字符串（String）类型。
usb各子元素
name usb名称。
字符串（String）类型。
vendorId 供应商ID。
字符串（String）类型。
value usb信息。
字符串（String）类型。
vendor 供应商。
字符串（String）类型。
productId 产品ID。
字符串（String）类型。
product 产品名称。
字符串（String）类型。
bus 总线信息。
字符串（String）类型。
device 设备类型。
字符串（String）类型。

返回结果

状态码
  • 200(OK)：成功。
消息头

HTTP/1.1 200 OK
Set-Cookie: JSESSIONID=23844B494C441FAAF93C53EB43F84324; Path=/cas
Content-Type: application/xml
Content-Length: 1414
Date: Thu, 18 Sep 2014 02:51:03 GMT
Server: H3C-CVM
