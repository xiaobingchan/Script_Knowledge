H3C CAS云管理平台

查询主机或集群虚拟交换机名称列表

根据主机id或集群id查询主机或集群虚拟交换机名称列表

接口 URI

/cas/casrs/vm/queryVSwtichNameList/{destType}/{id}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
destType 根据主机ID或集群ID查询标志位，2表示主机，3表示集群
整数（int）类型。必须。
id 集群ID或主机ID。
长整数（long）类型。必须。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Content-Type: application/xml
Content-Length: 193
Date: Wed, 17 Sep 2014 09:41:54 GMT
Server: H3C-CVM
消息体
<?xml version="1.0" encoding="UTF-8"?>
<rsResult>
  <state>0</state>
  <errorCode>0</errorCode>
  <data>[vswitch0]</data>
</rsResult>
元素含义
rsResult各子元素
state 状态。
整型（Int）类型。
errorCode 错误代码。
整型（Int）类型。
data 消息
字符串（String）类型。
应用举例

查询主机id为4的虚拟交换机名称列表。

请求
GET http://localhost:8080/cas/casrs/vm/queryVSwtichNameList/2/4
accept: application/xml
...
回应
HTTP/1.1 200 OK
Content-Type: application/xml
...
<?xml version="1.0" encoding="UTF-8"?>
<rsResult>
  <state>0</state>
  <errorCode>0</errorCode>
  <data>[vswitch0]</data>
</rsResult>
