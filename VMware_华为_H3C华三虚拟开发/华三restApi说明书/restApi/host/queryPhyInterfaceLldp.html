查询主机下指定物理网卡LLDP信息

查询主机下指定物理网卡LLDP信息。

接口 URI

/cas/casrs/host/{id}/interface/{name}/lldp

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 主机id 。
长整数（long）类型。必须。
name 物理网卡名称。
字符串（String）类型。必须。

返回结果

状态码
  • 200(OK)：成功。
  • 409(Conflict)：失败

元素含义
lldp各子元素

字符串（String）类型。
switchInfo 对端设备信息。
字符串（String）类型。
