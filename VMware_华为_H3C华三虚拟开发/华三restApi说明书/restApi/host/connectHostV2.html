H3C CAS云管理平台

连接主机

连接指定id的主机。

接口 URI

/cas/casrs/host/connect/v2

访问方法

HTTP 方法
PUT
消息体
<?xml version="1.0" encoding="UTF-8"?>
<host>
  <id>2</id>
  <name>server3</name>
  <ip>192.168.10.3</ip>
</host>
元素含义
host各子元素
id 物理机ID。
整数（Lomg）类型。
name 物理机名称。
字符串（String）类型。
ip 服务器IP地址。
字符串（String）类型。

返回结果

状态码
  • 200(OK)：成功。
消息头

<?xml version="1.0" encoding="UTF-8"?>
<taskMsg>
  <msgId>1410943942352</msgId>
  <completed>false</completed>
  <progress>0</progress>
  <eventType>0</eventType>
</taskMsg>
