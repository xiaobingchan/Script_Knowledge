H3C CAS云管理平台

进入维护状态

使指定id的主机进入维护状态。

接口 URI

/cas/casrs/host/into

访问方法

HTTP 方法
PUT
消息体
<?xml version="1.0" encoding="UTF-8"?>
<host>
  <id>7</id>
  <name>cvk011</name>
  <ip>192.168.200.11</ip>
</host>
元素含义
host各子元素
id 物理机ID。
整数（Lomg）类型。
name 物理机名称。
字符串（String）类型。
ip 服务器IP地址。
字符串（String）类型。

返回结果

状态码
  • 200(OK)：成功。
消息头

Content-Type: application/xml
Content-Length: 176
Date: Wed, 17 Sep 2014 07:59:11 GMT
Server: H3C-CVM

应用举例

1.使id为2的主机进入维护状态。
