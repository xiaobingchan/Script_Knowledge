分发虚拟机模板

分发虚拟机模板。

接口 URI

/cas/casrs/vmTemplate/distribution

访问方法

HTTP 方法
PUT
消息体 
<template>
    <id>1195</id>
    <targetHostId>1</targetHostId>
    <storagePoolName>pool</storagePoolName>
    <cover>true</cover>
</template>
元素含义
template参数
id 虚拟机模板id 。
长整数（long）类型。必须。
targetHostId 目的主机id 。
长整数（long）类型。必须。
storagePoolName 目的存储池名称。
字符串（String）类型，必须。
cover 是否覆盖。
布尔型（Boolean）类型（true：覆盖；false：不覆盖）

返回结果

状态码
  • 200(OK)： 成功。
  • 409(Conflict)：失败。

元素含义
返回分发虚拟机模板任务id，可根据任务id查询任务进度。
