创建虚拟机模板

创建虚拟机模板。

接口 URI

/cas/casrs/vmTemplate/addVmTemplate?id={id}&domainName={domainName}&desc={desc}&opType={opType}&templetStoragePath={path}

访问方法

HTTP 方法
POST
元素含义
查询参数（Query Parameter）
id 虚拟机id 。
长整数（long）类型。必须。
domainName 虚拟机模板名称 。
字符串（String）类型。必须。
desc 虚拟机模板描述 。
字符串（String）类型。
opType 创建虚拟机形式 。
整形（Integer）类型。必须。 枚举值 1克隆 2 转移
templetStoragePath 模板存储目录 。
字符串（String）类型。

返回结果

状态码
  • 200(OK)： 成功。
  • 409(Conflict)：失败。
