查询虚拟机模板列表

查询虚拟机模板列表。

接口 URI

/cas/casrs/vmTemplate/vmTemplateList?offset={offset}&limit={limit}

访问方法

HTTP 方法
GET
参数

查询参数（Query Parameter）
offset 查询起始位置 。
整形（Integer）。必须。
limit 查询记录数 。
整形（Integer）。必须。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Server: Apache-Coyote/1.1
Set-Cookie: JSESSIONID=C55839CBDD714849D8644CCA93562E78; Path=/cas
Content-Type: application/xml
Content-Length: 933
Date: Thu, 28 Nov 2013 08:56:24 GMT
消息体
<?xml version="1.0" encoding="UTF-8"?>
<list>
  <domain>
    <id>168</id>
    <name>cc-oa</name>
    <memory>128</memory>
    <cpu>1</cpu>
    <system>0</system>
    <createDate>2013-11-23T16:07:11+08:00</createDate>
    <storageCapacity>6802</storageCapacity>
  </domain>...
</list>
元素含义
event各子元素
id 虚拟机模板 ID。
长整数（long）类型。
name 虚拟机模板名称。
字符串（String）类型。
description 虚拟机模板描述。
字符串（String）类型。
memory 模板内存。
长整形（Long）类型。
cpu CPU个数。
整数（int）类型。
system 系统。
整数（int）类型。 枚举值：0 Windows 1 Linux
createDate 创建时间。
日期（Date）类型。
storageCapacity 存储。
长整形（Long）类型。
