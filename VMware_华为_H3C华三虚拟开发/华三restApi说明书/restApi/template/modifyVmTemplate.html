修改虚拟机模板

修改虚拟机模板。

接口 URI

/cas/casrs/vmTemplate/modifyVmTemplate

访问方法

HTTP 方法
PUT
消息体 
<domain>
    <id>1195</id>
    <name>lxwtt-template</name>
    <description>rest modify</description>
    <memory>1024</memory>
    <memoryInit>1024</memoryInit>
    <memoryUnit>MB</memoryUnit>
    <cpuSockets>1</cpuSockets>
    <cpuCores>1</cpuCores>
    <cpu>2</cpu>
    <cpuMax>8</cpuMax>
    <storage>
      <id>130</id>
      <storeFile>lxwtt-template</storeFile>
      <capacity>100</capacity>
    </storage>
</domain>
元素含义
domain参数
id 虚拟机模板id 。
长整数（long）类型。必须。
name 虚拟机模板名称 。
字符串（String）类型。必须。
description 描述,最大输入长度为120个字符。
字符串（String）类型，必须。
memory 虚拟机模板内存大小，单位为M。
整型（Integer）类型，必须。
memoryInit 虚拟机模板内存单位。
字符串（String）类型。必须。
cpuSockets CPU个数，最小值为1。
整型（Integer）类型，必须。
cpuCores CPU核数，最小值为1。
整型（Integer）类型，必须。
cpuMax 虚拟机模板CPU最大核数。
整型（Integer）类型，必须。
cpu

虚拟机模板CPU当前总核数（CPU个数 * CPU核数）。

整型（Integer）类型，必须。
storage 存储配置。
自定义类型，必须。
storage各子元素
id 存储id。
长整数（long）类型。必须。
storeFile 存储文件。
字符串（String）类型，必须。
capacity 分配存储容量。 单位（M）。
长整数（long）类型。

状态码
  • 200(OK)： 成功。
  • 409(Conflict)：失败。
