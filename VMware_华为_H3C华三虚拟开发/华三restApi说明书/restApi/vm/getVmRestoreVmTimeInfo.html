H3C CAS云管理平台

虚拟机备份时间信息列表

查询虚拟机备份时间信息列表。

接口 URI

查询cp方式备份的虚拟机

/cas/casrs/vm/vmRestoreInfo?vmName=WFG_DRX01-_001&type=cp&directory=/vms&hostId=3

查询scp方式备份的虚拟机

/cas/casrs/vm/vmRestoreInfo?hostId=3&type=scp&directory=/vms&targetAddr=192.168.0.28&userName=root&password=1q2w3e&vmName=WFG_DRX01-_001

查询ftp方式备份的虚拟机

/cas/casrs/vm/vmRestoreInfo?hostId=3&type=ftp&directory=/&targetAddr=192.168.1.80&userName=root&password=1q2w3e&vmName=WFG_DRX01-_001

访问方法

HTTP 方法
GET
参数

查询参数（Query Parameter）
type 虚拟备份类型。 枚举值cp,ftp,scp
字符串（String）。必须。
directory 虚拟机备份路径 。
字符串（String）。必须。
hostId 主机id 。
长整形（Long）。必须。
targetAddr IP地址 。
字符串（String）。type值为ftp、scp时必须。
userName 用户名。
字符串（String）。type值为ftp、scp时必须。
password 密码 。
字符串（String）。type值为ftp、scp时必须。
vmName 虚拟机名称 。
字符串（String）。必须。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Set-Cookie: JSESSIONID=C09B3DE2AEBBA834BC08C476081809D3; Path=/cas
Content-Type: application/xml
Content-Length: 132
Date: Thu, 18 Dec 2014 03:24:38 GMT
消息体

<?xml version="1.0" encoding="UTF-8"?><list>
  <time>2014-12-17 17:44:12</time>
  <time>2014-12-17 17:44:24</time>
</list>
元素含义
time 虚拟机备份时间。
字符串（String）类型。
