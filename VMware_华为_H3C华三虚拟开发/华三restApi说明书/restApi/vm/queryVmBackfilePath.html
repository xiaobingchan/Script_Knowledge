H3C CAS云管理平台

查询虚拟机镜像文件路径

查询虚拟机镜像文件路径。

接口 URI

/cas/casrs/vm/{id}/backfilepath

访问方法

HTTP 方法
GET
参数

查询参数（Query Parameter）
id 虚拟机id 。
长整形（Long）。必须。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
Content-Type: application/xml
Content-Length: 191
Date: Thu, 18 Dec 2014 06:14:42 GMT
Server: H3C-CVM
消息体
<?xml version="1.0" encoding="UTF-8"?>
<backingfiles>
  <backingfile>
    <incfilepath>/vms/images/a.img</incfilepath>
    <backfilepath>/vms/images/a_base1.img</backfilepath>
  </backingfile>

  <backingfile>
    <incfilepath>/vms/images/a_base1.img</incfilepath>
    <backfilepath>/vms/images/a_base0.img</backfilepath>
  </backingfile>
</backingfiles>
元素含义
backingfile各子元素
incfilepath 虚拟机增量文件路径。
字符串（String）类型。
backfilepath 虚拟机基础镜像路径。
字符串（String）类型。

应用举例

查询虚拟机镜像文件路径。

请求
GET http://localhost:8080/cas/casrs/vm/38/backfilepath
accept: application/xml
...
回应
HTTP/1.1 200 OK
Content-Type: application/xml
Content-Length: 191

<backingfiles>
  <backingfile>

    <incfilepath>/vms/images/a_base1.img</incfilepath>
