H3C CAS云管理平台

查询虚拟机的性能监控数据

根据虚拟机ID或虚拟机UUID查询CPU利用率、内存利用率、磁盘空间利用率、I/O读写吞吐量、磁盘读/写IOPS、磁盘I/O延迟、网卡入出流量、报文数。

接口 URI

/cas/casrs/vm/id/{vmId}/monitor

或者

/cas/casrs/vm/{uuid}/monitor

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
vmId 虚拟机id 。
长整数（long）类型。
uuid 虚拟机的UUID。
字符串（String）类型。

返回结果

状态码
  • 200(OK)：成功。
消息头

HTTP/1.1 200 OK
消息体
<?xml version="1.0" encoding="UTF-8"?><domain>
  <id>1191</id>
  <cpuRate>0.25</cpuRate>
  <memRate>15.43</memRate>
  <disk>
    <device>hda</device>
    <ioStat>2.4</ioStat>
    <readStat>0.0</readStat>
    <writeStat>2.4</writeStat>
    <readReqest>0.0</readReqest>
    <writeReqest>0.0</writeReqest>
    <readLatency>0.0</readLatency>
    <writeLatency>1.121</writeLatency>
  </disk>
  <net>
    <mac>0c:da:41:1d:ca:e9</mac>
    <readFlow>0.011817423502604167</readFlow>
    <readPackets>381.0</readPackets>
    <writeFlow>0.0</writeFlow>
    <writePackets>0.0</writePackets>
  </net>
  <partition>
    <device>C:</device>
    <usage>11.7</usage>
  </partition>
</domain>

元素含义
domain各子元素
cpuRate CPU利用率。
小数（double）类型。
memRate 内存利用率。
小数（double）类型。
disk各子元素
device 磁盘镜像物理名称。
字符串（String）类型。
ioStat I/O吞吐量。 单位：KBps
小数（double）类型。
readStat I/O读吞吐量。 单位：KBps
小数（double）类型。
writeStat I/O写吞吐量。 单位： KBps
小数（double）类型。
readReqest 磁盘读IOPS。
小数（double）类型。
writeReqest 磁盘写IOPS。
小数（double）类型。
readLatency 磁盘I/O读延迟。 单位：ms
小数（double）类型。
writeLatency 磁盘I/O写延迟。 单位： ms
小数（double）类型。
net各子元素
mac 网卡MAC地址。
字符串（String）类型。
readFlow 网卡读流量。 单位： Mbps
小数（double）类型。
readPackets 网卡读报文数。
小数（double）类型。
writeFlow 网卡写流量。 单位： Mbps
小数（double）类型。
writePackets 网卡写报文数。
小数（double）类型。
partition各子元素
device 逻辑磁盘名称。
字符串（String）类型。
usage 磁盘利用率数据。
小数（double）类型。
