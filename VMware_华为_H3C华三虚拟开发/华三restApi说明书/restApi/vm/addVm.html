创建虚拟机

根据指定的参数创建虚拟机。

接口 URI

/cas/casrs/vm/add

访问方法

HTTP 方法
POST

元素含义
domain各子元素
name 虚拟机名称，名称的最大输入长度为64个字符，且只允许输入字母、数字、减号、下划线及点。
字符串（String）类型，必须。
title 虚拟机显示名称，最大输入长度为64个字符。
字符串（String）类型，必须。
description 描述,最大输入长度为120个字符。
字符串（String）类型，可选项。
memory 内存大小，单位为MB，最小值为512最大值由主机实际情况决定。
整型（Integer）类型，必须。
memoryInit 内存大小，单位为由memoryUnit决定,换算成MB后，必须与memory指定的值相同。
整型（Integer）类型，必须。
memoryUnit 内存显示单位，字符串。
枚举类型：GB,MB。
memoryPriority 内存资源优先级。枚举值：0=低，50=中，100=高
整型（Integer）类型，取值:0,50,100。必填。
memoryBacking 内存预留。内存预留百分比。
整型（Integer）类型，取值范围:0-100。必填。
cpuSockets CPU个数，最小值为1，最大值由主机实际情况决定。
整型（Integer）类型，必须。
cpuCores CPU核数，最小值为1，最大值由主机实际情况决定。
整型（Integer）类型，必须。
cpuShares CPU调度优先级，枚举值：1024：高，512：中，256：低。
整型（Integer）类型，必须。
maxCpuSocket 主机允许的最大CPU核数。
整型（Integer）类型，必须。
blkiotune I/O优先级，枚举值：500：高，300：中，200：低。
整型（Integer）类型，必须。
system 操作系统类型，枚举值：0：Windows,1：Linux。
整型（Integer）类型，必须。
osVersion

操作系统版本，枚举值：

当操作系统为Windows:

Microsoft Windows Server 2012, Microsoft Windows Server 2012 R2, Microsoft Windows Server 2008 R2(64-bit), Microsoft Windows Server 2008(64-bit), Microsoft Windows Server 2008(32-bit), Microsoft Windows Server 2003(64-bit), Microsoft Windows Server 2003(32-bit), Microsoft Windows 8(64-bit), Microsoft Windows 8(32-bit), Microsoft Windows 7(64-bit), Microsoft Windows 7(32-bit), Microsoft Windows Vista(64-bit), Microsoft Windows Vista(32-bit), Microsoft Windows XP Professional(64-bit), Microsoft Windows XP Professional(32-bit), Microsoft Windows 2000, Microsoft Windows NT, Microsoft Windows 98, Microsoft Windows 95, Microsoft Windows 3.1, Microsoft MS-DOS

当操作系统为Linux:

Red Hat Enterprise Linux7(64-bit), Red Hat Enterprise Linux7(32-bit), Red Hat Enterprise Linux6(64-bit), Red Hat Enterprise Linux6(32-bit), Red Hat Enterprise Linux5(64-bit), Red Hat Enterprise Linux5(32-bit), Red Hat Enterprise Linux4(64-bit), Red Hat Enterprise Linux4(32-bit), Red Hat Enterprise Linux3(64-bit), Red Hat Enterprise Linux3(32-bit), Red Hat Enterprise Linux2.1, Novell SUSE Linux Enterprise 11(64-bit), Novell SUSE Linux Enterprise 11(32-bit), Novell SUSE Linux Enterprise 10(64-bit), Novell SUSE Linux Enterprise 10(32-bit), Novell SUSE Linux Enterprise 8/9(64-bit), Novell SUSE Linux Enterprise 8/9(32-bit), CentOS 4/5/6(64bit), CentOS 4/5/6(32bit), Debian GUN/Linux 6(64-bit), Debian GUN/Linux 6(32-bit), Debian GUN/Linux 5(64-bit), Debian GUN/Linux 5(32-bit), Debian GUN/Linux 4(64-bit), Debian GUN/Linux 4(32-bit), Asianux 4(64-bit), Asianux 4(32-bit), Asianux 3(64-bit), Asianux 3(32-bit), Oracle Linux 4/5/6(64-bit), Oracle Linux 4/5/6(32-bit), Ubuntu Linux(64-bit), Ubuntu Linux(32-bit), Other 2.6.x Linux(64-bit), Other 2.6.x Linux(32-bit), Other 2.4.x Linux(64-bit), Other 2.4.x Linux(32-bit), Other Linux(64-bit), Other Linux(32-bit)

字符串（String）类型，必须。
autoMigrate 是否允许虚拟机自动迁移，实现资源动态调整，枚举值：1=允许，0=不允许。
整型（Integer）类型，必须。
priority 启动优先级，当集群启用HA后有效，枚举值：0：低级，1：中级，2：高级。
整型（Integer）类型，必须。
hostId 主机id。
长整型（Long）类型，必须。
hostPoolId 主机池id。
长整型（Long）类型，必须。
formatEnable 是否可用。
整型（Integer）类型，设置为1，必须。
bindcpu 绑定物理CPU配置。
自定义类型，可选项。
network 虚拟机网络配置。
自定义类型，必须。
storage 虚拟机存储配置。
自定义类型，必须。
usb 虚拟机USB设备配置。
自定义类型，可选。
pci 虚拟机PCI设备配置。
自定义类型，可选。
gpu 虚拟机GPU直通显卡配置,最多只能有一个。
自定义类型，可选。
bindcpu各子元素
vcpu 指定虚拟CPU，0：表示第一个虚拟CPU，1：表示第二个虚拟CPU，依次类推。
整型（Integer）类型，必须。
pcpu 指定物理CPU，0：表示第一个物理CPU，1：表示第二个物理CPU，依次类推。一个虚拟CPU可绑定多个物理CPU。
整型（Integer）类型，必须。
network各子元素(非SR-IOV网卡)
deviceModel 网卡设备型号，枚举值：rtl8139：普通网卡;virtio：Virtio网卡;e1000：Intel e1000网卡。
字符串（String）类型，必须。
vsId 虚拟交换机id。
长整型（Long）类型，必须。
vsName 虚拟交换机名称。
字符串（String）类型，必须。
profileId 网络策略模板id。
长整型（Long）类型，必须。
isKernelAccelerated 是否设置内核加速，枚举值：1：代表设置内核加速，0：代表不设置。
整型（Integer）类型，默认为0。
ipAddr 网卡绑定IP地址。
字符串（String）类型，符合IPv4格式，可选项。
network各子元素(SR-IOV网卡)
deviceModel 网卡设备型号，SR_IOV:SR-IOV直通网卡。
字符串（String）类型，必须。
driverType 驱动类型，枚举值：VFIO：VFIO设备分配处理，KVM：KVM内核处理
字符串（String）类型，必须。
address 虚拟网卡地址。
字符串（String）类型，必须。。
ethName SR-IOV直通网卡名称。
字符串（String）类型，必须。
mac 虚拟网卡mac地址
字符串（String）类型，可选项。
vlan vlan
整型（Integer）类型，可选项。
storage各子元素
device 存储类型，枚举值：disk：硬盘， cdrom：光驱， floppy：软驱。
字符串（String）类型，必须。
targetBus 存储设备类型，枚举值：ide：IDE硬盘， scsi：SCSI硬盘， usb：USB硬盘;，virtio：Virtio 硬盘。
字符串（String）类型，必须。
storeFile 存储文件路径,当增加软驱时，若值为autoVirtio，则根据操作系统类型自动加载一个软驱设备。
字符串（String）类型，必须。
cacheType 磁盘缓存方，枚举值：directsync，writethrough，writeback，none。
字符串（String）类型，当存储卷为文件类型时不能为空。
assignType 分配类型:0指定 1动态分配。
整型（Integer）类型，必须。
type 类型。枚举值：file 文件， block 块设备。
字符串（String）类型，必须。
driveType 存储卷类型，必须和使用的存储卷的类型保持一致，枚举值：qcow2，raw
字符串（String）类型，必须。
clusterSize 存储卷簇大小，枚举值：65536（64K）、131072（128K）、262144（256K）、524288（512K）、1048576（1M）、2097152（2M）
字符串（String）类型。
usb各子元素
devName USB设备名称，必须与主机上查询的USB名称一致。
字符串（String）类型，必须。
vendorId USB设备的vendorId的值。
字符串（String）类型，必须。
vendor USB设备的vendor的值。
字符串（String）类型，必须。
productId USB设备的productId的值。
字符串（String）类型，必须。
product USB设备的product的值。
字符串（String）类型，必须。
bus USB设备的bus的值。
字符串（String）类型，必须。
device USB设备的device的值。
字符串（String）类型，必须。
controller USB控制器版本，取值为1：USB1.0,2：USB2.0,3：USB3.0。
整型（Integer）类型，必须。
pci各子元素
devName PCI设备名称，必须与主机上查询的PCI名称一致。
字符串（String）类型，必须。
vendorId PCI设备的vendorId的值。
字符串（String）类型，必须。
vendor PCI设备的vendor的值。
字符串（String）类型，必须。
productId PCI设备的productId的值。
字符串（String）类型，必须。
product PCI设备的product的值。
字符串（String）类型，必须。
function PCI设备的function的值。
字符串（String）类型，必须。
slot PCI设备的slot的值。
字符串（String）类型，必须。
bus PCI设备的bus的值。
字符串（String）类型，必须。
ethName PCI物理接口的名称。
字符串（String）类型，必须。
driver 驱动类型，取值为：KVM,VFIO。
字符串（String）类型，必须。
gpu各子元素
resPoolId GPU资源池ID。
长整型（Long）类型，必须。
businessTemId 业务模板ID。
长整型（Long）类型，必须。
