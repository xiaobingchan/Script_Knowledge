/cas/casrs/vm/batchDeploy

访问方法

HTTP 方法
POST

        <vSwitchName>vswitch0</vSwitchName>

元素含义
domainConfig各子元素
id 模板ID
长整数（long）类型。
domainName 部署虚拟机名字。
字符串（String）类型。
title 部署虚拟机显示名字。
字符串（String）类型。
desc
 虚拟机描述信息
字符串（String）类型。
targetHostId 目标主机ID
长整型（long）类型。
deployModel 部署虚拟机方式（0表述正常部署，1表示快速部署）。
整数（int）类型。
storage 虚拟机存储列表。
集合（List）类型。
network 虚拟机网卡信息列表。
集合（List）类型。
osInfo 设置系统IP信息。
storage各子元素
src 原存储镜像文件名。
字符串（String）类型。
dest 目的存储镜像文件名。
字符串（String）类型。
pool 目的存储池。
字符串（String）类型。
network各子元素
mac
 网卡MAC地址。
字符串（String）类型。
vSwitchName
 网络交换机名称。
字符串（String）类型。
ip
 网卡绑定的IP地址。
字符串（String）类型。
profileName
 网络策略模板名称。
字符串（String）类型。
osInfo各子元素
sysIp IP地址。
字符串（String）类型。
sysMask 子网掩码地址。
字符串（String）类型。
sysGateway 默认网关地址。
字符串（String）类型。
sysdns 首选DNS服务器。
字符串（String）类型。
secondaryDns 备用DNS服务器。
字符串（String）类型。
isBindIp 是否绑定IP。
布尔型。

返回结果

状态码
  • HTTP/1.1 200 OK：成功。
