修改虚拟机网络配置

修改虚拟机网络配置。

接口 URI

/cas/casrs/vm/modify

访问方法

HTTP 方法
PUT
消息体
<domain>
    <id>6</id>
    <name>test</name>
    <network>
        <mac>0c:da:41:1d:61:32</mac>
        <newMac>0c:da:41:1d:61:32</newMac>
        <netType>0</netType>
        <ipAddr>192.168.0.214</ipAddr>
        <maskAddr>255.255.255.0</maskAddr>
        <gateway>192.168.0.1</gateway>
        <dns>192.168.0.1</dns>
        <secondDns>192.168.0.2</secondDns>
        <vsId>1</vsId>
        <vsName>vswitch0</vsName>
        <profileId>1</profileId>
        <deviceModel>e1000</deviceModel>
        <isKernelAccelerated>0</isKernelAccelerated>
	<mode>veb</mode>
        <vsiMngId>1080::8:800:200C:417A</vsiMngId>
        <vsiTypeId>222</vsiTypeId>
        <vsiTypeVer>121</vsiTypeVer>
        <vsiIdFormat>UUID</vsiIdFormat>
        <vsiId></vsiId>
	<hotPluggable>true</hotPluggable>
    </network>
</domain>
元素含义
domain各子元素
name 虚拟机名称。
字符串（String）类型，必须。
id 虚拟机id。
长整型（Long）类型，必须。
network 网络配置。
自定义类型，必须。
network各子元素(非SR-IOV网卡)
mac 网卡mac地址。
字符串（String）类型，必须。
netType 网卡类型：0桥接 1NAT 2不使用网络 3物理直通网卡 。
字符串（String）类型，必须。
newMac 要更改的目的mac地址。
字符串（String）类型，可选项。
ipAddr 网卡绑定IP地址。
字符串（String）类型，符合IPv4格式，可选项。
maskAddr 子网掩码。
字符串（String）类型，符合子网掩码格式，可选项。
gateway 默认网关。
字符串（String）类型，符合IPv4格式，可选项。
dns 首选DNS。
字符串（String）类型，符合IPv4格式，可选项。
secondDns 备用DNS。
字符串（String）类型，符合IPv4格式，可选项。
vsId 虚拟交换机id。
长整型（Long）类型，必须。
vsName 虚拟交换机名称。
字符串（String）类型，必须。
profileId 虚拟交换机id。
长整型（Long）类型，必须。
deviceModel 网卡设备型号，枚举值：rtl8139：普通网卡;virtio：Virtio网卡;e1000：Intel e1000网卡。
字符串（String）类型，必须。
isKernelAccelerated 是否设置内核加速，当网卡设备型号为virtio时生效，枚举值：1：代表设置内核加速，0：代表不设置。
整型（Integer）类型，默认为0。
mode 网卡转发模式，枚举值：veb，vepa。
字符串（String）类型，必须。
vsiMngId VSI管理者ID，符合IPV6格式。
字符串（String）类型，可选项。
vsiTypeId VSI类型ID，取值范围为：0-16777215。
整型（Integer）类型，必须。
vsiTypeVer VSI类型版本，取值范围为0-255。
整型（Integer）类型，必须。
vsiIdFormat VSI 格式，目前只支持UUID。
字符串（String）类型，必须。
vsiId VSI 实例ID，必须符合UUID格式：XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX（8-4-4-12，每个X代表一个十六进制数[0-9|a-f]）。
字符串（String）类型，必须。
hotPluggable 热拔。（true：允许，false：不允许）
布尔（Boolean）类型，可选。
network各子元素(SR-IOV网卡)
mac 网卡mac地址。
字符串（String）类型，必须。
netType 网卡类型：4:SR-IOV网卡。
字符串（String）类型，必须。
newMac 要更改的目的mac地址。
字符串（String）类型，可选项。
driverType 驱动类型，枚举值：VFIO：VFIO设备分配处理，KVM：KVM内核处理
字符串（String）类型，可选项。
vlan VLAN
整型（Integer）类型，可选项。
