创建虚拟机快照

为指定ID的虚拟机创建快照。

接口 URI

/cas/casrs/vm/snapshot

访问方法

HTTP 方法
POST
消息体
<snapshot>
  <vmId>31</vmId>
  <name>test4</name>
  <desc>test4 snapshot</desc>
  <snapMem>false</snapMem>
  <noStopVirtSnapMem>false</noStopVirtSnapMem>
  <snapshotTimeOut>5</snapshotTimeOut>

</snapshot>
元素含义
snapshot各子元素
vmId 虚拟机 ID。
长整数（long）类型，必须。
name 快照名称。
字符串（String）类型，必须。
desc 快照描述。
字符串（String）类型，可选。
snapMem 快照内存：true 快照内存；false 快照不带内存。
布尔（boolean）类型，可选。
noStopVirtSnapMem 不停机快照内存：true 不停机；false 停机。
布尔（boolean）类型，可选。
snapshotTimeOut 虚拟机快照超时时间。
整数（int）类型，虚拟机进行不停机快照内存时必须。
