为指定虚拟机添加虚拟硬盘

为指定虚拟机添加虚拟硬盘。

接口 URI

/cas/casrs/vm/addDevice

访问方法

HTTP 方法
PUT
消息体
<domain>
	<id>10</id>
	<name>vm01</name>
	<storage>
		<device>disk</device>
		<fileType>file</fileType>
		<path>/vms/image/vol_test</path>
		<format>qcow2</format>
		<targetBus>ide</targetBus>
		<deviceName>/dev/sdb</deviceName>
		<cacheType>directsync</cacheType>
		<size>1000</size>
		<hotPluggable>true</hotPluggable>
	</storage>
</domain>
元素含义
domain各子元素
id 虚拟机ID。
长整型（Long）类型，必须。
name 虚拟机名称。
字符串（String）类型，必须。
storage 虚拟机存储配置。
自定义类型，必须。
storage各子元素
targetBus 存储设备类型，枚举值：ide：IDE硬盘， scsi：SCSI硬盘， usb：USB硬盘;，virtio：Virtio 硬盘，cdrom：IDE光驱，floppy：软驱。
字符串（String）类型，必须。
path 存储文件路径。
字符串（String）类型，必须。
cacheType 磁盘缓存方，枚举值：directsync，writethrough，writeback，none。
字符串（String）类型，当存储卷为文件类型时不能为空。
deviceName 设备名称。
字符串（String）类型，可选。
device 设备驱动类型，取值为disk, cdrom, floppy
字符串（String）类型，必须。
format 存储格式，取值为raw，qcow2。
字符串（String）类型。
size 分配存储容量
长整型（Long）类型。
fileType 存储卷类型，取值为file，block。
字符串（String）类型。
readBytesSec 限制I/O速率，读，单位KB/s。
长整型（Long）类型。
writeBytesSec 限制I/O速率，写，单位KB/s。
长整型（Long）类型。
readIopsSec 限制IOPS，读。
长整型（Long）类型。
writeIopsSec 限制IOPS，写。
长整型（Long）类型。
hotPluggable 热拔。（true：允许，false：不允许）
布尔（Boolean）类型，可选。

返回结果

状态码
  • 204 (No Content)：成功。
  • 409 (Conflict)：失败。

应用举例

为ID为10的虚拟机增加一块IDE磁盘，磁盘缓存方式为directsync。

请求
PUT http://192.168.0.4:8080/cas/casrs/vm/addDevice
accept: application/xml


<domain>
	<id>10</id>
	<name>vm01</name>
	<storage>
		<device>disk</device>
		<fileType>file</fileType>
		<path>/vms/image/vol_test</path>
		<format>qcow2</format>
		<targetBus>ide</targetBus>
		<deviceName>/dev/sdb</deviceName>
		<cacheType>directsync</cacheType>
		<size>1000</size>
		<hotPluggable>true</hotPluggable>
	</storage>
</domain>
回应
    	
<?xml version="1.0" encoding="UTF-8"?>
<taskMsg>
  <state>0</state>
  <errorCode>0</errorCode>
  <data>为虚拟机“test”增加磁盘“hde”。 </data>
  <message>为虚拟机“test”增加磁盘“hde”成功。 </message>
</taskMsg>
