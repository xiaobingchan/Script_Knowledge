为指定虚拟机添加虚拟网卡

为指定虚拟机添加虚拟网卡。

接口 URI

/cas/casrs/vm/addDevice

访问方法

HTTP 方法
PUT
消息体
  非SR-IOV网卡 
<domain>
	<id>72</id>
	<name>rest</name>
	<title>rest</title>
	<network>
		<mac>0c:da:41:1d:e6:32</mac>
		<ipAddr>2.2.2.3</ipAddr>
		<vsId>1</vsId>
		<vsName>vswitch0</vsName>
		<deviceModel>virtio</deviceModel>
		<isKernelAccelerated>1</isKernelAccelerated>		
		<mode>veb</mode>
		<profileId>1</profileId>
		<profileName>Default</profileName>
		<hotPluggable>true</hotPluggable>
	</network>
</domain>
  SR-IOV网卡 
<domain>
	<id>72</id>
	<name>rest</name>
	<title>rest</title>
    <network>
       <mac>0c:da:41:1d:75:9b</mac>
        <driverType>VFIO</driverType>
        <ethName>eth3</ethName>
        <address>0000:07:09.0</address>
        <deviceModel>SR_IOV</deviceModel>
        <vlan>1</vlan>
    </network>
</domain>
元素含义
domain各子元素
id 虚拟机ID。
长整型（Long）类型，必须。
name 虚拟机名称。
字符串（String）类型，必须。
network 虚拟机网络配置。
自定义类型，必须。
network各子元素(非SR-IOV网卡)
vsId 虚拟交换机Id。
长整型（Long）类型，必须。
vsName 虚拟交换机名称。
字符串（String）类型，必须。
mac 网卡MAC地址。
字符串（String）类型，必须。
ipAddr 网卡IP地址。
字符串（String）类型，必须。
deviceModel 网卡类型，枚举值：rtl8139，virtio，e1000。
字符串（String）类型。
isKernelAccelerated 内核加速，用于virtio网卡。1：加速，0：不加速。默认为1。
整型（Intager）类型，可选。
mode 转发模式，取值为veb,vepa
字符串（String）类型，必须。
profileId 网络策略模板ID。
长整型（Long）类型，必须。
profileName 网络策略模板名称
字符串（String）类型，必须。
hotPluggable 热拔。（true：允许，false：不允许）
布尔（Boolean）类型，可选。
network各子元素(SR-IOV网卡)
deviceModel 网卡类型，SR-IOV
字符串（String）类型，必须。
driverType 驱动类型，枚举值：VFIO：VFIO设备分配处理，KVM：KVM内核处理
字符串（String）类型，必须。
address 虚拟网卡地址。
字符串（String）类型，必须。
ethName SR-IOV直通网卡名称。
字符串（String）类型，必须。
vlan VLAN
整数（Integer）类型，可选项。
mac 虚拟网卡MAC地址。可选
字符串（String）类型，必须。

返回结果

状态码
  • 204 (No Content)：成功。
  • 409 (Conflict)：失败。

应用举例

为ID为72的虚拟机增加一块virtio虚拟网卡，转发模式为veb。

请求
PUT http://192.168.0.4:8080/cas/casrs/vm/addDevice
accept: application/xml
<domain>
	<id>72</id>
	<name>rest</name>
	<title>rest</title>
	<network>
		<mac>0c:da:41:1d:e6:32</mac>
		<ipAddr>2.2.2.3</ipAddr>
		<vsId>1</vsId>
		<vsName>vswitch0</vsName>
		<deviceModel>virtio</deviceModel>
		<isKernelAccelerated>1</isKernelAccelerated>		
		<mode>veb</mode>
		<profileId>1</profileId>
		<profileName>Default</profileName>
		<hotPluggable>true</hotPluggable>
	</network>
</domain>;
回应
    	
<?xml version="1.0" encoding="UTF-8"?>
<taskMsg>
	<state>0</state>
	<errorCode>0</errorCode>
	<data>为虚拟机“rest”增加网络“0c:da:41:1d:e6:32”。</data>
	<message>为虚拟机“rest”增加网络“0c:da:41:1d:e6:32”成功。 </message>
</taskMsg>
