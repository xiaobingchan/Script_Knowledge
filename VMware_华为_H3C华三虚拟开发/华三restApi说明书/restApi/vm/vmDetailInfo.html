H3C CAS云管理平台

查询指定虚拟机的所有详细信息

根据虚拟机id查询指定虚拟机的所有详细信息。用于修改虚拟机的接口中。

接口 URI

/cas/casrs/vm/detail/{id}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 虚拟机id 。
长整数（long）类型。必须，无缺省值。

返回结果

状态码
  • 200(OK)：成功。

HTTP/1.1 200 OK

元素含义
domain各子元素
id 虚拟机 ID。
长整数（long）类型。
name 虚拟机名称。
字符串（String）类型。
title 虚拟机显示名称。
字符串（String）类型。
videoType 显卡类型列表，取值为：cirrus、vga、qxl。
集合（List）类型。
basic 基本信息 。
对象类型。
cpu CPU配置信息。
对象类型。
memory 内存配置信息。
对象类型。
bootDev 引导配置信息。
对象类型。
input 输入设备列表。
集合（List）类型。
network 虚拟机网卡信息列表。
集合（List）类型。
storage 虚拟机存储列表。
集合（List）类型。
serial 串口设备列表。
集合（List）类型。
graphics 控制台信息列表。
集合（List）类型。
advance 高级配置信息。
对象类型。
video 显卡信息。
对象类型。
uptime 运行时长，单位：分钟。
整数（int）类型。
basic各子元素
pae 是否启用PAE：1-启用，0-不启用。
整数（int）类型。
acpi 是否启用ACPI：1-启用，0-不启用
整数（int）类型。
apic 是否启用APIC：1-启用，0-不启用。
整数（int）类型。
clock 时钟设置：本地：localtime，世界时钟 ：utc。
字符串（String）类型。
description 虚拟机描述信息。
字符串（String）类型。
blkiotune O优先级 高=500，中=300，低=200。
整数（int）类型。
startPriority 虚拟机启动优先级 0:低级 1:中级 2:高级。
整数（int）类型。
autoMigrate 虚拟机是否允许自动迁移。1:允许，0：否。
整数（int）类型。
haManage 虚拟机HA管理标志 1:开启高可靠性，0：关闭高可靠性。
字符串（String）类型。
architecture 系统架构 如：X86_64。
字符串（String）类型。
manager 管理程序。
字符串（String）类型。
emulator 模拟器。
字符串（String）类型。
osha 系统故障策略 0:不处理 1:故障重启 2:故障迁移。
整数（int）类型。
timeSync 同步虚拟机时间与主机时间 1:同步，0：不同步
整数（int）类型。
cpu各子元素
cpuSockets CPU个数。
整数（int）类型。
cpuCores CPU核数。
整数（int）类型。
cpuShares 虚拟机CPU优调度先级。 高=1024，中=512，低=256。
整数（int）类型。
maxCpuNum CPU最大核数（主机CPU核数）。
整数（int）类型。
cpuMode CPU工作模式，取值为：custom、host-model、host-passthrough。
字符串（String）类型。
cpuArch CPU体系结构，取值为：x86_64、i686。
字符串（String）类型。
cpuQuota CPU周期限制值。
小数（double）类型。
cpuQuotaUnit CPU周期限制 单位。
字符串（String）类型。
cpuMinRate CPU周期最小限制。
长整数（long）类型。
cpuMaxRate CPU周期最大限制。
长整数（long）类型。
cpuGurantee CPU预留。
长整数（long）类型。
memory各子元素
size 大小。
长整数（long）类型。
memoryBacking 内存预留百分比 0-10。
整数（int）类型。
memoryPriority 内存资源优先级。 高=100，中=50，低=0。
整数（int）类型。
memoryUnit 内存单位。
字符串（String）类型。
maxMemory 最大内存（主机内存）。
长整数（long）类型。
memoryLimit 内存限制。
小数（double）类型。
memoryLimitUnit 内存限制单位，枚举值：MB,GB。
字符串（String）类型。
memoryInit 显示内存。
小数（double）类型。
bootdev各子元素
autoStart 自动启动。 1-自动启动，0-不自动启动。
整数（int）类型。。
bootdevs 虚拟机引导设备及顺序，形式：dev1,dev2,net1...。
字符串（String）类型。
input各子元素
type 类型 ：tablet，mouse等。
字符串（String）类型。
model 光标移动方式。
字符串（String）类型。
bus 总线类型，取值为usb、ps2。
字符串（String）类型。
graphics各子元素
type 控制台类型，取值为：vnc、spice。
字符串（String）类型。
port 控制台端口。
整数（int）类型。
address 地址。
字符串（String）类型。
password 控制台密码。
字符串（String）类型。
kayboardMap 键盘映射。
字符串（String）类型。
enableVncProxy 是否启用VNC代理，1-启用，0-不启用。
整数（int）类型。
tlsPort SSL安全端口号。
字符串（String）类型。
spiceTlsChannel SSL加密通道名称。 取值为：main、display、inputs、cursor、playback、record、smartcard、usbredir。
字符串（String）类型。
 
network各子元素
mac 网卡MAC地址。
字符串（String）类型。
ipAddr 网卡IP地址。
字符串（String）类型。
vsName 网卡使用的虚拟交换机名称。
字符串（String）类型。
vlan 网卡所属的VLAN。
整数（int）类型。
vsId 虚拟交换机Id。
长整数（long）类型。
vsName 虚拟交换机名称。
字符串（String）类型。
deviceModel 设备型号，取值为:：rtl8139、virtio、e1000。
字符串（String）类型。
isKernelAccelerated 内核加速： 网卡驱动类型为 virtoIo时有效 1:加速，0：否。
整数（int）类型。
isLimitOutBound 是否限制出方向带宽。 true：限制 false：不限制。
布尔（boolean）类型。
isLimitInBound 是否限制入方向带宽。 true：限制 false：不限制。
布尔（boolean）类型。
mode 转发模式，取值为:：veb、vepa。
字符串（String）类型。
storage各子元素
path 存储镜像文件名。
字符串（String）类型。
size 分配存储容量。 单位（M）。
长整数（long）类型。
fileType 存储类型，取值为:： file、block。
字符串（String）类型。
format 存储格式，取值为： raw qcow2。
字符串（String）类型。
device 设备驱动类型，取值为:：disk, cdrom, floppy。
字符串（String）类型。
deviceName 设备名称，如： hda、vda。
字符串（String）类型。
targetBus 设备总线类型， 取值为：ide、scsi、 virtio、 usb。
字符串（String）类型。
allocation 已使用大小。
长整数（long）类型。
maxSize 最大可分配存储容量。
长整数（long）类型。
cacheType 磁盘缓存方式， 取值为：directsync、writethrough、writeback、none。
字符串（String）类型。
snapShot 包含快照: 1 包含；0 不包含。
整数（int）类型。
mode 预分配，0：延迟置零 1：置零 2：精简。
整型（Integer）类型。
clusterSize 存储卷簇大小，枚举值：65536（64K）、131072（128K）、262144（256K）、524288（512K）、1048576（1M）、2097152（2M）
字符串（String）类型。
numa各子元素
nodeSize 虚拟机所属主机的NUMA节点数
整数（int）类型。
nodeSet 虚拟机选用的numa节点
字符串（String）类型。
advance各子元素
redirUsb 虚拟机USB重定向数量，0表示不启用，取值范围为0-4。
整数（int）类型。
enableAntivirus 虚拟机是否启用防病毒策略，true表示启用，false表示不启用。
布尔（boolean）类型。
spiceSet 虚拟机SPICE客户端接入策略配置,true表示启用SSL，false表示不启用SSL。
布尔（boolean）类型。
video各子元素
type 显卡类型。
字符串（String）类型。
vram 显存大小，枚举值。0:16M, 1:64M, 2:128M, 3:256M, 4:512M。
整数（int）类型。
pciView 显卡的PCI地址，作为显卡的唯一标识。
字符串（String）类型。
