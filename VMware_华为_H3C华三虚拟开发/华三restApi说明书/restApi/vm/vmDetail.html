查询指定虚拟机详细信息

根据虚拟机id查询指定虚拟机的详细信息。

接口 URI

/cas/casrs/vm/{id}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 虚拟机id 。
长整数（long）类型。必须，无缺省值。

返回结果

状态码
  • 200(OK)：成功。

元素含义
domain各子元素
id 虚拟机 ID。
长整数（long）类型。
hostId 运行的主机ID。
长整数（long）类型。
clusterId 所属的集群 ID。
长整数（long）类型。
hostPoolId 所属的主机池 ID。
长整数（long）类型。
name 虚拟机名称。
字符串（String）类型。
title 虚拟机显示名称。
字符串（String）类型。
description 虚拟机描述信息。
字符串（String）类型。
memory 虚拟机内存大小，单位（M） 。
整数（int）类型。
cpu 虚拟机CPU个数。
整数（int）类型。
status 虚拟机状态。 2：运行 3：关闭 4：暂停。
整数（int）类型。
uuid 虚拟机UUID。
字符串（String）类型。
osBit 虚拟机安装的操作系统位数。
字符串（String）类型。
system 虚拟机安装的操作系统类型。0：Windows 1：Linux。
整数（int）类型。
osVersion

操作系统版本，枚举值：

当操作系统为Windows:

Microsoft Windows Server 2012, Microsoft Windows Server 2012 R2, Microsoft Windows Server 2008 R2(64-bit), Microsoft Windows Server 2008(64-bit), Microsoft Windows Server 2008(32-bit), Microsoft Windows Server 2003(64-bit), Microsoft Windows Server 2003(32-bit), Microsoft Windows 8(64-bit), Microsoft Windows 8(32-bit), Microsoft Windows 7(64-bit), Microsoft Windows 7(32-bit), Microsoft Windows Vista(64-bit), Microsoft Windows Vista(32-bit), Microsoft Windows XP Professional(64-bit), Microsoft Windows XP Professional(32-bit), Microsoft Windows 2000, Microsoft Windows NT, Microsoft Windows 98, Microsoft Windows 95, Microsoft Windows 3.1, Microsoft MS-DOS

当操作系统为Linux:

Red Hat Enterprise Linux6(64-bit), Red Hat Enterprise Linux6(32-bit), Red Hat Enterprise Linux5(64-bit), Red Hat Enterprise Linux5(32-bit), Red Hat Enterprise Linux4(64-bit), Red Hat Enterprise Linux4(32-bit), Red Hat Enterprise Linux3(64-bit), Red Hat Enterprise Linux3(32-bit), Red Hat Enterprise Linux2.1, Novell SUSE Linux Enterprise 11(64-bit), Novell SUSE Linux Enterprise 11(32-bit), Novell SUSE Linux Enterprise 10(64-bit), Novell SUSE Linux Enterprise 10(32-bit), Novell SUSE Linux Enterprise 8/9(64-bit), Novell SUSE Linux Enterprise 8/9(32-bit), CentOS 4/5/6(64bit), CentOS 4/5/6(32bit), Debian GUN/Linux 6(64-bit), Debian GUN/Linux 6(32-bit), Debian GUN/Linux 5(64-bit), Debian GUN/Linux 5(32-bit), Debian GUN/Linux 4(64-bit), Debian GUN/Linux 4(32-bit), Asianux 4(64-bit), Asianux 4(32-bit), Asianux 3(64-bit), Asianux 3(32-bit), Oracle Linux 4/5/6(64-bit), Oracle Linux 4/5/6(32-bit), Ubuntu Linux(64-bit), Ubuntu Linux(32-bit), Other 2.6.x Linux(64-bit), Other 2.6.x Linux(32-bit), Other 2.4.x Linux(64-bit), Other 2.4.x Linux(32-bit), Other Linux(64-bit), Other Linux(32-bit)
bootingDevice 引导设备。1：disk 2：cdrom。
整数（int）类型。
autoBooting 是否自动启动：0:不自动启动1:自动启动。
整数（int）类型。
autoMigrate 虚拟机是否允许自动迁移。0：不允许 1：允许
整数（int）类型。
createDate 虚拟机创建日期。
日期（Date）类型。
flag 虚拟机所属标志 0:不属于用户以及用户组 1:属于用户 2:属于用户组。
整数（int）类型。
type 虚拟机类型 0:CAS虚拟机 1:VMware虚拟机。
整数（int）类型。
deployed 是否快速部署过。注：该参数为模板参数，对虚拟机无意义。
布尔（Boolean）类型。
auto VNC端口自动配置 0：自动1：手动。
整数（int）类型。
existRaw 是否存在raw格式。
布尔（Boolean）类型。
protectModel 虚拟机是否启用保护模式。 1：启用，0：不启用
整数（int）类型。
network 虚拟机网卡信息列表。
集合（List）类型。
storage 虚拟机存储列表。
集合（List）类型。
network各子元素
mac 网卡MAC地址。
字符串（String）类型。
ipAddr 网卡IP地址。
字符串（String）类型。
vsName 网卡使用的虚拟交换机名称。
字符串（String）类型。
vlan 网卡所属的VLAN。
整数（int）类型。
isLimitInBound 是否限制入方向带宽。 true：限制 false：不限制。
布尔（boolean）类型。
inAvgBandwidth 入方向平均带宽。 单位（kbps）。
长整数（long）类型。
inPeakBandwidth 入方向峰值带宽。单位（kbps）。
长整数（long）类型。
inBurst 入方向突发缓冲大小。 单位（kBytes）。
长整数（long）类型。
isLimitOutBound 是否限制出方向带宽。 true：限制 false：不限制。
布尔（boolean）类型。
outAvgBandwidth 出方向平均带宽。 单位（kbps）。
长整数（long）类型。
outPeakBandwidth 出方向峰值带宽。单位（kbps）。
长整数（long）类型。
outBurst 出方向突发缓冲大小。 单位（kBytes）。
长整数（long）类型。
mode 转发模式。
字符串（String）类型。
virtualport 是否配置了vport。true:配置 false:未配置
布尔（boolean）类型。
address 物理网卡直连或者SR-IOV直连的地址。
字符串（String）类型。
netType 网卡类型：0：桥接 1：NAT 2：不使用网络 3：物理直通网卡 4：SR-IOV网卡。
整数（int）类型。
storage各子元素
storeFile 存储镜像文件名。
字符串（String）类型。
capacity 镜像文件大小。 单位（M）。
长整数（long）类型。
device 挂接逻辑设备名称。
字符串（String）类型。
diskDevice 磁盘类型。
字符串（String）类型。
targetBus 总线类型，枚举值：ide，scsi，virtio，usb。
字符串（String）类型。
type 存储类型，枚举值：file，block。
字符串（String）类型。
mode 预分配，0：延迟置零 1：置零 2：精简。
整型（Integer）类型。
clusterSize 存储卷簇大小，枚举值：65536（64K）、131072（128K）、262144（256K）、524288（512K）、1048576（1M）、2097152（2M）
字符串（String）类型。
gpu各子元素
bus GPU槽号。
字符串（String）类型。
producers 设备产商。
字符串（String）类型。
type vGPU的类型。
字符串（String）类型。
name vGPU的名称。
字符串（String）类型。
maxInstance vGPU的最大虚拟个数。
整数（int）类型。
framebuffer vGPU的缓存。
字符串（String）类型。
maxResolution vGPU的最大分辨率。
字符串（String）类型。
heads vGPU的显示器接口数。
整数（int）类型。
uuid vGPU的UUID唯一标识符。
字符串（String）类型。
