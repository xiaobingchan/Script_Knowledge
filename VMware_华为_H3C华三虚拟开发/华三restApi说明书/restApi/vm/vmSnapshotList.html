查询指定虚拟机快照信息

根据虚拟机id查询指定虚拟机的快照信息。

接口 URI

/cas/casrs/vm/snapshot/{id}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 虚拟机id 。
长整数（long）类型。必须，无缺省值。

返回结果

状态码
  • 200(OK)：成功。

元素含义
snapshot各子元素
name 快照名称。
字符串（String）类型。
desc 快照描述。
字符串（String）类型。
isCurr 虚拟机是否处于当前快照。 true：处于当前快照 false：不处于当前快照。
布尔（boolean）类型。
snapMem 快照内存：true 快照内存；false 快照不带内存。
布尔（boolean）类型。
noStopVirtSnapMem 不停机快照内存：true 不停机；false 停机。
布尔（boolean）类型。
snapshots 虚拟机子快照列表。
集合（List）类型。
