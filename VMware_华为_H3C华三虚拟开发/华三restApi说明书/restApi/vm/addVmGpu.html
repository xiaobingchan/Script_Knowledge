为指定虚拟机添加GPU设备

为指定虚拟机添加GPU设备。

接口 URI

/cas/casrs/resPool/addGpu

访问方法

HTTP 方法
POST
消息体
<respoolVm>
    <domainId>35</domainId>
    <businessTempId>1</businessTempId>
    <resPoolId>1</resPoolId>
</respoolVm>
元素含义
domain各子元素
domainId 虚拟机ID。
长整型（Long）类型，必须。
businessTempId 业务模板ID。
长整型（Long）类型，必须。
resPoolId 智能资源调度业务ID。
长整型（Long）类型，必须。

返回结果

状态码
  • 200 ：成功。
  • 409 (Conflict)：失败。

应用举例

为ID为35的虚拟机添加GPU设备。

请求
POST http://192.168.0.28:8080/cas/casrs/resPool/addGpu
accept: application/xml

<respoolVm>
    <domainId>35</domainId>
    <businessTempId>1</businessTempId>
    <resPoolId>1</resPoolId>
</respoolVm>
回应
<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<rsResult>
	<state>0</state>
	<errorCode>0</errorCode>
	<data>虚拟机“zxktest”增加GPU设备成功。</data>
</rsResult>
