为指定虚拟机添加显卡

为指定虚拟机添加显卡。

接口 URI

/cas/casrs/vm/addDevice

访问方法

HTTP 方法
PUT
消息体
<domain>
    <id>1</id>
    <videoType>qxl</videoType>
    <vram>1</vram>
</domain>
元素含义
domain各子元素
id 虚拟机ID。
长整型（Long）类型，必须。
videoType 虚拟机显卡类型。
字符串（String）类型，必须。 显卡类型必须为qxl 。
vram 枚举值：0:16M, 1:64M, 2:128M, 3:256M, 4:512M。
整型（Integer）类型，可选，不传代表增加的显卡显存大小为64M。

返回结果

状态码
  • 200(OK)：成功。
  • 409 (Conflict)：失败。

应用举例

为ID为1的虚拟机增加显存为64M的显卡。

请求
PUT http://192.168.0.4:8080/cas/casrs/vm/addDevice
accept: application/xml


<domain>
    <id>1</id>
    <videoType>qxl</videoType>
    <vram>1</vram>
</domain>
