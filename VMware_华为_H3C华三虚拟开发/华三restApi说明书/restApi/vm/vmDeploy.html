H3C CAS云管理平台

虚拟机部署

部署虚拟机

接口 URI

/cas/casrs/vm/deploy

访问方法

HTTP 方法
POST
消息体
demo 1：	
<domainConfig>
    <id>69</id>
    <domainName>ccc</domainName>
    <title>ccc</title>
    <targetHostId>1</targetHostId>
    <deployMode>1</deployMode>
    <protectMode>0</protectMode>
    <userData>mkdir vmFile</userData>
   <storage>
        <src>mini_for_v_cvkzyb_test4</src>
        <dest>ccc</dest>
        <pool>serverpool</pool>
    </storage>
    <network>
        <mac>0c:da:41:1d:cc:7b</mac>
        <vSwitchName>vswitch0</vSwitchName>
        <profileName>Default</profileName>
        <sysIpType>2</sysIpType>
        <ip>192.168.20.20</ip>
        <sysMask>255.255.0.0</sysMask>
        <sysGateway>192.168.20.1</sysGateway>
        <sysDns>8.8.8.8</sysDns>
        <secondaryDns>8.8.8.7</secondaryDns>
        <isBindIp>false</isBindIp>
    </network>
</domainConfig>

元素含义
domainConfig各子元素
id
 模板ID
长整数（long）类型。
domainName
 部署虚拟机名字。
字符串（String）类型。
title
 部署虚拟机显示名字。
字符串（String）类型。
desc
 虚拟机描述信息
字符串（String）类型。
targetHostId
 目标主机ID
长整型（long）类型。
deployModel
 部署虚拟机方式（0表述正常部署，1表示快速部署）。
整数（int）类型。
protectMode
 保护模式（0表示非保护模式，1表示保护模式），可选。
整数（int）类型。
userData
 部署保护模式虚拟机时，用户定制命令。可选。
字符串（String）类型。
storage
 虚拟机存储列表。
集合（List）类型。
network
 虚拟机网卡信息列表。
集合（List）类型。
storage各子元素
src
 原存储镜像文件名。
字符串（String）类型。
dest
 目的存储镜像文件名。
字符串（String）类型。
pool
 目的存储池。
字符串（String）类型。
network各子元素
mac
 虚拟机模板的网卡MAC地址。
字符串（String）类型。
vSwitchName
 网络交换机名称。
字符串（String）类型。
profileName
 网络策略模板名称。
字符串（String）类型。
sysIpType
 网络设置模式（1表示DHCP，2表示手工指定IP等信息）。
整数（int）类型。
ip
 网卡分配的IP地址。
字符串（String）类型。
sysMask
 子网掩码。
字符串（String）类型。
sysGateway
 网关地址。
字符串（String）类型。
sysDns
 首选DNS服务器地址。
字符串（String）类型。
secondaryDns
 备用DNS服务器地址。
字符串（String）类型。
isBindIp
 是否绑定网卡IP地址。
枚举值：true 或 false（字母需全部小写）。
osInfo各子元素（可选）
initType
 设置系统参数方式（0：快速初始化，1：完全初始化。）。
整数（Integer）类型。
sysName
 设置主机名。
字符串（String）类型。
regOrGroupType
 隶属于域或者工作组（1：域，2：工作组。）。
字符串（String）类型。
regOrGroup
 域或者工作组值。
字符串（String）类型。
loginAccount
 登录帐号。
字符串（String）类型。
loginPassword
 登录密码。
字符串（String）类型。
localgroup
 本地组。
字符串（String）类型。
sysIp
 设置一个网卡的IP地址。可选。
字符串（String）类型。
sysMask
 设置该网卡的子网掩码。可选(注：sysIp配置，该值必须配置。)。
字符串（String）类型。
sysGateway
 设置该网卡的默认网关地址。可选。
字符串（String）类型。
sysdns
 主DNS服务器。可选。
字符串（String）类型。
secondaryDns
 备DNS服务器。可选。
字符串（String）类型。
isBindIp
 设置ip信息的网卡是否设置绑定IP。
枚举值：true 或 false（字母需全部小写）。
