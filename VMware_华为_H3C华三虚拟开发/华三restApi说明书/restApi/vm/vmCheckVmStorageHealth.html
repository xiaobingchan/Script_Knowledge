检查虚拟机存储卷

根据虚拟机id查询指定虚拟机的存储卷是否存在。

接口 URI

/cas/casrs/vm/checkVmStorageHealth/{id}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 虚拟机id 。
长整数（long）类型。必须，无缺省值。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
消息体
<vmCheck>
	<domainId>5</domainId>
	<domainName>vv</domainName>
	<title>vv</title>
	<health>true</health>
</vmCheck>
元素含义
vmCheck各子元素
domainId 虚拟机ID。
长整数（long）类型。
domainName 虚拟机名称。
字符串（String）类型。
title 虚拟机显示名称。
字符串（String）类型。
health 存储卷是否都存在。true：存在 false：不存在。
布尔（boolean）类型。
result 检测不通过的原因。
字符串（String）类型。
 

应用举例

检测 ID为”52“的虚拟机存储卷信息。

请求
GET http://localhost:8080/cas/casrs/vm/checkVmStorageHealth/52
accept: application/xml
...
回应
HTTP/1.1 200 OK
Content-Type: application/xml
...
<vmCheck>
	<domainId>5</domainId>
	<domainName>vv</domainName>
	<title>vv</title>
	<health>true</health>
</vmCheck>
