修改虚拟机显卡配置

修改虚拟机显卡配置。

接口 URI

/cas/casrs/vm/modify

访问方法

HTTP 方法
PUT
消息体
<domain>
    <id>6</id>
    <name>test</name>
    <videoType>vga</videoType>
    <pciView>0000.00.02.0</pciView>
    <vram>1</vram>
</domain>
元素含义
domain各子元素
name 虚拟机名称。
字符串（String）类型，必须。
id 虚拟机id。
长整型（Long）类型，必须。
videoType 虚拟机显卡类型，枚举值：vga、cirrus、qxl。
字符串（String）类型，必须。
pciView 待修改显存大小的显卡的PCI 地址。
字符串（String）类型，可选。
vram 显存大小，枚举值：0:16M, 1:64M, 2:128M, 3:256M, 4:512M。
整型（Integer）类型，可选。不传代表不修改显存大小。
