H3C CAS云管理平台

注册虚拟机

注册指定的虚拟机

接口 URI

/cas/casrs/vm/register

访问方法

HTTP 方法
POST
消息体
<vm>
  <id>38</id>
  <name>testVM</name>
  <path>/root/testVM.xml</path>
</vm>
元素含义
vm各子元素
id 需要注册的虚拟机 ID。
长整数（long）类型。
name 需要注册的虚拟机名称。
字符串（String）类型。
path 需要注册的虚拟xml文件的绝对路径。
字符串（String）类型。

返回结果

状态码
  • 200(OK)：提交注册虚拟机任务成功。
  • 409(Conflict)：失败。
消息头
HTTP/1.1 200 OK
Server: Apache-Coyote/1.1
Set-Cookie: JSESSIONID=BDF7B4EDFDBA9FBF003D195F802D705F; Path=/imcrs
Date: Fri, 23 Dec 2011 04:44:21 GMT
<?xml version="1.0" encoding="UTF-8"?>
<taskMsg>
  <msgId>1393831560400</msgId>
  <targetId>38</targetId>
  <targetName>testVM</targetName>
  <completed>false</completed>
  <progress>0</progress>
  <eventType>0</eventType>
</taskMsg>
