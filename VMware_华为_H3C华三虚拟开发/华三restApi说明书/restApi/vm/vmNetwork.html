H3C CAS云管理平台

查询指定虚拟机网络信息

根据虚拟机id查询指定虚拟机的网络信息。

接口 URI

/cas/casrs/vm/network/{id}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 虚拟机id 。
长整数（long）类型。必须，无缺省值。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK
消息体
<list>
  <network>
    <mac>02:36:b8:a5:f2:3d</mac>
    <ipAddr>192.168.0.253</ipAddr>
    <vsName>vswitch0</vsName>
    <vlan>0</vlan>
    <isLimitInBound>false</isLimitInBound>
    <isLimitOutBound>false</isLimitOutBound>
    <mode>VEB</mode>
    <virtualport>false</virtualport>
  </network>
  <network>
    <mac>02:36:b8:c4:db:a6</mac>
    <vsName>vs3</vsName>
    <vlan>20</vlan>
    <isLimitInBound>false</isLimitInBound>
    <isLimitOutBound>false</isLimitOutBound>
    <mode>VEPA</mode>
    <virtualport>false</virtualport>
  </network>
</list>
元素含义
network各子元素
mac 网卡MAC地址。
字符串（String）类型。
ipAddr 网卡IP地址。
字符串（String）类型。
vsName 网卡使用的虚拟交换机名称。
字符串（String）类型。
vlan 网卡所属的VLAN。
整数（int）类型。
isLimitInBound 是否限制入方向带宽。 true：限制 false：不限制。
布尔（boolean）类型。
inAvgBandwidth 入方向平均带宽。 单位（kbps）。
长整数（long）类型。
inPeakBandwidth 入方向峰值带宽。单位（kbps）。
长整数（long）类型。
inBurst 入方向突发缓冲大小。 单位（kBytes）。
长整数（long）类型。
isLimitOutBound 是否限制出方向带宽。 true：限制 false：不限制。
布尔（boolean）类型。
outAvgBandwidth 出方向平均带宽。 单位（kbps）。
长整数（long）类型。
outPeakBandwidth 出方向峰值带宽。单位（kbps）。
长整数（long）类型。
outBurst 出方向突发缓冲大小。 单位（kBytes）。
长整数（long）类型。
mode 转发模式。
字符串（String）类型。
virtualport 是否配置了vport。true:配置 false:未配置
布尔（boolean）类型。
address 物理网卡直连或者SR-IOV直连的地址。
字符串（String）类型。
netType 网卡类型：0：桥接 1：NAT 2：不使用网络 3：物理直通网卡 4：SR-IOV网卡。
整数（int）类型。
