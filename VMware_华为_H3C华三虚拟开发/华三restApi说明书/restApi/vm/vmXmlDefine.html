H3C CAS云管理平台

查询虚拟机XML信息

根据虚拟机id查询指定虚拟机XML信息。

接口 URI

/cas/casrs/vm/xml/{id}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 虚拟机id 。
长整数（long）类型。必须，无缺省值。

返回结果

状态码
  • 200(OK)：成功。
