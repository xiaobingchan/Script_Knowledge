还原虚拟机

从备份文件还原指定的虚拟机

接口 URI

/cas/casrs/vm/v2/restore?backupId={backupId}&tmpDir={tmpDir}&isForce={isForce}

访问方法

HTTP 方法
PUT
参数
查询参数（Query Parameter）
backupId 备份记录 id 。
长整数（Long）类型。必须，无缺省值。
tmpDir 还原时所用的临时目录（空间必须足够大）。
字符串（String）类型。必须，无缺省值。
isForce 在虚拟机磁盘配置发生变化时（增删了磁盘）是否强制还原
字符串（String）类型。非必须，缺省值 true 。

  • 409(Conflict)：失败。
