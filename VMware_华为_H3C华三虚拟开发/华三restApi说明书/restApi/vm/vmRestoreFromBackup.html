H3C CAS云管理平台

还原虚拟机

从备份文件还原指定的虚拟机

接口 URI

/cas/casrs/vm/restore

访问方法

HTTP 方法
PUT
消息体

1.从主机本地目录还原

<vmRestoreParameter>
    <backupId>10</backupId>
    <entryType>VIRTUAL_HOST</entryType>
    <type>cp</type>
    <directory>/vms/images</directory>
    <vmName>eq</vmName>
    <hostId>72</hostId>
    <time>2014-11-03 19:43:19</time>
    <rsDiskForRestore>
        <dataName>/vms/images/ddd</dataName>
        <format>qcow</format>
        <size>6</size>
        <path>/vms/images/ddd</path>
    </rsDiskForRestore>
    <title>qw</title>
    <isForce>false</isForce>
    <tmpDir>/vms/vmbackuptmp</tmpDir>
</vmRestoreParameter>

2.从远端服务器还原

<vmRestoreParameter>
    <backupId>10</backupId>
    <entryType>VIRTUAL_HOST</entryType>
    <type>scp</type>
    <directory>/vms/images</directory>
    <targetAddr>192.168.10.4</targetAddr>
    <userName>root</userName>
    <password>1q2w3e</password>
    <vmName>eq</vmName>
    <hostId>72</hostId>
    <time>2014-11-03 19:44:55</time>
    <rsDiskForRestore>
        <dataName>/vms/images/ddd</dataName>
        <format>qcow</format>
        <size>6</size>
        <path>/vms/images/ddd</path>
    </rsDiskForRestore>
    <title>qw</title>
    <isForce>false</isForce>
    <tmpDir>/vms/vmbackuptmp</tmpDir>
</vmRestoreParameter>
元素含义
vmRestoreParameter各子元素
backupId 备份记录ID。
长整数（Long）类型。
entryType 操作类型，此处必须为VIRTUAL_HOST。
字符串（String）类型。
type 恢复方式，cp：主机本地目录，scp：SSH/SCP方式，ftp：FTP方式。
枚举(enum):cp，scp，ftp
directory 备份文件位置 。
字符串（String）类型。
targetAddr 备份文件所在服务器IP地址
字符串（String）类型。
userName 备份文件所在服务器用户名
字符串（String）类型。
password 备份文件所在服务器器密码
字符串（String）类型。
vmName 虚拟机名称
字符串（String）类型。
hostId 主机ID
长整形(Long)类型。
time 备份文件时间
字符串（String）类型。
hostId 主机ID
长整形(Long)类型。
title 虚拟机显示名称。
字符串（String）类型。
isForce 虚拟机磁盘发生变化，是否强制还原。
枚举（enum类型。 true， fasle
rsDiskForRestore 磁盘信息
自定义类型。
rsDiskForRestore各子元素
dataName 备份磁盘路径。
字符串（String）类型。
format 磁盘文件格式。 qcow2或raw
字符串（String）类型。
size 备份文件大小 。
整形(Integer)
path 磁盘还原路径。
字符串（String）类型。
