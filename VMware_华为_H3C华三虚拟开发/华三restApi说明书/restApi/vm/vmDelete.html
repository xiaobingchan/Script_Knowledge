H3C CAS云管理平台

删除虚拟机

删除指定ID的虚拟机。

接口 URI

/cas/casrs/vm/delete/{id}

访问方法

HTTP 方法
DELETE
参数
路径参数（Path Parameter）
id 虚拟机id 。
长整数（Long）类型。必须，无缺省值。

返回结果

状态码
  • 200(OK)：提交删除任务成功。
  • 409(Conflict)：失败。
