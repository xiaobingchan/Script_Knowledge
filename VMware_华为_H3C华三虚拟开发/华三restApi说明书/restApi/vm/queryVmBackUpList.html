H3C CAS云管理平台

查询虚拟机备份列表信息

查询指定ID查看虚拟机做过的备份信息。

接口 URI

/cas/casrs/backupStrategy/backupFileTree?domainId={id}

访问方法

HTTP 方法
GET
消息体
无
元素含义
路径参数（Path Parameter）
id 虚拟机 ID。
长整数（long）类型，必须。

  • 200 OK：成功。

元素含义
各子元素
id 备份文件的id。
长整型（Long）类型。
name 备份文件名称。
字符串（String）类型。
type 备份目的地：cp ：本地，scp ：远程ssh方式， ftp。：远程ftp方式。
字符串（String）类型。
mode 备份方式 0-全量，1-增量 10-首次增量的全量备份， 11-增量备份，20-首次差异备份，22-差异备份。
整型（Integer）类型。
location 备份目的描述。
字符串（String）类型。
domainId 虚拟机ID
长整数（Long）。
time 备份时间。
字符串（String）类型。
size 备份文件大小。
数字（Double）类型。
parentId 父备份文件的ID。
长整数（Long）。
directory 备份路径。
字符串（String）类型。
backupType 备份类型：0 –整机备份，1-磁盘单独备份。
整型（Integer）类型。
targetAddr 远端备份时，服务器地址
字符串（String）类型。
children 子节点，机基于当前备份做的差异备份或者增量备份。
自定义类型。
