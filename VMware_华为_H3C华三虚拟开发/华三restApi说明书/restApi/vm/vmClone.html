H3C CAS云管理平台

克隆虚拟机

克隆指定的虚拟机

接口 URI

/cas/casrs/vm/clone

访问方法

HTTP 方法
PUT
消息体
<vmCloneParameter>
  <id>38</id>
  <targetHostId>3</targetHostId>
  <domainName>jj02-clone5</domainName>
  <title>jj02-clone5</title>
  <cloneMode>1</cloneMode>
  <cloneType>0</cloneType>
  <diskFormat>qcow2</diskFormat>
  <storage>
    <src>jj02</src>
    <dest>jj02-clone5</dest>
    <pool>defaultpool</pool>
  </storage>
  <network>
    <key>0c:da:41:1d:5e:f2</key>
    <value>3.3.3.3</value>
  </network>
  <net>
    <mac>0c:da:41:1d:59:be</mac>
    <name>vswitch0</name>
    <sysIpType>2</sysIpType>
    <sysIp>192.168.1.185</sysIp>
    <sysGateway>192.168.1.1</sysGateway>
    <sysDns>2.2.2.2</sysDns>
    <secondaryDns>3.3.3.3</secondaryDns>
    <sysMask>255.255.255.0</sysMask>
    <isBindIp>true</isBindIp>
  </net>
</vmCloneParameter>
元素含义
vmParamter各子元素
id 需要克隆的虚拟机 ID。
长整数（long）类型。
targetHostId 目标主机ID。
长整数（long）类型。
domainName 目标虚拟机名称。
字符串（String）类型。
title 虚拟机显示名称。
字符串（String）类型。
cloneMode 克隆方式：枚举:0表示普通克隆，1表示快速克隆。
枚举(enum):0，1
cloneType 克隆目的位置：枚举:0表示本地克隆，1表示主机间克隆。
枚举(enum):0，1
diskFormat 磁盘类型，支持raw和qcow2格式，如不填写，则不修改磁盘类型。
枚举：raw，qcow2。
storage 虚拟机存储列表。
集合（List）类型。
network 虚拟机网卡信息列表。
集合（List）类型。
net OS网络信息列表（通过castools设置）。
集合（List）类型。
storage各子元素
src 原存储镜像文件的名称。
字符串（String）类型。
dest 目的存储镜像文件名。
字符串（String）类型。
pool 目的存储池。
字符串（String）类型。
network各子元素
key 网卡MAC地址。
字符串（String）类型。
value 网卡绑定的IP地址。
字符串（String）类型。
net各子元素
mac 网卡MAC地址。
字符串（String）类型。
name 虚拟交换机名称。
字符串（String）类型。
sysIpType ip分配方式 0:默认 1:DHCP 2:手动分配
整数（Integer）类型。
sysIp ip地址。
字符串（String）类型。
sysGateway 默认网关。
字符串（String）类型。
sysDns 主DNS。
字符串（String）类型。
secondaryDns 备DNS。
字符串（String）类型。
sysMask 子网掩码。
字符串（String）类型。
isBindIp 是否绑定ip。
Boolean类型。

  • 409(Conflict)：失败。
