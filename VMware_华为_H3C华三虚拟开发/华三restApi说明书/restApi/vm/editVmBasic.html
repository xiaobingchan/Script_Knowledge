修改虚拟机基本概要信息

修改虚拟机基本概要信息。

接口 URI

/cas/casrs/vm/modify

访问方法

HTTP 方法
PUT
消息体
<domain>
    <id>6</id>
    <name>test</name>
    <title>test</title>
    <basic>
        <pae>0</pae>
        <acpi>0</acpi>
        <apic>0</apic>
        <clock>utc</clock>
        <blkiotune>300</blkiotune>
        <startPriority>0</startPriority>
        <autoMigrate>0</autoMigrate>
        <desc>test rest modify domain</desc>
        <osha>1</osha>
        <timeSync>1</timeSync>
        <haManage>0</haManage>
    </basic>
</domain>
元素含义
domain各子元素
id 虚拟机id。
长整型（Long）类型，必须。
name 虚拟机名称。
字符串（String）类型，必须。
title 虚拟机显示名称。
字符串（String）类型，必须。
basic 虚拟机概要信息。
自定义类型，必须。
basic各子元素
pae 是否开启PAE，枚举值，1：开启，0：不开启。
整型（Integer）类型，必须。
acpi 是否开启ACPI，枚举值，1：开启，0：不开启。
整型（Integer）类型，必须。
apic 是否开启APIC，枚举值，1：开启，0：不开启。
整型（Integer）类型，必须。
clock 时钟设置，枚举值：utc：世界时钟，localtime：本地时钟。
字符串（String）类型，必须。
blkiotune I/O优先级，枚举值：500：高，300：中，200：低。
整型（Integer）类型，必须。
startPriority 启动优先级，当集群启用HA后有效，枚举值：0：低级，1：中级，2：高级。
整型（Integer）类型，必须。
autoMigrate 是否允许虚拟机自动迁移，实现资源动态调整，枚举值：1=允许，0=不允许。
整型（Integer）类型，必须。
desc 描述,最大输入长度为120个字符。
字符串（String）类型，可选项。
osha 系统故障策略 0:不处理 1:故障重启 2:故障迁移。
整型（Integer）类型，可选项。
timeSync 同步虚拟机时间与主机时间 1:同步，0：不同步
整型（Integer）类型，可选项。
haManage 虚拟机HA管理标志 1:开启高可靠性，0：关闭高可靠性。
整型（Integer）类型，可选项。

返回结果

状态码
  • 204(No Content)：成功。
  • 409(Conflict)：失败。
消息头
HTTP/1.1 204 No Content
Server: Apache-Coyote/1.1
Date: Mon, 25 Nov 2013 12:52:27 GMT
消息体
无
