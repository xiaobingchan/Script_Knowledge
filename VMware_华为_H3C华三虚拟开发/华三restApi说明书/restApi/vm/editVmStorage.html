H3C CAS云管理平台

修改虚拟机存储配置

修改虚拟机存储容量、格式、磁盘缓存方式。

接口 URI

/cas/casrs/vm/modify

访问方法

HTTP 方法
PUT
消息体
<domain>
    <id>6</id>
    <name>test</name>
    <storage>
        <device>hda</device>
        <format>qcow2</format>
        <size>2048</size>
        <cacheType>writeback</cacheType>
        <readBytesSec>16</readBytesSec>
        <writeBytesSec>16</writeBytesSec>
        <readIopsSec>16</readIopsSec>
        <writeIopsSec>16</writeIopsSec>
	<hotPluggable>true</hotPluggable>
    </storage>
</domain>
元素含义
domain各子元素
name 虚拟机名称。
字符串（String）类型，必须。
id 虚拟机id。
长整型（Long）类型，必须。
storage 虚拟机存储配置。
自定义类型，必须。
storage各子元素
device 存储卷设备名称，如hda，hdb。
字符串（String）类型，必须。
format 存储格式，枚举值：raw,qcow2。
字符串（String）类型，必须。
size 存储容量，单位为MB。
长整型（Long）类型，必须。
cacheType 磁盘缓存方，枚举值：directsync，writethrough，writeback，none。
字符串（String）类型，必须。
readBytesSec 限制I/O速率，读，单位KB/s。
长整型（Long）类型。
writeBytesSec 限制I/O速率，写，单位KB/s。
长整型（Long）类型。
readIopsSec 限制IOPS，读。
长整型（Long）类型。
writeIopsSec 限制IOPS，写。
长整型（Long）类型。
hotPluggable 热拔。（true：允许，false：不允许）
布尔（Boolean）类型，可选。
