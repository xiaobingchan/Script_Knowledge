修改虚拟机内存配置

修改虚拟机内存配置，包括内存大小分配和内存预留配置。

接口 URI

/cas/casrs/vm/modify

访问方法

HTTP 方法
PUT
消息体
<domain>
    <id>6</id>
    <name>test</name>
    <memory>
        <size>128</size>
        <memoryBacking>10</memoryBacking>
        <memoryLimit>10</memoryLimit>
        <memoryLimitUnit>MB</memoryLimitUnit>
        <memoryPriority>100</memoryPriority>
    </memory>
</domain>
元素含义
domain各子元素
name 虚拟机名称。
字符串（String）类型，必须。
id 虚拟机id。
长整型（Long）类型，必须。
memory 虚拟机内存配置。
自定义类型，必须。
memory各子元素
size 内存大小，单位为MB，最小为128MB，最大根据主机实际情况决定。
长整型（Long）类型，必须。
memoryBacking 内存预留百分比。
整型（Integer）类型，取值范围为0-100，必须。
memoryLimit 内存限制。
小数（double）类型，可选项。
memoryLimitUnit 内存限制单位，取值为MB，GB。
字符串（String）类型，可选项。
memoryPriority 内存资源优先级。 高=100，中=50，低=0。
整数（int）类型，可选项。
