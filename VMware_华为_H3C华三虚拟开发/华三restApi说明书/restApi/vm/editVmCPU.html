修改虚拟机CPU配置

修改虚拟机CPU配置，包括虚拟机个数和核数分配，绑定物理CPU。

接口 URI

/cas/casrs/vm/modify

访问方法

HTTP 方法
PUT
消息体
<domain>
    <id>6</id>
    <name>test</name>
    <cpu>
        <cpuSockets>1</cpuSockets>
        <cpuCores>2</cpuCores>
        <cpuShares>256</cpuShares>
        <cpuArch>i686</cpuArch>
        <cpuMode>host-model</cpuMode>
        <cpuQuota>1</cpuQuota>
        <cpuQuotaUnit>GHz</cpuQuotaUnit>
        <cpuGurantee>10</cpuGurantee>
        <bindcpu>
            <vcpu>0</vcpu>
            <pcpu>0</pcpu>
            <pcpu>1</pcpu>
        </bindcpu>
        <bindcpu>
            <vcpu>1</vcpu>
            <pcpu>0</pcpu>
        </bindcpu>
    </cpu>
</domain>
元素含义
domain各子元素
name 虚拟机名称。
字符串（String）类型，必须。
id 虚拟机id。
长整型（Long）类型，必须。
cpu 虚拟机CPU配置。
自定义类型，必须。
cpu各子元素
cpuSockets CPU个数，最小值为1，最大值由主机实际情况决定。
整型（Integer）类型，必须。
cpuCores CPU核数，最小值为1，最大值由主机实际情况决定。
整型（Integer）类型，必须。
cpuShares CPU调度优先级，枚举值：1024：高，512：中，256：低。
整型（Integer）类型，必须。
cpuMode CPU工作模式，取值为：custom、host-model、host-passthrough。
字符串（String）类型，可选项。
cpuArch CPU体系结构，取值为：x86_64、i686。
字符串（String）类型，可选项。
cpuQuota CPU周期限制值
小数（double）类型，可选项。
cpuQuotaUnit CPU周期限制 单位，取值为MHz,GHz
字符串（String）类型，可选项。
cpuGurantee CPU预留，单位：MHz
长整数（long）类型。，可选项。
bindcpu 虚拟CPU绑定物理CPU配置。
自定义类型，可选项。
bindcpu各子元素
vcpu 指定虚拟CPU，0：表示第一个虚拟CPU，1：表示第二个虚拟CPU，依次类推。
整型（Integer）类型，必须。
pcpu 指定物理CPU，0：表示第一个物理CPU，1：表示第二个物理CPU，依次类推。一个虚拟CPU可绑定多个物理CPU。
整型（Integer）类型，必须。
