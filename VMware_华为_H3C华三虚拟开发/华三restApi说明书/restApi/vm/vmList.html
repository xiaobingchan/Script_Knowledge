H3C CAS云管理平台

按条件查询所有符合条件的虚拟机

根据条件查询所有符合条件的虚拟机。

接口 URI

/cas/casrs/vm/vmList

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
hostId 主机id 。 可以按主机id查询指定主机下的虚拟机。
长整数（long）类型。可选。
clusterId 集群id 。 可以按集群id查询指定集群下的虚拟机。
长整数（long）类型。可选。
hpId 主机池id 。 可以按主机池id查询指定主机池下的虚拟机。
长整数（long）类型。可选。
offset 返回的第一条记录在总结果集中的位置。
整数（int）类型，非必须，缺省为 0。如果传入参数小于 0，则使用缺省值 0。
limit 返回记录总数。
整数（int）类型，非必须，缺省为 10。如果传入参数小于 0，则使用缺省值 10。
orderBy 排序字段。
字符串（String）类型，非必须。如按虚拟机名称排序，则参数值为title,按创建日期排序，则参数值为createDate。其他支持的排序字段：id description memory cpu system

状态码
  • 200(OK)：成功。

元素含义
domain各子元素
id 虚拟机 ID。
长整数（long）类型。
hostId 运行的主机ID。
长整数（long）类型。
clusterId 所属的集群 ID。
长整数（long）类型。
hostPoolId 所属的主机池 ID。
长整数（long）类型。
name 虚拟机名称。
字符串（String）类型。
title 虚拟机显示名称。
字符串（String）类型。
memory 虚拟机内存大小，单位（M） 。
整数（int）类型。
cpu 虚拟机CPU个数。
整数（int）类型。
vmStatus 虚拟机状态。
字符串（String）类型。 running：运行 shutOff：关闭 paused：暂停。
cpuRate 虚拟机CPU利用率。
浮点数（Double）类型。
memRate 虚拟机内存利用率。
浮点数（Double）类型。
system 虚拟机安装的操作系统类型。0：Windows 1：Linux。
整数（int）类型。
osDesc 虚拟机操作系统详细版本号信息。 如： Microsoft Windows Server 2012
字符串（String）类型。
hostName 虚拟机属主主机名称。
字符串（String）类型。
