修改指定虚拟机名称

修改指定ID的虚拟机名称

接口 URI

/cas/casrs/vm/rename/{id}/{newName}

访问方法

HTTP 方法
PUT
参数
路径参数（Path Parameter）
id 虚拟机id 。
长整数（Long）类型。必须，无缺省值。
newName 虚拟机新名称。
字符串（String）类型。必须，无缺省值。

返回结果

状态码
  • 200(OK)：提交删除任务成功。
  • 409(Conflict)：失败。

应用举例

修改ID为“331”的虚拟机名称为vm

请求
PUT http://192.168.0.4:8080//cas/casrs/vm/rename/331/vm
accept: application/xml
...
回应
HTTP/1.1 204 No Content
Date: Wed, 17 Sep 2014 00:48:42 GMT
Server: H3C-CVM

无消息体
