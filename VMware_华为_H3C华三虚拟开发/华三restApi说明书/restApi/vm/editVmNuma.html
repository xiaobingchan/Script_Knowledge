H3C CAS云管理平台

修改虚拟机NUMA配置

配置虚拟机NUMA节点。

接口 URI

/cas/casrs/vm/modify

访问方法

HTTP 方法
PUT
消息体
<domain>
	<id>10</id>
	<name>vm01</name>
	<title>vm01</title>
	<numa>
	   <relevance>1</relevance>
	   <nodeSet>0</nodeSet>
	</numa>
</domain>
元素含义
domain各子元素
name 虚拟机名称。
字符串（String）类型，必须。
id 虚拟机id。
长整型（Long）类型，必须。
title 虚拟机显示名称。
字符串（String）类型，必须。
numa 虚拟机NUMA配置信息。
自定义类型，必须。
numa各子元素
relevance 虚拟机NUMA关联关系，0：不关联；1：关联
整型（Intager）类型。
nodeSet 虚拟机选用的numa节点
字符串（String）类型。

返回结果

状态码
  • 200(No Content)：成功。
  • 409(Conflict)：失败。
消息头
HTTP/1.1 200 OK
Set-Cookie: JSESSIONID=83A09EEC3A8073532609C059582B99F3; Path=/cas
Content-Type: application/xml
Content-Length: 325
消息体
    	
<?xml version="1.0" encoding="UTF-8"?>

  <name>修改虚拟机“test”NUMA配置成功。 </name>

</taskMsg>
