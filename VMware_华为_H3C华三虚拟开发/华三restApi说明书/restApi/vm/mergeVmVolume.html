H3C CAS云管理平台

合并虚拟机镜像

根据虚拟机ID合并关闭状态下的虚拟机的多级镜像文件。

接口 URI

/cas/casrs/vm/mergeDomainVol/{vmId}

访问方法

HTTP 方法
PUT
参数
路径参数（Path Parameter）
vmId 虚拟机id 。
长整数（long）类型。

返回结果

状态码
  • 200(OK)：成功。
消息头

HTTP/1.1 200 OK
消息体
<?xml version="1.0" encoding="UTF-8"?><taskMsg>
  <msgId>1503450562485</msgId>
  <completed>false</completed>
  <progress>0</progress>
  <eventType>0</eventType>
</taskMsg>

应用举例

1.合并虚拟机ID为50的虚拟机镜像文件。

请求
PUT http://localhost:8080/cas/casrs/vm/mergeDomainVol/50
accept: application/xml
...
回应
HTTP/1.1 200 OK
Cache-Control: no-cache, no-store, max-age=0, must-revalidate
Pragma: no-cache
...
<?xml version="1.0" encoding="UTF-8"?>
<taskMsg>
  <msgId>1503450562485</msgId>
  <targetId>50</targetId>
  <targetName>vm50</targetName>
  <completed>false</completed>
  <progress>0</progress>
  <eventType>0</eventType>
</taskMsg>
