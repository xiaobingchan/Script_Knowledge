虚拟机光驱断开连接

虚拟机光驱断开连接。

接口 URI

/cas/casrs/vm/modify

访问方法

HTTP 方法
PUT
消息体
<domain>
    <id>6</id>
    <name>test</name>
    <cdrom>
        <device>hdc</device>
        <operation>disconnect</operation>
    </cdrom>
</domain>
元素含义
domain各子元素
name 虚拟机名称。
字符串（String）类型，必须。
id 虚拟机id。
长整型（Long）类型，必须。
cdrom 光驱配置。
自定义类型，必须。
cdrom各子元素
device 光驱设备名称。
字符串（String）类型，必须。
operation 光驱操作，必须为：disconnect。
字符串（String）类型，必须。

返回结果

状态码
  • 204(No Content)：成功。
  • 409(Conflict)：失败。
消息头
HTTP/1.1 204 No Content
Server: Apache-Coyote/1.1
Date: Mon, 25 Nov 2013 12:52:27 GMT
消息体
    	
<?xml version="1.0" encoding="UTF-8"?>
<taskMsg>
  <msgId>-1</msgId>
  <name>断开虚拟机“test”光驱成功。 </name>
  <detail>断开虚拟机“test”光驱。 </detail>
  <completed>false</completed>
  <progress>0</progress>
  <eventType>0</eventType>
</taskMsg>

应用举例

断开虚拟机test的光驱hdc。

请求
POST http://192.168.0.4:8080/cas/casrs/vm/modify
accept: application/xml


<domain>
    <id>6</id>
    <name>test</name>
    <cdrom>
        <device>hdc</device>
        <operation>disconnect</operation>
    </cdrom>
</domain>
