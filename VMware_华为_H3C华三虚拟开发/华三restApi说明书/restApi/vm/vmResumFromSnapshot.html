H3C CAS云管理平台

从快照还原

为指定虚拟机的快照还原。

接口 URI

/cas/casrs/vm/snapshot/resume

访问方法

HTTP 方法
PUT
消息体
<snapshot>
  <vmId>31</vmId>
  <name>test4</name>
</snapshot>
元素含义
snapshot各子元素
vmId 虚拟机 ID。
长整数（long）类型。
name 快照名称。
字符串（String）类型。

返回结果

状态码
  • 200(OK)：提交删除任务成功。
  • 409(Conflict)：失败。
消息头
HTTP/1.1 200 OK
Server: Apache-Coyote/1.1
Set-Cookie: JSESSIONID=BDF7B4EDFDBA9FBF003D195F802D705F; Path=/imcrs
Date: Fri, 23 Dec 2011 04:44:21 GMT
<?xml version="1.0" encoding="UTF-8"?>
<taskMsg>
  <msgId>1393831560400</msgId>
  <targetId>3</targetId>
  <targetName>vm110-1</targetName>
  <completed>false</completed>
  <progress>0</progress>
</taskMsg>

应用举例

从名称为"test4"的快照还原虚拟机。

请求
PUT http://192.168.0.4:8080/cas/casrs/vm/snapshot/resume
accept: application/xml


<snapshot>
  <vmId>87</vmId>
  <name>test4</name>
</snapshot>

<taskMsg>
  <msgId>1393831560400</msgId>
