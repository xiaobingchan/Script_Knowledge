H3C CAS云管理平台

修改虚拟机高级配置

配置USB重定向和SSL通道。

接口 URI

/cas/casrs/vm/modify

访问方法

HTTP 方法
PUT
消息体
<domain>
	<id>10</id>
	<name>vm01</name>
	<title>vm01</title>
	<advance>
		<redirUsb>0</redirUsb>
		<spiceTls>true</spiceTls>
		<spiceChannel>main</spiceChannel>
		<spiceChannel>inputs</spiceChannel>
		<enableAntivirus>true</enableAntivirus>
	</advance>
</domain>
元素含义
domain各子元素
name 虚拟机名称。
字符串（String）类型，必须。
id 虚拟机id。
长整型（Long）类型，必须。
title 虚拟机显示名称。
字符串（String）类型，必须。
advance 虚拟机高级配置信息。
自定义类型，必须。
advance各子元素
redirUsb USB重定向数量，0表示没有，其他表示增加个数。
整型（Intager）类型。
spiceTls SSL的通道名称
布尔型（Boolean）类型（true：设置SSL通道，false：不设置SSL通道）。
spiceChannel SSL的通道名称
字符串（String）类型。
enableAntivirus 是否启用防病毒
布尔型（Boolean）类型（true：启用，false：不启用）。

状态码
  • 200(No Content)：成功。
  • 409(Conflict)：失败。

Content-Length: 325
