为指定虚拟机添加Spice控制台

为指定虚拟机添加Spice控制台。

接口 URI

/cas/casrs/vm/addDevice

访问方法

HTTP 方法
PUT
消息体
<domain>
	<id>72</id>
	<name>test</name>
	<title>test</title>
	<vnc>
		<type>spice</type>
		<password>123456</password>
		<port>5903</port>
		<enableSpiceCompress>1</enableSpiceCompress>		
	</vnc>
</domain>
元素含义
domain各子元素
id 虚拟机ID。
长整型（Long）类型，必须。
name 虚拟机名称。
字符串（String）类型，必须。
title 虚拟机显示名称。
字符串（String）类型，必须
vnc 控制台配置。
自定义类型，必须。
vnc各子元素
type 控制台类型，枚举值：spice，vnc。
字符串（String）类型。必须。
password 控制台密码。
字符串（String）类型。可选。
port 控制台端口。
整型（Integer）类型，可选。
enableSpiceCompress 是否启用图像压缩。
整型（Integer）类型，必须。1是启用，0不启用。

返回结果

状态码
  • 204 (No Content)：成功。
  • 409 (Conflict)：失败。

应用举例

为ID为72的虚拟机增加Spice控制台。

请求
PUT http://192.168.0.4:8080/cas/casrs/vm/addDevice
accept: application/xml
<domain>
	<id>72</id>
	<name>rest</name>
	<title>rest</title>
	<vnc>
		<type>spice</type>
		<password>123456</password>
		<port>5903</port>
		<enableSpiceCompress>1</enableSpiceCompress>		
	</vnc>
</domain>
回应
    	
<?xml version="1.0" encoding="UTF-8"?>
<taskMsg>
	<state>0</state>
	<errorCode>0</errorCode>
	<data>为虚拟机“rest”增加 SPICE控制台。</data>
	<message>为虚拟机“rest”增加 SPICE控制台成功。 </message>
</taskMsg>
