H3C CAS云管理平台

卸载虚拟机的虚拟磁盘

卸载虚拟机的虚拟磁盘，包括光驱、软驱。

接口 URI

/cas/casrs/vm/delDevice

访问方法

HTTP 方法
PUT
消息体
<domain>
	<id>10</id>
	<name>vm01</name>
	<storage>
		<deviceName>/dev/sdb</deviceName>
	</storage>
</domain>
元素含义
domain各子元素
id 虚拟机ID。
长整型（Long）类型，必须。
name 虚拟机名称。
字符串（String）类型，必须。
storage各子元素
deviceName 磁盘设备名称。
字符串（String）类型，必须。

返回结果

状态码
  • 204 (No Content)：成功。

应用举例

删除ID为10的虚拟机上的磁盘hdb。

请求
PUT http://192.168.0.4:8080/cas/casrs/vm/delDevice/10/hdb
accept: application/xml

<domain>
	<id>10</id>
	<name>vm01</name>
	<storage>
		<deviceName>/dev/hdb</deviceName>
	</storage>
</domain>
回应
HTTP/1.1 204 No Content
Server: Apache-Coyote/1.1
Date: Mon, 25 Nov 2013 12:27:38 GMT

无消息体
