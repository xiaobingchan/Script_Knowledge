卸载虚拟机的GPU设备

卸载虚拟机的GPU设备

接口 URI

/cas/casrs/vm/delDevice

访问方法

HTTP 方法
PUT
消息体
<domain>
    <id>35</id>
    <name>zxktest</name>
    <title>zxktest</title>
    <gpu/>
</domain>
元素含义
domain各子元素
id 虚拟机ID。
长整型（Long）类型，必须。
name 虚拟机名称。
字符串（String）类型，必须。
title 虚拟机显示名称。
字符串（String）类型，必须。
gpu 虚拟机gpu设备信息。
自定义类型，必须。

返回结果

状态码
  • 204 (No Content)：成功。

应用举例

删除ID为35的虚拟机上的GPU设备。

请求
PUT http://192.168.0.28:8080/cas/casrs/vm/delDevice
accept: application/xml
<domain>
    <id>35</id>
    <name>zxktest</name>
    <title>zxktest</title>
    <gpu/>
</domain>
回应
HTTP/1.1 204 No Content
Server: Apache-Coyote/1.1
Date: Mon, 25 Nov 2013 12:27:38 GMT
无消息体
