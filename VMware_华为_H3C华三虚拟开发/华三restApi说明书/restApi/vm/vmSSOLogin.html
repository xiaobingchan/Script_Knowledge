H3C CAS云管理平台

自动登录虚拟机中的操作系统

自动指定ID的虚拟机中的操作系统，实现单点登录。

接口 URI

/cas/casrs/spice/login

访问方法

HTTP 方法
PUT
消息体
<spice>
  <id>31</id>
  <userName>administrator</userName>
  <domain>h3c/domain>
  <password>123456/password>
</spice>
元素含义
spice各子元素

id

 虚拟机 ID。
长整数（long）类型。
userName 操作系统登录用户名。
字符串（String）类型。
domain 登录域。
字符串（String）类型。
password 操作系统登录密码。
字符串（String）类型。

返回结果

状态码
  • 200 (OK)：成功。
消息头
HTTP/1.1 200 OK
Server: Apache-Coyote/1.1
Set-Cookie: JSESSIONID=BDF7B4EDFDBA9FBF003D195F802D705F; Path=/imcrs
Date: Fri, 23 Dec 2011 04:44:21 GMT

应用举例

为ID为”87“的虚拟机创建快照。

请求
POST http://192.168.0.4:8080/cas/casrs/spice/login
accept: application/xml


<spice>
  <id>31</id>
  <userName>administrator</userName>
  <domain>h3c/domain>
  <password>123456/password>
</spice>
