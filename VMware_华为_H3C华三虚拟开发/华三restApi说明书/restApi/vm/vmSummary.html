H3C CAS云管理平台

查询指定虚拟机概要信息

根据虚拟机id查询指定虚拟机的概要信息。

接口 URI

/cas/casrs/vm/{id}/summary/new

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 虚拟机id 。
长整数（long）类型。必须，无缺省值。

返回结果

状态码
  • 200(OK)：成功。
消息头
HTTP/1.1 200 OK

元素含义
domain各子元素
title 虚拟机显示名称。
字符串（String）类型。
domainName 虚拟机名称。
字符串（String）类型。
isEdit 是否可修改虚拟机，返回true。
布尔（Boolean）类型。
desc 虚拟机描述信息。
字符串（String）类型。
hostName 虚拟机所在主机名称
字符串（String）类型。
hostIp 虚拟机所在主机IP。
字符串（String）类型。
status 虚拟机状态。
字符串（String）类型。
osVersion 虚拟机操作系统名称。
字符串（String）类型。
system 虚拟机安装的操作系统类型。0：Windows 1：Linux。
整数（int）类型。
cpu 虚拟机CPU总个数。
整数（int）类型。
cpuSocket CPU个数。
整数（int）类型。
cpuCore CPU核数。
整数（int）类型。
memory 虚拟机内存大小。
字符串（String）类型。
cpuRate 虚拟机CPU利用率。
浮点型（Double）类型。
memoryRate 虚拟机内存利用率。
浮点型（Double）类型。
storage 虚拟机存储容量。
字符串（String）类型。
castools CASTools状态。
整数（int）类型。
castoolRunning CASTools是否运行。
布尔（Boolean）类型。
castoolVersion CASTools版本信息。
字符串（String）类型。
castoolStatus 虚拟机CASTools状态。 0或者null：不需要升级 1：待升级 2：须登录系统升级。
整数（int）类型。
createTime 虚拟机创建时间。
字符串（String）类型。
uptime 连续运行时长。
整数（int）类型。
lastOffTime 虚拟机最后关机时间。
字符串（String）类型。
uuid 虚拟机UUID。
字符串（String）类型。
osBit 虚拟机体系结构。
字符串（String）类型。
displayType 虚拟机显示类型。
字符串（String）类型。
autoConfig 虚拟机自动配置。
字符串（String）类型。
vncport 虚拟机VNC端口号。
整数（int）类型。
haStatus 虚拟机HA状态。0：正常 1：未加入HA 2：已加入HA，配置文件不同步。
整数（int）类型。
haManage 虚拟机HA管理状态。0：未被管理 1：被HA管理。
整数（int）类型。
vncProxy 启用VNC代理
字符串（String）类型。
pae 物理地址扩展(PAE)。
字符串（String）类型。
acpi 高级配置与电源管理接口(ACPI)。
字符串（String）类型。
apic 高级可编程中断控制器(APIC)。
字符串（String）类型。
cpuTune CPU调度优先级。
字符串（String）类型。
ioTune I/O优先级。
字符串（String）类型。
memoryTune 内存资源优先级。
字符串（String）类型。
autoMigrate 自动迁移。
字符串（String）类型。
protectModel 保护模式。
字符串（String）类型。
protect 虚拟机是否启用保保护模式。0：不启用 1：启用。
整数（int）类型。
hostStatus 虚拟机所在主机状态。0：不正常 1：正常。
整数（int）类型。
pageHostStatus 主机维护模式。 1：正常模式 3：维护模式。
整数（int）类型。
haEnable 虚拟机所在主机是否启用HA。0：未启用 1：启用。
整数（int）类型。
