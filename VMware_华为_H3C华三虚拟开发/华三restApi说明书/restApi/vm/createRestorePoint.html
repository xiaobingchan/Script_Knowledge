H3C CAS云管理平台

创建虚拟机还原点

根据虚拟机ID为关闭状态下的虚拟机创建还原点。

接口 URI

/cas/casrs/vm/{vmId}/restorePoint

访问方法

HTTP 方法
POST
参数
路径参数（Path Parameter）
vmId 虚拟机id 。
长整数（long）类型。

返回结果

状态码
  • 200(OK)：成功。
消息头

HTTP/1.1 200 OK
消息体
<?xml version="1.0" encoding="UTF-8"?><taskMsg>
  <msgId>1499328443517</msgId>
  <completed>false</completed>
  <progress>0</progress>
  <eventType>0</eventType>
</taskMsg>

应用举例

1.创建虚拟机ID为50的还原点。

请求
POST http://localhost:8080/cas/casrs/vm/50/restorePoint
accept: application/xml
...
回应
HTTP/1.1 200 OK
Content-Type: application/xml
...

</taskMsg>
