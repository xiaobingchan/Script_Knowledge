H3C CAS云管理平台

查询快照策略列表

查询快照策略列表。

接口 URI

/cas/casrs/snapStrategy/snapStrategyList

访问方法

HTTP 方法
GET
消息体
无

返回结果

状态码
  • 200 OK：成功。
  • 409(Conflict)：失败。
回应
<?xml version="1.0" encoding="UTF-8"?>
<list>
  <snapStrategy>
    <id>1</id>
    <name>test</name>
    <desc>desc</desc>
    <state>1</state>
    <flag>1</flag>
    <maxSnapshotNum>0</maxSnapshotNum>
    <frequency>0</frequency>
    <day>5</day>
    <hour>7</hour>
    <minutes>0</minutes>
  </snapStrategy>
</list>
 
元素含义
snapStrategy各子元素
id 快照策略id。
长整型（Long）类型。
name 快照策略名称。
字符串（String）类型。
desc 快照策略描述。
字符串（String）类型。
state 快照策略状态：1：生效；0：不生效。
整型（Integer）类型。
flag 是否快照内存标志，0：快照磁盘，1：快照内存。
整型（Integer）类型。
maxSnapshotNum 允许快照保存的最大个数, 0为不限制，大于0为限制保留最大快照个数。
整型（Integer）类型。
frequency 频率 :0：每月:1：每周，2：每日。
整型（Integer）类型。
day 自动快照的日期-日。
整型（Integer）类型。
hour 自动快照的日期-时。
整型（Integer）类型。
minutes 自动快照的日期-分。
整型（Integer）类型。
