H3C CAS云管理平台

删除快照策略

删除指定的快照策略。

接口 URI

/cas/casrs/snapStrategy/delSnapStrategy?id={id}

访问方法

HTTP 方法
DELETE
消息体
无
元素含义
路径参数（Path Parameter）
id 快照策略 ID。
长整数（long）类型，必须。

返回结果

状态码
  • 204 No Content：成功。

应用举例

删除ID为”2“的快照策略。

请求
DELETE http://192.168.0.4:8080/cas/casrs/snapStrategy/delSnapStrategy?id=2
accept: application/xml
回应
HTTP/1.1 204 No Content
Date: Wed, 17 Sep 2014 07:21:09 GMT
Server: H3C-CVM

无消息体
