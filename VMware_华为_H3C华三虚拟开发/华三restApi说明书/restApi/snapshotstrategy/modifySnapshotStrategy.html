H3C CAS云管理平台

修改快照策略

根据指定的参数修改快照策略。

接口 URI

/cas/casrs/snapStrategy/modifySnapStrategy

访问方法

HTTP 方法
POST
消息体
<snapStrategy>
    <id>3</id>
    <name>name</name>
    <desc>description</desc>
    <state>0</state>
    <flag>1</flag>
    <maxSnapshotNum>3</maxSnapshotNum>
    <frequency>0</frequency>
    <day>5</day>
    <hour>7</hour>
    <minutes>0</minutes>
    <snapStrategyVm>
        <domainId>315</domainId>
        <snapStrategyId>3</snapStrategyId>
    </snapStrategyVm>
    <snapStrategyVm>
        <domainId>316</domainId>
        <snapStrategyId>3</snapStrategyId>
    </snapStrategyVm>
</snapStrategy>
元素含义
snapStrategy各子元素
id 快照策略id。
长整型（Long）类型，必须。
name 快照策略名称，名称的最大输入长度为32个字符，且只允许输入汉字、字母、数字、减号、下划线及点。
字符串（String）类型，必须。
desc 快照策略描述，最大输入长度为120个字符。
字符串（String）类型，可选。
state 快照策略状态：1：生效；0：不生效。
整型（Integer）类型，必须。
flag 是否快照内存标志，0：快照磁盘，1：快照内存。
整型（Integer）类型，必须。
maxSnapshotNum 允许快照保存的最大个数, 0为不限制，大于0为限制保留最大快照个数。
整型（Integer）类型，必须。
frequency 频率 :0：每月:1：每周，2：每日。
整型（Integer）类型，必须。
day 自动快照的日期-日。
整型（Integer）类型，必须。
hour 自动快照的日期-时。
整型（Integer）类型，必须。
minutes 自动快照的日期-分。
整型（Integer）类型，必须。
snapStrategyVm 快照策略中的虚拟机。
列表（List）类型。
snapStrategyVm各子元素
domainId 虚拟机ID。
长整型（Long）类型，必须。
snapStrategyId 快照策略id。
长整型（Long）类型，必须。

返回结果

状态码
  • HTTP/1.1 204 No Content：成功。
  • 409(Conflict)：失败。
回应
无
