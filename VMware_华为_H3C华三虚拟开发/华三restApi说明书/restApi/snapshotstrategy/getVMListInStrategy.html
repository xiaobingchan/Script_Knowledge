查询指定的快照策略下的虚拟机列表

查询指定的快照策略下的虚拟机列表。

接口 URI

/cas/casrs/snapStrategy/snapStrategyVms/{snapId}

访问方法

HTTP 方法
GET
消息体
无
元素含义
路径参数（Path Parameter）
snapId 快照策略 ID。
长整数（long）类型，必须。

返回结果

状态码
  • 200 OK：成功。
消息头
HTTP/1.1 200 OK
Content-Type: application/xml
Content-Length: 306
Date: Wed, 17 Sep 2014 06:50:11 GMT
Server: H3C-CVM
消息体
<?xml version="1.0" encoding="UTF-8"?>
<list>
  <snapStrategyVm>
    <domainId>321</domainId>
    <domainName>gess</domainName>
    <title>tesss</title>
    <domainDesc/>
  </snapStrategyVm>
  <snapStrategyVm>
    <domainId>319</domainId>
    <domainName>winxp</domainName>
    <title>winxp</title>
  </snapStrategyVm>
</list>
元素含义
snapStrategyVm各子元素
domainId 虚拟机 ID。
长整型（Long）类型。
domainName 虚拟机名称。
字符串（String）类型。
title 虚拟机显示名称。
字符串（String）类型。
domainDesc 虚拟机描述。
字符串（String）类型。

应用举例

查询ID为”1“的快照策略中的虚拟机列表。

请求
GET http://192.168.10.80:8080/cas/casrs/snapStrategy/snapStrategyVms/1
accept: application/xml
回应

Content-Type: application/xml
Content-Length: 306
Date: Wed, 17 Sep 2014 06:50:11 GMT
Server: H3C-CVM
