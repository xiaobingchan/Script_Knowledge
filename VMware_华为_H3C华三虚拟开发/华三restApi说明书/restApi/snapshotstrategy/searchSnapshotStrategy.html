H3C CAS云管理平台

查询快照策略详细信息

查询指定ID快照策略的详细信息。

接口 URI

/cas/casrs/snapStrategy/snapStrategyInfo/{snapId}

访问方法

HTTP 方法
GET
消息体
无
元素含义
路径参数（Path Parameter）
snapId 快照策略 ID。
长整数（long）类型，必须。

返回结果

状态码
  • 200 OK：成功。
回应
<?xml version="1.0" encoding="UTF-8"?>
<snapStrategy>
  <id>2</id>
  <name>名称</name>
  <desc>描述</desc>
  <state>0</state>
  <flag>0</flag>
  <maxSnapshotNum>4</maxSnapshotNum>
  <frequency>0</frequency>
  <day>5</day>
  <hour>7</hour>
  <minutes>0</minutes>
  <snapStrategyVm>
    <domainId>312</domainId>
    <snapStrategyId>2</snapStrategyId>
  </snapStrategyVm>
  <snapStrategyVm>
    <domainId>313</domainId>
    <snapStrategyId>2</snapStrategyId>
  </snapStrategyVm>
</snapStrategy>
元素含义
snapStrategy各子元素
id 快照策略id。
长整型（Long）类型。
name 备份策略名称，名称的最大输入长度为32个字符，且只允许输入汉字、字母、数字、减号、下划线及点。
字符串（String）类型。
desc 描述,最大输入长度为120个字符。
字符串（String）类型。
state 快照策略是否生效 1：生效，0：无效。
整型（Integer）类型。
flag 是否快照内存标志，0：快照磁盘，1：快照内存。
整型（Integer）类型。
maxSnapshotNum 允许快照保存的最大个数, 0为不限制，大于0为限制保留最大快照个数。
整型（Integer）类型。
frequency 频率 ：0：每月:1：每周，2：每日。
整型（Integer）类型。
hour 自动快照的时间：小时。
整型（Integer）类型。
minutes 自动快照的时间：分钟。
整型（Integer）类型。
day 自动快照的日期：天 。
整型（Integer）类型。
snapStrategyVm各子元素
domainId 虚拟机 ID。
长整型（Long）类型。
domainName 虚拟机名称。
字符串（String）类型。
title 虚拟机显示名称。
字符串（String）类型。
domainDesc 虚拟机描述。
字符串（String）类型。
