修改监控策略。

接口 URI

/cas/casrs/monitorStrategy/modifyMonStrategy

访问方法

HTTP 方法
PUT
消息体
<?xml version="1.0" encoding="UTF-8"?>
<monitorStrategy>
 <id>1</id>
 <name>动态资源调度监控策略</name>
 <description>abc</description>
 <conditionRelation>0</conditionRelation>
 <createTime>2014-09-15T08:00:27+08:00</createTime>
 <monitorRule>
  <id>9</id>
  <fieldNo>0</fieldNo>
  <fieldName>CPU</fieldName>
  <comparison>&gt;=</comparison>
  <threshold>50</threshold>
 </monitorRule>
 <monitorRule>
  <id>10</id>
  <fieldNo>1</fieldNo>
  <fieldName>RAM</fieldName>
  <comparison>&gt;=</comparison>
  <threshold>50</threshold>
 </monitorRule>
</monitorStrategy>
元素含义
monitorStrategy各子元素
id 监控策略 ID。
长整数（long）类型。
name 监控策略名称。
字符串（String）类型。
description 描述信息。
字符串（String）类型。
conditionRelation 条件关系。
整数（int）类型。
id 监控规则 ID。
长整数（long）类型。
fieldNo 字段编号。
整数（int）类型。
fieldName 字段名称。
字符串（String）类型。
comparison 比较符号。
字符串（String）类型。
threshold 阈值。
整数（int）类型。

状态码
  • 204 (No Content)：成功。
