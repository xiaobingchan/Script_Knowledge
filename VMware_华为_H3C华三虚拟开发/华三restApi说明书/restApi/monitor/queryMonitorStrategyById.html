H3C CAS云管理平台

查询指定监控策略信息

根据ID查询监控策略。

接口 URI

/cas/casrs/monitorStrategy/monStrategyInfo/{id}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 监控策略id 。
长整数（long）类型。必须。

返回结果

状态码
    200(OK)：成功。
消息头

HTTP/1.1 200 OK
Content-Type: application/xml
Content-Length: 346
Date: Wed, 17 Sep 2014 07:05:17 GMT
消息体

<?xml version="1.0" encoding="UTF-8"?>
<list>
  <monitorRule>
    <id>1</id>
    <fieldNo>0</fieldNo>
    <fieldName>CPU</fieldName>
    <comparison>&gt;=</comparison>
    <threshold>80</threshold>
  </monitorRule>
  <monitorRule>
    <id>2</id>
    <fieldNo>1</fieldNo>
    <fieldName>RAM</fieldName>
    <comparison>&gt;=</comparison>
    <threshold>80</threshold>
  </monitorRule>
</list>
元素含义
portProfile各子元素
id 监控规则 ID。
长整数（long）类型。
fieldNo 字段编号。
整数（int）类型。
fieldName 字段名称。
字符串（String）类型。
comparison 比较符号。
字符串（String）类型。
threshold 阈值。
整数（int）类型。
