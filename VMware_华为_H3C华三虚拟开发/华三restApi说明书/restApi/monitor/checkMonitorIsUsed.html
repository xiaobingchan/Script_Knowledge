H3C CAS云管理平台

判定指定监控策略是否存在

根据ID判定监控策略是否存在。

接口 URI

/cas/casrs/monitorStrategy/isExistMonStrategy/{id}

访问方法

HTTP 方法
GET
参数
路径参数（Path Parameter）
id 监控策略id 。
长整数（long）类型。必须。

返回结果

状态码
    200(OK)：成功。
消息头

HTTP/1.1 200 OK
IsExist: true
Content-Type: application/xml
Content-Length: 0
Date: Tue, 16 Sep 2014 10:09:13 GMT

应用举例

1.判定指ID为5的监控策略是否存在。

请求
GET http://localhost:8080/cas/casrs/monitorStrategy/isExistMonStrategy/5
accept: application/xml
...
回应
HTTP/1.1 200 OK
IsExist: true
Content-Type: application/xml
Content-Length: 0
Date: Tue, 16 Sep 2014 10:09:13 GMT
