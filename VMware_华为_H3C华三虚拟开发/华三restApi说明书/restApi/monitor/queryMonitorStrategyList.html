查询CAS系统中监控策略列表。

接口 URI

/cas/casrs/monitorStrategy/monStrategyList

访问方法

HTTP 方法
GET
参数

无

返回结果

状态码
  • 200(OK)：成功。

Server: H3C-CVM
消息体

<list>
  <monitorStrategy>
    <id>1</id>
    <name>动态资源调度监控策略</name>

  </monitorStrategy>
......
</list>
元素含义
monitorStrategy各子元素
id 监控策略 ID。
长整数（long）类型。
name 监控策略名称。
字符串（String）类型。
description 描述信息。
字符串（String）类型。
conditionRelation 条件关系。
整数（int）类型。
createTime 创建时间。
时间（datetime）类型。
